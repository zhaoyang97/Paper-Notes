# AkaCE: A Multimodal Multi-party Dataset for Emotion Recognition in Movie Dialogues

**会议**: ACL 2025
**arXiv**: [2502.10973](https://arxiv.org/abs/2502.10973)
**代码**: [GitHub](https://github.com/dsasu1/AkaCE)
**领域**: 多模态 / 情感识别 / 低资源语言
**关键词**: 阿坎语, 对话情感识别, 多模态数据集, 非洲语言, 韵律标注, 声调语言

## 一句话总结
构建 AkaCE——首个非洲语言多模态对话情感识别数据集，覆盖阿坎语（加纳主要语言，约 2000 万使用者），含 385 段对话 6162 条发言（音频+视觉+文本三模态）、308 名说话人（性别平衡 155男/153女），并提供首个非洲语言词级韵律突出标注。

## 研究背景与动机
1. **领域现状**：对话情感识别（ERC）依赖多模态数据集，但现有基准集中在高资源语言——IEMOCAP/MSP-IMPROV/MELD 覆盖英语，M³ED 覆盖中文，没有任何非洲语言的多模态 ERC 数据集。
2. **现有痛点**：非洲拥有约 3000 种语言和全球 18.3% 人口，但在 ERC 领域完全缺席。现有数据集的情感分类体系基于西方文化视角，跨文化适用性未被验证。
3. **核心矛盾**：如何为低资源非洲语言构建高质量多模态对话情感识别数据集？阿坎语是声调语言——韵律对情感表达至关重要，但既没有 ASR 系统也没有声学对齐模型。
4. **本文要解决什么？** 构建首个非洲语言多模态 ERC 数据集，建立基线并验证数据集质量。
5. **切入角度**：从 21 部阿坎语电影中提取对话，手动标注（因无可用 ASR），并添加词级韵律突出标注——利用声调语言的独特特性。
6. **核心 idea**：AkaCE = 首个非洲语言多模态 ERC + 首个非洲语言韵律标注 + 性别平衡数据集。

## 方法详解

### 数据集构建

1. **数据源选择**：
   - 21 部阿坎语电影（Internet Archive 下载），质量过滤：完整电影 + 语音可辨识 + 面部表情清晰
   - 385 段对话，4477 个话轮，6162 条发言

2. **文本和说话人标注**：
   - **全手动转录**：现有阿坎语 ASR 系统错误率高，因此所有语音由 7 名标注者手动转录+记录时间戳
   - 专业阿坎语语言学家校对所有转录
   - 说话人用唯一标识符标记（出场顺序+性别）

3. **情感标注**：
   - 7 类情感：悲伤、恐惧、愤怒、惊讶、厌恶、快乐、中性（Ekman 6 种基本情感 + 中性）
   - 预培训会议统一标准，参考 Gong et al. (2024) 的标注指南
   - 二次标注 + 多数投票确定最终标签，争议由外部阿坎语情感分析专家裁决
   - 标注者间一致性 Fleiss' κ = 0.488，与 MELD (0.43)、IEMOCAP (0.48)、MSP-IMPROV (0.49) 可比

4. **韵律突出标注**：
   - 两名标注者听音频后为每个词标注是否韵律突出（1/0）
   - 标注者间一致性 Fleiss' κ = 1.0（完美一致）——可能因阿坎语作为声调语言，韵律线索高度清晰（Kügler & Genzel 2012 证实阿坎语说话者通过一致的音高模式标记突出性）
   - 总计 37314 个突出词 vs 79991 个非突出词

### 数据集统计

| 统计项 | 数值 |
|--------|------|
| 电影数 | 21 |
| 对话数 | 385 |
| 发言数 | 6,162 |
| 说话人 | 308 (155男/153女) |
| 总词数 | 117,305 |
| 平均每条发言词数 | 19 |
| 平均每条发言秒数 | 6.7s |
| 总时长 | ~24.3小时 |

### 基线实验设置
- 数据划分：7:1.5:1.5（训/验/测），训练 3888 条、验证 816 条、测试 834 条
- 单模态：文本（BERT/XLM-R）、音频（wav2vec 2.0）、视频（ResNet-50 提取面部特征）
- 多模态融合：特征拼接 + Transformer-based 融合

## 实验关键数据

### 情感分布

| 情感 | 数量 | 占比 |
|------|------|------|
| 中性 | 2,941 | 47.7% |
| 愤怒 | 1,107 | 18.0% |
| 悲伤 | 806 | 13.1% |
| 快乐 | 568 | 9.2% |
| 惊讶 | 364 | 5.9% |
| 厌恶 | 162 | 2.6% |
| 恐惧 | 134 | 2.2% |

### 关键发现
- 多模态融合 > 单模态——整合三种模态后性能一致提升
- **音频对声调语言 ERC 特别重要**：在阿坎语中音频单模态效果接近文本，不同于英语数据集中文本通常远优于音频
- 韵律标注进一步提升情感识别效果
- 情感分布严重不平衡（中性 47.7%，恐惧仅 2.2%）影响少数类识别

## 亮点与洞察
- **三个"首个"**：首个非洲语言多模态 ERC + 首个非洲语言韵律标注 + 首个包含声调语言特征的 ERC 数据集
- **性别平衡设计（155男/153女）**避免了性别偏差，为公平性研究提供基础
- **韵律标注的完美一致性**很有意思——揭示了声调语言中韵律突出的高度共识性，与英语中韵律标注的中等一致性形成对比
- **电影是低资源语言多模态数据的实用来源**——绕过了 ASR、对齐模型等技术瓶颈

## 局限性 / 可改进方向
- **仅阿坎语一种语言**，规模相对小（6162 条）——可扩展到其他非洲语言
- **电影对话可能不反映日常对话**：表演性质的情感表达可能更夸张
- **西方 Ekman 情感分类框架的文化适用性未充分讨论**：不同文化对情感类别的理解可能不同，Ahmad et al. (2025) 的 15 非洲语言研究也用了类似分类，但深入的文化差异分析仍缺少
- **数据不平衡严重**：恐惧/厌恶类别样本极少，可能需要数据增强或采样策略

## 相关工作与启发
- **vs IEMOCAP/MELD（英语）/M³ED（中文）**：AkaCE 填补了非洲语言空白，且独特地包含韵律标注
- **vs 单模态数据集（EmoryNLP/DailyDialog）**：AkaCE 提供音频+视觉+文本三模态，对多模态融合研究更有价值
- 声调语言韵律标注对语音情感分析有独特研究价值——可推广到中文等其他声调语言

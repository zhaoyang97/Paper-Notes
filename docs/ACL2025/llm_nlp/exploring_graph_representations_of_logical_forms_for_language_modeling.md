# Exploring Graph Representations of Logical Forms for Language Modeling

**会议**: ACL 2025  
**arXiv**: [2505.14523](https://arxiv.org/abs/2505.14523)  
**代码**: [https://github.com/mjs227/GFoLDS](https://github.com/mjs227/GFoLDS)  
**领域**: llm_nlp  
**关键词**: 图Transformer, 逻辑形式, DMRS, 语言模型预训练, 语言知识催化假说

## 一句话总结

提出 GFoLDS，一种在 DMRS 逻辑形式图表示上预训练的图 Transformer 语言模型，并提出"语言知识催化假说"(LKCH)：逻辑形式语言模型几乎立刻学会基础语言现象，进而加速复杂模式学习，在相同数据量下大幅超越 BERT。

## 研究背景与动机

1. **传统语言模型的局限**：当前主流的预训练语言模型（如 BERT、GPT）直接在原始文本序列上训练，缺乏对语言深层逻辑结构的显式建模能力。
2. **逻辑形式的潜力**：语言学中的逻辑形式（Logical Forms）如 DMRS（Dependency Minimal Recursion Semantics）能精确刻画句子的语义结构，将自然语言映射为有向图表示。
3. **图表示与 Transformer 的结合**：近年来图 Transformer 在分子建模、知识图谱等领域取得成功，但将其应用于语言逻辑形式的预训练探索还很少。
4. **数据效率问题**：BERT 等模型需要海量文本数据才能学到良好的语言表征，而逻辑形式可能提供更高效的信息编码方式。
5. **语言现象的分层学习**：不同层次的语言现象（词法、句法、语义）在模型训练过程中的学习动态尚不清楚。
6. **可扩展性验证**：需要验证基于逻辑形式的语言模型是否具备向更大规模扩展的潜力。

## 方法详解

### 整体框架

GFoLDS 的整体流程：(1) 将 Wikipedia 文本通过 ERG（English Resource Grammar）解析器转换为 DMRS 图；(2) 在 DMRS 图上设计图 Transformer 架构进行预训练；(3) 通过下游任务微调验证表征质量。

### 关键设计 1：DMRS 图表示

DMRS 是一种基于依存关系的最小递归语义表示。每个节点代表一个语义谓词（predicate），每条边代表谓词之间的语义关系。相比抽象语义表示（AMR），DMRS 保留了更多的语法信息，同时具备形式化的语义解释。论文从 Wikipedia 中提取了 14.6M 个 DMRS 图用于预训练。

### 关键设计 2：图 Transformer 架构

GFoLDS 采用图 Transformer 架构，总参数量约 174M。在标准 Transformer 注意力机制基础上引入图结构信息：通过边类型编码和图位置编码（基于随机游走或最短路径）来注入图拓扑信息，使模型能感知节点间的语义依赖关系。

### 关键设计 3：图级预训练目标

预训练任务包括节点掩码预测（类似 MLM，但在图节点上操作）和图级对比学习目标，使模型同时学习局部节点语义和全局图结构信息。

### 关键设计 4：语言知识催化假说 (LKCH)

论文提出并验证了 Linguistic Knowledge Catalysis Hypothesis（LKCH）：在逻辑形式上训练的语言模型几乎在训练初期就能掌握基础语言现象（如词性、基本句法关系），这种快速习得的基础能力"催化"了更复杂语言模式（如语义蕴涵、词汇关系）的学习过程。

### 损失函数

预训练使用节点掩码预测的交叉熵损失与图对比学习损失的加权组合。下游微调根据具体任务采用分类交叉熵或排序损失。

## 实验关键数据

### 主实验：与 BERT 对比（相同数据量预训练）

| 任务 | 指标 | GFoLDS | BERT (same data) |
|------|------|--------|-------------------|
| RELPRON | MAP | **0.651** | 0.193 |
| SNLI | Accuracy | **81.0%** | 79.9% |

### 消融实验

| 设置 | RELPRON MAP |
|------|-------------|
| GFoLDS (完整) | **0.651** |
| 去除图位置编码 | 0.58x |
| 去除边类型编码 | 0.55x |
| 随机初始化（无预训练） | 0.12x |

### 关键发现

1. GFoLDS 在 RELPRON 上的 MAP 为 0.651，远超 BERT 的 0.193，表明逻辑形式图表示在关系型语义任务上有压倒性优势。
2. 在 SNLI 自然语言推理任务上，GFoLDS 以 81.0% 对 79.9% 小幅领先 BERT，说明即使是需要表层文本理解的任务，图表示也不逊色。
3. LKCH 假说通过训练过程中的探测实验得到验证：基础语言现象在训练早期（<1% 步数）即达到接近最终水平的表现。

## 亮点与洞察

1. **范式创新**：首次系统地在逻辑形式图上进行大规模预训练，证明了"非文本"预训练的可行性和优势。
2. **LKCH 假说**：提出了一个有启发性的学习动态假说，揭示了结构化表示如何加速语言知识的习得过程。
3. **数据效率**：仅用 14.6M 图（远少于 BERT 的文本量）就取得了可比甚至更优的下游表现，展现了逻辑形式表示的信息密度优势。
4. **RELPRON 上的大幅领先**：在关系型语义理解任务上 MAP 从 0.193 提升到 0.651，说明图结构表示在捕捉语义关系方面具有根本性优势。

## 局限性 / 可改进方向

1. **解析器瓶颈**：依赖 ERG 解析器将文本转换为 DMRS 图，解析器的覆盖率和准确率限制了系统的适用性，尤其对非正式文本和非英语语言。
2. **规模限制**：174M 参数和 14.6M 图的规模相比当前主流模型仍然偏小，虽然论文展示了可扩展性证据，但尚未在更大规模上充分验证。
3. **生成能力未验证**：当前仅验证了判别式下游任务，逻辑形式图的表征是否能有效用于生成任务尚不明确。
4. **图构建成本**：大规模文本-图转换的计算成本可能成为实际应用的障碍。
5. **与 LLM 的结合**：未探索与当前大型语言模型融合的可能性，如将逻辑形式作为辅助表征注入 LLM。

## 相关工作与启发

- **AMR 语言模型**：相关工作尝试在 AMR 图上训练模型，但 DMRS 保留了更多语法信息；GFoLDS 的设计可推广到其他逻辑形式。
- **Graph Transformer**：借鉴了 Graphormer、GPS 等图 Transformer 的架构设计，在语言领域的应用是新颖的。
- **探测分析（Probing）**：LKCH 假说的验证方法与 BERTology 中的探测研究一脉相承，但关注的是训练动态而非静态表征。
- **启发**：该工作暗示未来可能存在"混合预训练"范式——同时在文本和结构化语义表示上训练，兼得两者优势。
- **Semantic Parsing LM**：与 seq2seq 语义解析方向互补——后者将文本转为逻辑形式做推理，本文则直接在逻辑形式上构建语言模型。
- **神经符号融合**：GFoLDS 可视为神经符号 AI 在预训练范式下的一次成功实践，符号化的 DMRS 图为图 Transformer 提供了结构化的归纳偏置。

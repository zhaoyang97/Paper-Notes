# ACORD: An Expert-Annotated Retrieval Dataset for Legal Contract Clause Retrieval

**会议**: ACL 2025  
**arXiv**: [2501.06582](https://arxiv.org/abs/2501.06582)  
**代码**: 待确认  
**领域**: 法律NLP / 信息检索  
**关键词**: 合同条款检索, 法律NLP, 专家标注, 信息检索, RAG, 合同起草  

## 一句话总结

构建首个面向合同起草的专家标注条款检索基准 ACORD——114 个律师编写的查询、126,000+ 查询-条款对、1-5 星相关性评分，聚焦责任限制/赔偿/控制权变更/最惠国等复杂条款；bi-encoder 检索 + LLM 点式重排序表现有前景但距律师需求仍有显著差距。LLM 直接起草对比律师修改暴露了多种缺陷。

## 背景与动机

合同是现代商业的基础——43% 公司法务花费过半时间在起草/编辑/谈判合同上。律师几乎从不从零起草，而是检索和改编先例条款。条款检索面临独特挑战：

1. **多层结构**：条款含章节/子章节/段落/例外/交叉引用，可跨数页
2. **语义复杂**：相关性判断高度主观，取决于行业/经验/需求（标注者间 21% 不一致率）
3. **LLM 不能直接起草**：Table 1 展示律师对 LLM 起草的条款标注了多处缺陷（条款冲突、非标准语言等）
4. **缺乏领域基准**：现有法律 NLP 数据集多为 QA/分类格式，无专门的合同检索基准

## 核心问题

现有检索模型能否有效为律师找到最相关的先例条款？需要什么样的基准来评估合同条款检索？

## 方法详解

### ACORD 数据集

- **114 个查询**：由资深律师编写，覆盖责任限制、赔偿、肯定性契约、限制性契约、终止条款、适用法律等 6 大类
- **126,000+ 查询-条款对**：每对 1-5 星专家相关性评分
- **标注团队**：资深律师 + 学生标注员，三阶段质量控制（初标→交叉检查→专家委员会审核）
- **标注成本估算**：按律师时薪 $550 + 非律师 $150 计算，超过 100 万美元

### 基线评估

- **检索器**：BM25、bi-encoder 密集检索（E5、GTE 等）
- **重排序器**：点式 LLM 重排序（让 LLM 评估查询-条款对的相关性）
- **指标**：NDCG + 任务特定的归一化精度分数

### LLM 起草 vs 先例检索

Table 1 的律师标注清楚展示 LLM 起草的缺陷：
- 条款间冲突（"Notwithstanding" 使例外条款无效化了限制条款本身）
- 非标准语言（"Application of Limitations" 在商业合同中罕见）
- 缺少关键概念（如"paid and payable" 而非仅"paid"）

## 实验关键数据

- Bi-encoder + LLM 重排序表现最好，但距完美仍有显著差距
- 条款检索比标准 IR 更难——长文本、交叉引用、主观性高
- LLM 直接起草 vs ACORD 5 星条款差距明显——证明检索先例的必要性

## 亮点

- **首个合同条款检索基准**：填补了法律 NLP 中 IR 任务的空白
- **专家标注极其昂贵（>$1M 等值）**：标注质量有保证
- **LLM 起草缺陷的律师标注**：直观展示为何检索比生成更可靠
- **实用性强**：直接服务于律师日常工作中最常见的任务——找先例条款

## 局限性 / 可改进方向

- **仅英语合同**：其他法律体系/语言未覆盖
- **条款类型有限**：6 大类 114 查询，虽有代表性但不全面
- **21% 标注不一致**：法律条款相关性固有的主观性
- **未包含 RAG 端到端评估**：仅检索环节，未测试检索后的起草质量

## 与相关工作的对比

- **vs CUAD**：CUAD 是条款分类/提取；ACORD 是条款检索排序——不同任务
- **vs LePARD/CLERC**：案例法检索，用引用自动构建查询；合同无引用，需律师手工编写查询
- **vs BigLaw Bench**：通用法律任务，标注数量级少于 ACORD

## 启发与关联

- 合同条款检索是 RAG 在法律领域最直接的杀手级应用
- LLM 起草缺陷的定量分析为"检索+改编优于直接生成"提供了证据
- 法律 NLP 需要更多专家标注的任务特定基准（而非通用 QA 格式）

## 评分

- 新颖性: ⭐⭐⭐⭐ 首个合同条款检索基准，LLM 起草缺陷分析有价值
- 实验充分度: ⭐⭐⭐⭐ 多模型基线 + 详细评估，但缺 RAG 端到端
- 写作质量: ⭐⭐⭐⭐⭐ 律师标注案例展示特别有说服力
- 价值: ⭐⭐⭐⭐⭐ 对法律 NLP 和 RAG 社区有重要资源价值

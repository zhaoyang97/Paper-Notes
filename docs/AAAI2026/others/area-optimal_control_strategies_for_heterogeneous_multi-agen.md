# Area-Optimal Control Strategies for Heterogeneous Multi-Agent Pursuit

**会议**: AAAI 2026  
**arXiv**: [2511.15036v2](https://arxiv.org/abs/2511.15036v2)  
**代码**: 无  
**领域**: 多智能体系统 / 博弈论 / 控制  
**关键词**: 追逃博弈, 异构多智能体, Apollonius圆, 面积最优, 闭式控制律, 安全可达集  

## 一句话总结
研究异构速度下多追逐者-单逃避者的追逃博弈——定义逃避者安全可达集为所有追逐者-逃避者对的Apollonius圆的交集，将捕获策略建模为追逐者最小化/逃避者最大化该交集面积的零和博弈，推导出闭式最优航向控制律，仿真验证追逐者可系统性缩小安全区域实现保证捕获。

## 背景与动机
多智能体追逃（pursuit-evasion）是多智能体系统和博弈论中的经典问题，在军事拦截、搜索救援、机器人围捕等场景有广泛应用。现有方法的主要局限：（1）大多假设同构速度（所有追逐者速度相同），无法处理异构场景；（2）数值优化方法计算成本高，不适合实时控制；（3）缺乏对最优策略的解析刻画。Apollonius圆——两个移动体以不同速度同时到达的点的几何轨迹——为异构速度追逃提供了自然的几何描述工具。

## 核心问题
**异构速度的多追逐者如何协同最小化逃避者的安全可达区域？能否得到闭式最优控制律以支持实时决策？**

## 方法详解

### 整体框架
将追逃博弈的目标从传统的"最小化捕获时间"转化为"最小/最大化安全可达集面积"。安全可达集定义为逃避者能在所有追逐者之前到达的区域——几何上是所有追逐者-逃避者对的Apollonius圆的交集。这个面积成为零和博弈的支付函数。

### 关键设计
1. **Apollonius圆交集**: 对每对追逐者-逃避者，Apollonius圆的半径由速度比决定，圆心位于两者连线上按速度比内分的点。多追逐者场景下，安全可达集是所有Apollonius圆的交集——一个凸区域。
2. **面积梯度解析计算**: 对交集面积关于各智能体位置求解析梯度。追逐者的最优航向为面积关于其位置梯度的负方向（减小面积），逃避者的最优航向为正方向（增大面积）。
3. **闭式瞬时最优控制**: 最终控制律仅需计算Apollonius圆参数和面积梯度，不涉及数值优化或前向模拟，计算复杂度极低，适合实时嵌入式系统。
4. **异构速度处理**: 不同速度直接编码在各自Apollonius圆的半径和圆心中，框架天然支持异构——速度越快的追逐者对应更大的Apollonius圆，对逃避者施加更强的约束。

### 损失函数 / 训练策略
无学习/训练组件。控制律是解析推导的瞬时最优策略，基于当前状态（位置、速度）直接计算最优航向角。

## 实验关键数据
| 场景 | 追逐者数 | 速度配置 | 结果 | 备注 |
|------|---------|---------|------|------|
| 2v1异构 | 2 | 不同速度 | 安全区域单调递减 | 保证捕获 |
| 3v1异构 | 3 | 不同速度 | 收敛更快 | 协同效应 |
| 对比数值方法 | - | - | 轨迹一致 | 验证解析解正确性 |
| 逃避者反制 | - | - | 逃避者最大化面积 | 零和博弈均衡 |

### 消融实验要点
- 异构vs同构：异构速度时最优策略与同构假设下的策略显著不同——速度快的追逐者承担更多包围职责
- 追逐者数量增加时安全区域收缩加速，但边际效益递减
- Apollonius圆交集的几何形状随时间演化，从接近圆形逐渐变为细长区域直到退化为零面积

## 亮点
- **解析之美**: 完全闭式的最优控制律，不依赖数值优化——闭式解的价值远超数值解
- **几何直觉清晰**: 面积最小化作为空间控制的优化目标极具直觉性，Apollonius圆交集可视化让策略一目了然
- **实时可行**: 闭式解的计算开销极低，可直接部署在资源受限的嵌入式平台

## 局限性 / 可改进方向
- 仅有仿真验证，缺少物理机器人实验——通信延迟、定位误差、运动学约束等未考虑
- 2D平面假设，3D空间（如无人机追逃）需处理Apollonius球的交集
- 单逃避者假设，多逃避者场景下安全可达集的定义和博弈结构更复杂
- 瞬时最优可能非全局最优——贪心策略在某些极端初始配置下可能不是Nash均衡

## 与相关工作的对比
与传统Voronoi划分追逃方法相比，Apollonius圆方法天然处理异构速度（Voronoi假设等速）。与基于RL的追逃方法相比，解析方法提供最优性保证和可解释性，但缺少对环境不确定性的适应能力。与基于微分博弈的Hamilton-Jacobi方法相比，面积目标更直观且计算更高效。

## 启发与关联
- Apollonius圆是处理异构速度问题的万能工具——在无人机路径规划、传感器覆盖等问题中也有潜在应用
- "面积最小化"替代"时间最小化"的目标转换是关键洞察——面积目标使梯度解析化成为可能
- 理论工作的典范：问题定义清晰、推导完整、结论简洁

# TDCNet: Spatio-Temporal Context Learning with Temporal Difference Convolution for Moving IRSTD

**会议**: AAAI 2026  
**arXiv**: [2511.09352](https://arxiv.org/abs/2511.09352)  
**代码**: https://github.com/IVPLaboratory/TDCNet  
**领域**: 遥感 / 红外目标检测  
**关键词**: 红外小目标检测, 时间差分卷积, 重参数化, 时空注意力, 无人机

## 一句话总结
提出 TDCNet，将时间差分和 3D 卷积融合为统一的时间差分卷积 (TDC)，通过重参数化实现推理零额外开销，配合 TDC 引导的时空注意力，在自建 IRSTD-UAV 数据集上 F1 达 97.12%（AP50 93.83%），同时发布 15,106 帧真实红外无人机数据集。

## 研究背景与动机

1. **领域现状**：运动红外小目标检测 (IRSTD) 需要融合时序信息。现有方法要么用帧差分（捕获运动但丢失外观），要么用 3D 卷积（保留外观但忽略显式运动线索）。

2. **现有痛点**：帧差分和 3D 卷积各有偏倚、互补但未被统一；公开红外小目标视频数据集稀缺。

3. **核心矛盾**：如何在不增加推理开销的情况下同时利用运动和外观特征？

4. **本文要解决什么？** 将帧差分隐式融入 3D 卷积权重结构，训练多分支多尺度、推理零开销。

5. **切入角度**：从 3D 卷积权重分解的数学角度证明时间差分可以被编码为特殊的权重结构。

6. **核心 idea 一句话**：TDC = 帧差分 ⊕ 3D卷积 的统一表示，可重参数化为标准 3D 卷积。

## 方法详解

### 整体框架
三并行骨干：TDC 骨干(时间差分特征) + 3D 骨干(时空特征) + 2D 骨干(当前帧空间特征)。TDCSTA 注意力融合后进入检测头。渐进训练——先预训练 2D/3D 骨干，再联合训练 TDC+TDCSTA。

### 关键设计

1. **时间差分卷积 (TDC)**：将 3D 卷积权重沿时间维度重结构化以隐式编码帧差分。三种时间尺度：
   - **S-TDC**: 短期，连续帧差分 ($F_t - F_{t-1}$)
   - **M-TDC**: 中期，隔帧差分 ($F_t - F_{t-2}$)
   - **L-TDC**: 长期，当前帧与所有历史帧差分 ($F_5 - F_t$)

2. **TDCR 重参数化模块**：三个并行 TDC 块各自有独立 BN，推理时数学等价合并为单个 3D 卷积。FLOPs 减少 7.3G (102.96→95.67) 且精度完全不变。

3. **TDCSTA 时空注意力**：(1) 三流各自做 3D 局部窗口自注意力；(2) TDC 特征做 Q，3D 特征做 K，2D 特征做 V 的交叉注意力——TDC 特征"引导"空间外观信息的聚合。

4. **IRSTD-UAV 数据集**：17 个序列、15,106 帧真实红外无人机图像，公开发布。

### 损失函数 / 训练策略
检测损失（YOLOv8风格）。Adam, lr=0.001。输入：5帧连续帧，640×640。RTX 3090。

## 实验关键数据

### 主实验

| 方法 | 类型 | IRSTD-UAV F1/AP50 | IRDST F1/AP50 |
|------|------|------------------|---------------|
| YOLO11-L | 单帧 | 95.99/91.20 | 96.35/92.10 |
| MOCID | 多帧 | 96.05/91.32 | 97.88/94.74 |
| **TDCNet** | **多帧** | **97.12/93.83** | **97.91/94.79** |

### 消融实验

| 方法 | F1 | AP50 | FLOPs(G) |
|------|-----|------|----------|
| 仅时间差分 | 92.25 | 89.73 | 41.4 |
| 仅3D卷积 | 87.36 | 75.97 | 45.8 |
| TD + 3D（分开）| 93.87 | 89.81 | 45.9 |
| **TDC（统一）** | **96.76** | **92.50** | **45.7** |

| TDC尺度 | F1 | AP50 |
|---------|-----|------|
| 仅S-TDC | 94.91 | 90.31 |
| 仅M-TDC | 95.65 | 90.46 |
| 仅L-TDC | 96.29 | 92.35 |
| **全部** | **96.76** | **92.50** |

### 关键发现
- TDC 比 TD+3D 分开使用好 2.89 F1，且 FLOPs 相当——统一表示的优势
- L-TDC 单独贡献最大(F1 96.29)，但三尺度组合最优
- 重参数化零精度损失、减少 7% FLOPs
- TDC 做 Q 的交叉注意力关键（替换为 SF 做 Q 则 F1 降至 91.26）

## 亮点与洞察
- **TDC 的数学统一**非常优雅——将两种互补的时序建模方式融入单个卷积核，训练多分支、推理零开销
- **IRSTD-UAV 数据集**填补了红外小目标视频数据的空白，对社区有直接贡献

## 局限性 / 可改进方向
- 固定 5 帧输入，更长序列可能进一步提升
- 仅在小目标场景验证，对常规目标检测的适用性未测试
- 渐进训练策略增加了训练复杂度

## 相关工作与启发
- **vs MOCID**: 同为多帧方法，TDCNet 在两个数据集上均超越
- **vs STMENet**: STMENet F1 仅 87.36，远低于 TDCNet 的 97.12

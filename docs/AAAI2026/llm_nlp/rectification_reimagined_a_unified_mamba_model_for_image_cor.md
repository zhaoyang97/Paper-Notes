# Rectification Reimagined: A Unified Mamba Model for Image Correction and Rectangling with Prompts

**会议**: AAAI 2026  
**arXiv**: [2512.18718](https://arxiv.org/abs/2512.18718)  
**代码**: [https://github.com/yyywxk/UniRect](https://github.com/yyywxk/UniRect)  
**领域**: 图像校正 / 图像矫形  
**关键词**: image correction, image rectangling, unified rectification, Mamba, thin-plate spline, mixture-of-experts

## 一句话总结

从统一畸变矫正视角出发，提出 UniRect 框架，通过 Residual Progressive TPS 处理几何形变 + Residual Mamba Blocks 补偿退化，统一处理肖像校正、广角矩形化、拼接矩形化、旋转校正四种任务，并通过 Sparse MoE 实现 four-in-one 多任务学习，拼接矩形化 PSNR 提升 3.82 dB，旋转校正提升 0.87 dB。

## 研究背景与动机

智能手机摄影系统涉及多种图像校正任务，现有方法均采用任务专用架构，缺乏统一性。本文首次从畸变物理模型角度建立四种任务的数学联系：

| 任务 | 畸变来源 | 对应模型 | 核心挑战 |
|------|---------|---------|---------|
| 肖像校正 (T1) | 广角镜头畸变 | Kannala-Brandt 模型 | 人脸形状恢复 |
| 广角矩形化 (T2) | 广角校正后的不规则边界 | Brown-Conrady 模型 | 非矩形边界填充 |
| 拼接矩形化 (T3) | 全景拼接后的不规则边界 | Brown-Conrady 变体 + 偏移 T₀ | 复杂非线性形变 |
| 旋转校正 (T4) | 拍摄时的旋转角度 | Brown-Conrady + 旋转矩阵 Rα | 微妙旋转感知 |

现有方法的局限：(1) 每个任务一个专用模型，边缘设备部署资源浪费；(2) 不同任务间的畸变耦合关系未被利用；(3) 多任务联合训练时存在严重的任务竞争（mixed learning 导致各任务性能下降）。

## 方法详解

### 整体框架

UniRect 由两大模块组成：**Deformation Module (DM)** 基于 RP-TPS 估计并矫正几何畸变；**Restoration Module (RM)** 基于 Residual Mamba Blocks 补偿形变过程中的插值退化。输入图像 $X_0^i$ 与对应的视觉提示 $M_0^i$ 共同送入框架，提示指示任务类型。

### 关键设计

1. **通用畸变模型**：建立统一数学方程 $[x_d, y_d]^T = \frac{1}{r} \sum_{j=1}^N (k_j \theta^{2j-1} + k'_j r^{2j-1}) \mathcal{R}_\alpha [x, y]^T + T_0$，四种任务通过不同参数简化对应不同畸变类型
2. **RP-TPS (Residual Progressive TPS)**：两级控制点预测器 $\mathcal{C}_0, \mathcal{C}_1$ 渐进预测控制点偏移，关键创新在于两次采样均从原始输入 $X_0^i$ 进行（而非中间结果），有效避免中间插值累积误差。控制点网格 12×10
3. **Residual Mamba Blocks (RMBs)**：4 个 RMB（32 通道）构成 RM，利用 Mamba 的序列扫描特性捕获全局几何信息和长程依赖。对存在不规则边界的 T2/T3 任务通过 partial convolution 处理无效区域
4. **视觉提示设计**：T1 = 人脸掩码（关注人脸畸变）、T2/T3 = 边界掩码（增强边界感知，参与边界损失计算）、T4 = 全白图（关注整体内容感知旋转）
5. **SMoEs (Sparse Mixture-of-Experts)**：5 个 UniRect 专家网络 + ResNet18 门控网络，Top-1 选择激活。门控网络根据输入自动分配任务到合适专家，解决多任务竞争问题

### 损失函数

$$\mathcal{L}_{DM} = \sum_{j=0}^{1} \gamma^j (\mathcal{L}_a^j + \alpha_1 \mathcal{L}_b^j + \alpha_2 \mathcal{L}_p^j + \alpha_3 \mathcal{L}_g^j)$$

其中 $\mathcal{L}_a$ 为 L1 外观损失，$\mathcal{L}_b$ 为边界损失（驱动控制点趋近边界），$\mathcal{L}_p$ 为线条/形状惩罚（防止过度矫正），$\mathcal{L}_g$ 为梯度损失（对齐局部纹理）。$\gamma=0.9$, $\alpha_1=0.01$, $\alpha_2=1.0$, $\alpha_3=0.01$。RM 使用 L1 + 感知损失。4×V100, batch 4, 200 epochs, poly lr (初始 1e-4)。

## 实验关键数据

### 主实验：四任务 SOTA 对比

| 任务 | 方法 | PSNR↑ | SSIM↑ | FID↓ | LPIPS↓ |
|------|------|-------|-------|------|--------|
| 广角矩形化 (T2) | RecRecNet | 18.68 | 0.5450 | 19.01 | 0.1136 |
| | **UniRect** | **19.90** | **0.5721** | 27.02 | 0.1245 |
| 拼接矩形化 (T3) | Nie et al.'22 | 21.28 | 0.7141 | 21.77 | 0.1557 |
| | **UniRect** | **25.10** | **0.7526** | **19.59** | **0.1120** |
| 旋转校正 (T4) | CoupledTPS | 22.29 | 0.6790 | 7.90 | 0.1970 |
| | **UniRect** | **23.16** | **0.7179** | **6.55** | **0.0873** |

### 消融实验：多任务学习策略对比

| 策略 | T1 ShapeACC↑ | T2 PSNR↑ | T3 PSNR↑ | T4 PSNR↑ | 参数量 |
|------|-------------|----------|----------|----------|--------|
| Mixed Learning | 97.223 | 13.07 | 15.74 | 21.73 | 357.9M |
| Sequential (1-4-3-2) | 97.401 | 15.23 | 16.94 | 21.51 | 357.9M |
| Sequential (3-2-4-1) | 97.409 | 17.54 | 23.55 | 15.10 | 357.9M |
| **UniRect (SMoEs)** | **97.390** | **19.90** | **25.07** | **23.16** | 369.6M |
| 单任务学习 (上界) | 97.454 | 19.90 | 25.10 | 23.16 | 357.9M×4 |

### 关键发现

- 混合学习 T3 仅 15.74 dB vs 单任务 25.10 dB，任务竞争极其严重
- 顺序学习存在"先入为主"效应：首个训练任务始终主导性能
- SMoEs 几乎逼近单任务上界，参数仅增加 3.3%（369.6M vs 357.9M）
- RP-TPS 对旋转校正贡献最大：PSNR 从 21.56→23.16 (+1.6 dB)
- 视觉提示对 T1 (ShapeACC +1.13) 和 T3 (PSNR +1.28 dB) 作用最大

## 亮点与洞察

- 首次从物理畸变模型角度统一四种图像校正/矩形化任务，数学推导严谨
- RP-TPS 的"始终从原始输入采样"设计巧妙避免了中间插值退化
- Mamba 架构在几何校正中的有效性得到验证（优于 CNN 和 Transformer）
- 视觉提示机制简洁高效，不同任务通过不同掩码自然区分
- SMoEs 以极低开销解决多任务竞争，four-in-one 接近 four-by-one 性能

## 局限性

- 无法同时对一张图施加多种矫正（数据集限制，每张图仅对应一种畸变类型）
- 门控网络难区分 T1 和 T4（两者数据分布相似，均无边界变化）
- T2 的 FID 指标 (27.02) 弱于 RecRecNet (19.01)，感知质量有待提升
- 未探索更多校正任务（如鱼眼校正、透视校正）的扩展

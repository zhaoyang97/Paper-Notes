# Seeing the Unseen: Zooming in the Dark with Event Cameras

**会议**: AAAI 2026  
**arXiv**: [2601.02206](https://arxiv.org/abs/2601.02206)  
**代码**: [https://github.com/DachunKai/RetinexEVSR](https://github.com/DachunKai/RetinexEVSR)  
**领域**: 低光照视频超分辨率 / 事件相机融合  
**关键词**: low-light video super-resolution, event camera, Retinex decomposition, cross-modal fusion, bidirectional fusion  

## 一句话总结
提出首个事件驱动低光照视频超分辨率框架RetinexEVSR，通过Retinex启发的双向融合策略（光照引导事件增强+事件引导反射增强），在SDSD-indoor上较EvTexture提升2.95 dB且FLOPs减少86%、推理加速65%。

## 背景与动机
- LVSR面临对比度不足+高频缺失双重挑战，级联策略(LVE+VSR或反之)各阶段误差放大
- 事件相机120dB高动态范围可在暗光下提供互补边缘/运动信息
- 但低光下事件数据也有噪声/拖尾/非平稳分布，两个退化信号直接融合引入伪影
- 核心洞察：两模态退化主要源于光照不足，需额外低光先验辅助

## 核心问题
1. 如何从两个退化模态中有效提取有意义信息？
2. 如何利用Retinex分解为跨模态融合提供结构化引导？

## 方法详解

### 整体框架
输入LR帧+事件数据。每帧Retinex分解为光照I+反射R。IEE用光照引导事件增强→ERE用增强事件恢复反射细节→三者联合引导上采样。双向循环框架进行时序传播，从反射图(非原帧)计算光流实现更精确对齐。

### 关键设计
1. **RBF双向融合策略**: 光照(平滑/低噪声)→引导事件精炼；增强事件(高频)→补充反射缺失细节
2. **IEE (光照引导事件增强)**: 双分支提取浅层特征→四尺度(1/½/¼/⅛)多尺度融合，光照特征在各尺度通过拼接+卷积校准事件表示→自底向上渐进上采样，保留顶三尺度增强特征
3. **ERE (事件引导反射增强)**: 编码器-瓶颈-解码器架构。动态对比度提取器(DCE)用深度卷积计算4个时间投影维度的动态key/value，自适应捕获事件时间变化。输出同时作为时序状态更新

### 损失函数 / 训练策略
- Charbonnier损失, Adam, 余弦退火
- SpyNet+SCI lr=2.5e-5(前5K冻结), 其余lr=2e-4
- 15帧输入, batch 8, 64×64, 300K迭代, 2×RTX4090, ~4天/数据集

## 实验关键数据
| 数据集 | 指标 | 本文 | EvTexture | 提升 |
|--------|------|------|----------|------|
| SDSD-indoor | PSNR/SSIM | 30.28/0.8932 | 27.33/0.8776 | +2.95 dB |
| SDSD-outdoor | PSNR/SSIM | 25.15/0.7737 | 24.20/0.7587 | +0.95 dB |
| SDE-indoor | PSNR/SSIM | 21.24/0.6525 | 20.29/0.6301 | +0.95 dB |
| SDE-outdoor | PSNR/SSIM | 20.68/0.6541 | 19.75/0.6046 | +0.93 dB |
| RELED | PSNR/tOF | 28.92/4.60 | 28.07/4.69 | +0.85 dB |

**计算效率**: 8.07M参数 / 159.1G FLOPs / 44.5ms (vs EvTexture: 8.90M / 1141.1G / 126.9ms)

### 消融实验要点
- 完整30.28, 去IEE→28.27(-2.01), 去ERE→27.31(-2.97)
- IEE 1/2/3尺度: 28.64/28.83/30.28, 三尺度增益1.64 dB
- 反射图vs低光帧计算光流: +0.43 dB
- 无监督SCI优于有监督URetinex-Net: +0.66 dB

## 亮点
- 首个事件+Retinex结合的LVSR框架
- FLOPs仅159.1G (EvTexture的14%)，效果效率双优
- 真实世界SR评估同样优异(NIQE/PI/CLIP-IQA/Q-Align)

## 局限性 / 可改进方向
- 依赖预训练SCI Retinex模型；SDSD事件通过模拟器生成
- 极端运动模糊+极暗同时存在的挑战; 未探索>4×放大

## 与相关工作的对比
对比5类策略×多种方法组合，一阶段事件方法全面最优。vs FMA-Net(前最优RGB): SDSD-in +2.75 dB。

## 启发与关联
Retinex分解作为跨模态融合中间表示有广泛适用性。事件相机在极端条件下互补价值充分验证。

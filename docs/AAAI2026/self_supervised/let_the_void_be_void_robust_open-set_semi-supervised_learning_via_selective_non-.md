# Let the Void Be Void: Robust Open-Set Semi-Supervised Learning via Selective Non-Alignment

**会议**: AAAI 2026  
**arXiv**: [2504.12569](https://arxiv.org/abs/2504.12569)  
**代码**: 无  
**领域**: 自监督 / 半监督学习 / OOD 检测  
**关键词**: 开放集半监督学习, 对比学习, OOD检测, 选择性非对齐, 原型学习

## 一句话总结
提出 SkipAlign 框架，在对比学习的传统 pull/push 操作之外引入第三种 "skip" 操作，对低置信度样本选择性跳过对齐（只做温和排斥），使 ID 类形成紧凑"星系"、OOD 样本自然散布于"星际虚空"，在未见过的 OOD 检测中平均 AUC 提升 +3.1，最高 +7.1。

## 研究背景与动机

1. **领域现状**：半监督学习（SSL）假设未标注数据都属于已知类别，但实际场景中未标注数据是开放集——包含未知分布外（OOD）样本。开放集半监督学习（OSSL）需要同时提升 ID 分类精度并检测 OOD 样本。

2. **现有痛点**：现有方法走两个极端——（a）**激进方案**（SCOMatch, ProSub）：将所有 OOD 样本压缩到单一合成类或 ID 子空间，导致几何坍缩和对已见 OOD 的过拟合，泛化到未见 OOD 时失败；（b）**保守方案**（SSB）：直接丢弃低置信度样本，浪费了宝贵的几何线索。

3. **核心矛盾**：OOD 空间本质上是巨大且异质的，但训练中只能看到有限的 OOD 样本。强制将 OOD 对齐到某处（激进）会引入偏见，完全丢弃不确定样本（保守）又浪费信息。

4. **本文要解决什么？** 在不强制对齐 OOD 样本、也不丢弃不确定样本的前提下，产生对未见 OOD 的泛化检测能力。

5. **切入角度**：将嵌入空间类比为宇宙——ID 类是紧凑的"星系"，OOD 是"星际虚空"。不对不确定样本做对齐，只施加温和排斥，让虚空保持虚空。

6. **核心 idea 一句话**：在 pull/push 之外加入 "skip"——对不确定样本不拉向任何原型，仅做非选择性角度排斥，保留 OOD 的自然多样性和分散性。

## 方法详解

### 整体框架
SkipAlign 由共享骨干网络 + 三个任务头组成：
- **闭集分类器 CC**：K 类 softmax 分类，输出 $\mathbf{p} \in \mathbb{R}^K$
- **OOD 检测器 OD**：K 个 OVA 二分类器，每个输出 $(\varphi_k^{ID}, \varphi_k^{OOD})$
- **SNA 模块**：在投影空间中对特征做选择性对比学习（核心创新）

### 关键设计

1. **选择性非对齐（SNA）**:
   - 做什么：对未标注样本选择性地决定是 pull 向原型还是 skip(只做温和排斥)
   - 核心思路：对每个未标注嵌入 $z_i$，通过**双门控 ID 选择**——同时要求闭集分类器高置信 ($p_{i,\hat{k}} > \tau_{ID}$) 和 OOD 检测器高置信 ($\varphi_{i,\hat{k}}^{ID} > \eta_{ID}$)——计算置信掩码 $\Phi_i$。SNA 损失为 $\mathcal{L}_{USNA}(z_i) = -\Phi_i \cdot \text{sim}(z_i, \mu_{\hat{k}})/T + \log\sum_j\exp(\text{sim}(z_i, \mu_j)/T)$。当 $\Phi_i=1$ 时有完整的 pull+push；当 $\Phi_i=0$ 时 pull 项消失，只剩 softmax 加权的非选择性角度排斥
   - 设计动机：梯度分析表明，$\Phi_i=0$ 时的更新方向是 $\sum_j \alpha_j \hat{\mu}_j$（ID 原型的软加权平均方向），且被投影算子 $(I-\hat{z}\hat{z}^T)$ 约束为纯角度更新。离 ID "星系"越近的 OOD 样本受到越强的排斥力，但不会被拉向任何特定"星系"

2. **双门控 ID 选择**:
   - 做什么：严格筛选哪些未标注样本可以被 pull 向原型
   - 核心思路：同时要求闭集分类器和 OOD 检测器都对同一类别给出高置信度判定，大幅降低将 OOD 样本误拉入 ID 簇的风险
   - 设计动机：单一分类器的伪标签容易出错，双门控相当于"两把锁"，错误同时发生的概率远低于单门控

3. **自适应原型精炼**:
   - 做什么：动态更新类原型，不仅用有标签样本，还纳入高置信伪标签的未标注样本
   - 核心思路：最终原型 $\mu = w_l \mu_l + w_u \mu_u$，是标注样本原型和精选未标注样本原型的加权平均，权重自适应调节
   - 设计动机：仅靠少量标注样本（如每类 25-50 个）的原型代表性不足，纳入可靠伪标签样本可提升原型质量

### 损失函数 / 训练策略
- **总损失**：$\mathcal{L} = \mathcal{L}_{CC} + \mathcal{L}_{OD} + \mathcal{L}_{SNA}$
- 闭集分类器用 FixMatch 框架（监督 CE + 一致性正则化）
- OOD 检测器用 OVA 损失 + 熵最小化 + SOCR 一致性 + 伪负样本损失
- SNA 损失 = $\lambda_{USNA}\mathcal{L}_{USNA} + \lambda_{IA}\mathcal{L}_{IA} + \lambda_{PA}\mathcal{L}_{PA}$（未标注 SNA + 有标注实例对齐 + 有标注原型对齐）
- 有标注样本的实例对齐使用监督对比学习（SupCon），原型对齐始终 pull 向真实类原型

### 关键发现
- **t-SNE 可视化**清晰显示：ID 类形成 6 个紧凑簇，已见和未见 OOD 都散布在簇间"虚空"中，没有黏附到任何 ID 簇
- **特征范数差异**：ID 样本范数显著高于 OOD（>2 倍），因为只有被 pull 的 ID 样本积累了范数
- **余弦相似度**：ID 样本与对应原型高相似度（~0.8），已见和未见 OOD 与所有原型都低相似度（<0.3）
- SCOMatch 在 CIFAR-100 上 AUC 仅 62.2%（vs SkipAlign 82.8%），证实将 OOD 强制压到单一类会严重过拟合已见 OOD
- 双门控比单门控 AUC 提升约 2 个点，证明两个检测器的交叉验证有效

## 亮点与洞察
- **"skip" 作为第三种对比操作**的概念优雅简洁：不 pull 不代表不处理，温和排斥让信息不被浪费，也避免了错误对齐。这个设计哲学——"让虚空保持虚空"——值得在其他开放集/长尾/噪声标签场景中借鉴
- **梯度分析清晰**：论文从 $\nabla_{z_i}\mathcal{L}_{USNA}$ 的角度精确解释了 skip 操作的几何效应——纯角度排斥 + 距离自适应强度
- **星系-虚空类比**直观有效，将复杂的几何结构用天文学概念解释得很清楚

## 局限性 / 可改进方向
- 阈值 $\tau_{ID}$ 和 $\eta_{ID}$ 是超参数，对不同数据集需要调节，没有自适应选择机制
- 实验主要在中小规模数据集上验证（CIFAR-10/100, ImageNet-30, TinyImageNet），大规模场景未测试
- SNA 的非选择性排斥是对所有原型的 softmax 加权平均，当 ID 类数很多时可能排斥力分散

## 相关工作与启发
- **vs SCOMatch**：将所有 OOD 压到一个类，导致几何坍缩。在 CIFAR-100 上 AUC 62.2% vs SkipAlign 82.8%，差距 20 个点
- **vs ProSub**：定义 ID 子空间并排斥 OOD，但子空间内的未见 OOD 会被误判为 ID。SkipAlign 不使用子空间概念，只用各类原型
- **vs SSB**：丢弃低置信度样本，SkipAlign 将这些样本转化为纯排斥信号保留其几何价值。AUC 一致优于 SSB

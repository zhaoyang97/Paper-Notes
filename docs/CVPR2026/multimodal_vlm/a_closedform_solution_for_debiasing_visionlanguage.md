# A Closed-Form Solution for Debiasing Vision-Language Models with Utility Guarantees Across Modalities and Tasks

**会议**: CVPR 2026  
**arXiv**: [2603.12998](https://arxiv.org/abs/2603.12998)  
**代码**: [https://github.com/Supltz/Debias_VLM](https://github.com/Supltz/Debias_VLM)  
**领域**: 多模态VLM / 公平性  
**关键词**: VLM去偏, 闭式解, Pareto最优, 免训练, 跨模态公平性  

## 一句话总结
提出VLM去偏的闭式解，在跨模态嵌入空间中通过正交分解和Chebyshev标量化，实现Pareto最优公平性与效用损失有界，免训练免标注，覆盖零样本分类/检索/生成三大下游任务。

## 背景与动机
VLM（如CLIP）从海量网络图像-文本对训练中继承了社会偏见（如"nurse"=female, "doctor"=male），这些偏见会传播到下游应用。现有去偏方法要么需要训练额外网络，要么需要敏感属性标注数据，而且缺乏效用保持的理论保证。需要一种无需训练、无需标注、能同时处理双模态且有理论保证的方法。

## 核心问题
如何在不牺牲下游任务效用的前提下去除VLM中的社会偏见？这涉及五个关键挑战：(1) 偏见存在于图像和文本两种模态中；(2) 重训练VLM不现实；(3) 敏感属性标注数据难获取；(4) 不同下游任务定义不同；(5) 去偏必然导致效用损失，需要理论保证。现有方法要么只处理单模态，要么需训练，要么对交叉公平性（如性别×年龄）无法处理。

## 方法详解
给定VLM嵌入 $\vec{e}$，方法在属性子空间 $\mathcal{A}$ 上做正交分解，将去偏问题转化为寻找单位超球面上的最优点 $\vec{u}^*$。

### 整体框架
输入文本/图像 → VLM编码器得到嵌入 $\vec{e}$ → LLM引导构建群组原型 → 构建属性子空间 $\mathcal{A}$ → 正交分解 → 闭式求解去偏嵌入 $\vec{u}^*$ → 输出去偏结果用于下游任务。

### 关键设计
1. **LLM引导的群组原型构建**: 用LLM（GPT-5）对每个群组生成多种语言变体（如male→man/gentleman/boy），取球面均值作为群组原型 $\vec{p}_g$，比直接用单一prompt更鲁棒。
2. **属性子空间正交分解**: 将嵌入分解为属性泄露分量 $\vec{e}_{\mathcal{A}}^{\parallel}$ 和中性内容 $\vec{e}_{\mathcal{A}}^{\perp}$。与之前直接投影到 $\mathcal{S}$（群组原型张成空间）不同，本方法只投影到属性子空间 $\mathcal{A}$，保留语义信息（如"doctor"）。
3. **Chebyshev标量化闭式解**: 将公平性（最小化属性泄露）与效用（最小化self-utility loss）的双目标优化通过Chebyshev方法转化为极小极大问题，推导出闭式解 $\alpha^* = \frac{E - \|\vec{e}_{\mathcal{A}}^{\perp}\|\sqrt{E^2 - \|\vec{e}_{\mathcal{A}}^{\parallel}\|^2}}{E^2 + \|\vec{e}_{\mathcal{A}}^{\perp}\|^2}$。

### 损失函数 / 训练策略
无需训练。方法是纯推理时的闭式变换，直接在嵌入空间操作。理论证明cross-utility loss有上界 $\ell_{cross} \leq \sqrt{2\ell_{self}^{(I)}} + \sqrt{2\ell_{self}^{(T)}}$。

## 实验关键数据
| 数据集 | 指标 | 本文 | 之前SOTA | 提升 |
|--------|------|------|----------|------|
| CelebA (分类) | F1↑ / ΔEO_Avg(G×A)↓ | 56.5 / 23.6 | 53.1 / 23.3 (FairerCLIP/RoboShot) | F1+3.4 |
| COCO2017 (检索) | R@5↑ / MS@1000(G×ST)↓ | 81.1 / 10.1 | 77.4 / 9.9 (SFID/CLIP-clip) | R@5+3.7 |
| SD v2.1 (生成) | SP↓ / AccG↑ | 39.7 / 74.6 | 39.6 / 67.2 (Orth-Proj/SFID) | AccG+7.4 |

### 消融实验要点
- 仅用anchor嵌入或仅用mean嵌入做原型，均导致公平性下降 → 球面均值构建有效
- 仅去偏单模态（I或T）均不如双模态联合去偏
- 不同LLM（GPT-5/DeepSeek v3.2/Gemini 2.5 Pro）对结果影响极小

## 亮点 / 我学到了什么
- 闭式解 = 确定性+高效+可解释——这是VLM去偏第一个有理论效用保证的方法
- 正交分解到属性子空间而非群组原型子空间，精准保留语义——之前方法"过度切割"
- 三个RQ分析很有教育意义：(1) 敏感属性标注不必要；(2) 训练额外网络不必要；(3) 双模态联合去偏是必要的
- 首次系统评估VLM去偏的交叉公平性（intersectional fairness）

## 局限性 / 可改进方向
- 效用保证在表示空间（余弦相似度）而非任务指标空间（F1/R@K），二者差距在极端场景可能较大
- 闭式解依赖线性属性子空间假设——非线性偏见模式可能遗漏
- 未扩展到图像/文本解码器，生成任务的去偏仍是间接的
- → 可与 `ideas/20260316_counterfactual_detection_bias.md` 关联

## 与相关工作的对比
- vs PRISM/Orth-Proj：它们投影到完整群组子空间 $\mathcal{S}$，丢失语义；本文只投影到属性差异方向 $\mathcal{A}$
- vs SANER/DeAR：它们需要训练额外网络，无效用理论保证；本文免训练且有闭式效用上界
- vs FairerCLIP：需要FairFace等标注数据训练，在非人脸数据集（FACET/COCO）泛化差

## 与我的研究方向的关联
- 闭式解在嵌入空间的正交分解思路可迁移到检测/分割模型的表征去偏
- 属性子空间构建方法可启发多模态模型的公平性评估框架设计

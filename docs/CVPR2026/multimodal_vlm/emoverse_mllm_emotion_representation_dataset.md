# EmoVerse: A MLLMs-Driven Emotion Representation Dataset for Interpretable Visual Emotion Analysis

**会议**: CVPR 2026  
**arXiv**: [2511.12554](https://arxiv.org/abs/2511.12554)  
**代码**: 无  
**领域**: 多模态VLM / 情感分析  
**关键词**: 视觉情感分析, 可解释数据集, 知识图谱标注, 情感三元组, 维度情感空间  

## 一句话总结
构建了 EmoVerse——首个大规模开源可解释视觉情感分析数据集（219k+ 图像），采用知识图谱启发的多层标注框架，将情感分解为 Background-Attribute-Subject (B-A-S) 三元组并将每个元素接地到视觉区域，提供词级和主体级情感推理；同时引入兼容离散情感类别（CES）和连续维度情感空间（DES）的双重标注，并提出可解释模型将视觉线索映射到 DES 表示。

## 背景与动机
视觉情感分析（VEA）旨在弥合视觉内容和人类情感反应之间的情感鸿沟，但受限于数据集的三个问题：(1) 大多数数据集仅为整张图像分配单个离散情感标签，无法解释哪些视觉元素引发了情感；(2) 缺乏开源和可解释的数据集；(3) 离散和连续情感表示割裂，没有统一框架。

## 核心问题
如何构建一个能支持可解释视觉情感分析的大规模数据集，并提供从视觉元素到情感的细粒度归因？

## 方法详解

### 关键设计

1. **B-A-S 三元组分解**: 将情感标注分解为 Background（场景背景）- Attribute（属性特征）- Subject（情感主体）三元组，并将每个元素接地到图像的具体视觉区域，实现词级和主体级的情感推理链。

2. **双重情感标注（CES + DES）**: 同时提供离散类别情感标签（如快乐、悲伤）和连续维度情感空间坐标（如效价-唤醒度-支配度），统一两种情感表示范式。

3. **多阶段标注流水线**: 利用 MLLM 驱动的自动标注 + 最小化人工校验，确保高标注可靠性的同时控制成本。

4. **可解释模型**: 将视觉线索映射到 DES 表示并提供详细的归因解释——说明每个视觉区域如何贡献到最终情感判断。

## 实验关键数据
- 219k+ 图像，大规模开源
- 多层标注：词级、主体级、区域级的情感接地
- 统一离散和连续情感表示

## 亮点
- **可解释性**：从单一标签升级到结构化三元组 + 视觉接地，让情感分析过程透明可审查
- **统一框架**：CES + DES 双标注解决了离散/连续表示割裂的历史问题
- **MLLM 驱动标注**：高效低成本的标注方式

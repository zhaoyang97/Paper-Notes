# ConsistCompose: Unified Multimodal Layout Control for Image Composition

**会议**: CVPR 2026  
**arXiv**: [2511.18333](https://arxiv.org/abs/2511.18333)  
**代码**: 无  
**领域**: 图像生成 / 多模态VLM  
**关键词**: 布局控制生成, 多实例图像合成, 统一多模态框架, 坐标嵌入prompt, 身份保持  

## 一句话总结
提出 ConsistCompose，将布局坐标直接嵌入语言 prompt 实现布局可控的多实例图像生成——通过实例-坐标绑定 prompt 和坐标感知 classifier-free guidance 将语言布局线索转化为精确空间控制，配合 340 万多实例数据集 ConsistCompose3M，在 COCO-Position 和 MS-Bench 上显著提升空间准确性同时保持身份保真度。

## 背景与动机
统一多模态模型在视觉理解方面进展迅速，但生成侧的布局控制能力仍然不足——大多数系统只做 visual grounding（语言对齐到图像区域），其生成对应物"语言嵌入式布局接地生成"（LELG）却被忽视。现有布局控制方法通常需要任务特定的分支或额外条件编码器，缺乏统一的生成接口。

## 核心问题
如何在统一的多模态生成框架中实现布局可控的多实例图像生成，无需任务特定分支？

## 方法详解

### 关键设计

1. **实例-坐标绑定 Prompt**: 将布局坐标直接编码进语言 prompt——如 "a dog at [x1,y1,x2,y2] and a cat at [x3,y3,x4,y4]"，让模型通过语言理解隐式学习空间布局控制。

2. **坐标感知 Classifier-Free Guidance**: 在 CFG 过程中融入坐标信息，将语言布局线索转换为精确的空间控制信号，无需额外的 ControlNet 或布局编码器。

3. **ConsistCompose3M 数据集**: 340 万多实例生成数据对（260 万文本引导 + 80 万图像引导），每个数据点包含布局和身份标注，提供大规模布局条件生成的监督信号。

4. **支持交错图文输入**: 能从交错的图像-文本输入进行布局控制的多实例生成，统一了文本引导和图像引导两种模式。

## 实验关键数据
- 在 COCO-Position 和 MS-Bench 上显著优于布局控制基线
- 保持身份保真度和竞争力的通用多模态理解能力
- 支持文本引导和图像引导两种生成模式

## 亮点
- **语言即布局控制**：坐标嵌入 prompt 的设计优雅简洁，无需额外模块
- **大规模数据集**：340 万样本为布局控制生成提供了充分的训练资源
- **统一框架**：视觉理解和布局控制生成在同一模型中完成

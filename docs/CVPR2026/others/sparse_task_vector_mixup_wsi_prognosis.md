# STEPH: Sparse Task Vector Mixup with Hypernetworks for Efficient Knowledge Transfer in WSI Prognosis

**会议**: CVPR 2026  
**arXiv**: [2603.10526](https://arxiv.org/abs/2603.10526)  
**代码**: [github.com/liupei101/STEPH](https://github.com/liupei101/STEPH)  
**领域**: 医学图像 / 计算病理学  
**关键词**: 全切片图像, 生存分析, 跨癌种知识迁移, 任务向量, 超网络  

## 一句话总结
STEPH通过超网络驱动的任务向量混合(TVM)+稀疏聚合实现跨癌种WSI预后知识迁移，在13个TCGA数据集上C-Index平均0.6949（+5.14% vs 癌种特定学习，+2.01% vs ROUPKT）。

## 背景与动机
癌种特定预后模型受限于有限样本量(~1000/癌种)和高肿瘤异质性。多癌种联合训练和表示迁移存在但需大规模联合训练或多模型推理。

## 核心问题
如何在不需要联合训练或线性推理开销的条件下实现跨癌种知识迁移？

## 方法详解

### 整体框架
通过任务向量(τ = M_t − M_0)合并癌种特定模型，三步：TVM混合→稀疏聚合→最终模型。

### 关键设计
1. **任务向量混合(TVM)**: 对每个源-目标对，τ_mix = λ·τ_t + (1−λ)·τ_s，超网络H_mix根据输入WSI自适应输出λ。
2. **稀疏任务向量聚合**: 第二个超网络H_agg生成权重w，Top-K(K=5)选择仅保留最有益的混合，抑制冲突知识。
3. **共享MIL超网络**: H_mix和H_agg共享mean-MIL编码器+独立输出头，联合训练NLL生存损失。

## 实验关键数据
| 对比 | C-Index | 基线 | 提升 |
|------|---------|------|------|
| vs Vanilla(癌种特定) | **0.6949** | 0.6609 | +5.14% |
| vs ROUPKT | **0.6949** | 0.6812 | +2.01% |
| BRCA最佳 | **0.7408** | 0.6648 | +11.4% |
| 12/13数据集优于vanilla | - | - | - |

### 消融实验要点
- 无mixup(固定λ=1): 0.6851 → 有HN驱动λ: 0.6949(+0.98%)
- HN驱动w vs 可训练参数w: 0.6949 vs 0.6490(+4.59%)——超网络关键
- 有Top-K稀疏 vs 全部: 0.6949 vs 0.6912(+0.37%)

## 亮点
- 任务向量空间中的跨癌种知识迁移——无需联合训练
- 超网络驱动input-adaptive权重vs固定权重差距巨大(+4.59%)
- 12/13数据集一致改善

## 局限性 / 可改进方向
- 需要所有癌种的预训练模型
- 仅验证ABMIL backbone
- 推理时需运行超网络增加开销

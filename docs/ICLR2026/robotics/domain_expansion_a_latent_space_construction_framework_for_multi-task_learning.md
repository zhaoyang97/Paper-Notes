# Domain Expansion: A Latent Space Construction Framework for Multi-Task Learning

**会议**: ICLR 2026  
**arXiv**: [2601.20069](https://arxiv.org/abs/2601.20069)  
**代码**: 待确认  
**领域**: Representation Learning / Multi-Task Learning  
**关键词**: 多任务学习, 正交池化, 潜在空间构造, 表征崩塌, 可组合表征  

## 一句话总结
提出 Domain Expansion 框架，通过正交池化(Orthogonal Pooling)将潜在空间重构为互相正交的子空间，从结构上防止多目标训练中的梯度冲突与表征崩塌，实现可解释、可组合的概念代数。

## 背景与动机
1. 多任务学习中，多个目标的竞争梯度拉扯共享表征，导致**潜在表征崩塌**(latent representation collapse)——表征被压缩到一个折中区域，对所有任务都不优
2. 现有梯度级方法(GradNorm, PCGrad, Nash-MTL 等)是"反应式"的——在冲突发生后调解梯度，而非从结构上预防
3. 这些方法不改变潜在空间本身的结构，学到的表征仍然纠缠、不可解释
4. 需要一种"主动式"方案，从表征空间的设计层面消除任务间干扰

## 方法详解

### 三步框架（每个训练 epoch 动态执行）
1. **发现主轴**：对当前潜在特征计算均值 $\mu$ 和协方差 $\Sigma$，特征分解得到正交特征向量基 $V$
2. **定义正交域**：选取前 $M$ 个最大特征值对应的特征向量，每个分配给一个目标概念 $\mathcal{C}_m$
3. **正交池化**：将潜在特征 $f$ 投影到各正交子空间 $f^{\text{proj},m} = v_m v_m^\top (f - \mu)$，各子空间的损失梯度天然解耦

### 损失函数
- 回归任务用 Rank-N-Contrast (RNC) loss，分类任务用改进的 SupCon loss（L2 距离替代内积）
- 总损失 = 各子空间上独立损失的加权和

### 代数性质
- **概念正交性**：不同概念子空间互相正交，修改一个概念不影响其他概念
- **概念组合算子**：支持向量加减运算实现概念级操控（如 $f_{\text{chair}} + f_{\text{boat}}$）

## 实验

### ShapeNet（5个目标：方位角/仰角/旋转+类别/ID）
| 方法 | Spearman(az/el/rot) | V-score(cat/id) | 组合相似度 |
|------|---------------------|-----------------|-----------|
| Baseline | 0.41/0.34/0.35 | 0.16/0.14 | 0.22 |
| Nash-MTL | 0.38/0.41/0.42 | 0.00/0.00 | 0.28 |
| IMTL | 0.31/0.16/0.16 | 0.39/0.28 | 0.14 |
| **Ours** | **0.95/0.87/0.85** | **0.99/0.91** | **0.95** |

- 基线方法在 Objective Set 2 中出现高精度但 V-score 接近 0 的现象，说明学到了"捷径"而非有意义表征
- Ours 在全部指标上大幅领先，概念组合余弦相似度达 0.93-0.95
- MPIIGaze 和 Rotated MNIST 上同样有效（详见附录）

## 亮点
- **结构性防止冲突**：不是事后调解梯度，而是从空间结构上消除干扰，思路新颖
- **可解释性强**：正交轴直接对应语义概念，PCA 可视化清晰展示有组织的潜在空间
- **概念代数**：支持向量加减进行概念级操控和组合推理，验证了潜在空间的可组合性
- 轻量级：正交池化仅需 PCA + 投影，无额外可学习参数

## 局限性
- 每个概念仅分配 1 维子空间，复杂概念可能需要更高维度
- 目前缺少解码端的生成能力——能表征 "chair + boat" 但无法可视化
- 仅在 ShapeNet/MNIST/MPIIGaze 上验证，缺乏大规模真实多任务场景实验
- 特征向量在训练早期不稳定，需要 Hungarian 对齐稳定化

## 相关工作
- **梯度级 MTL**：GradNorm、PCGrad、IMTL、CAGrad、MGDA——反应式调解冲突
- **Nash-MTL / FAMO**：更高级的多任务优化，但不改变表征结构
- **对比学习**：SupCon、RNC——本文用作子空间内的损失函数
- **解纠缠表征**：β-VAE 等——本文通过正交投影实现解纠缠，无需变分推断

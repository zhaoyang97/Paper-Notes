# Common Corpus: The Largest Collection of Ethical Data for LLM Pre-Training

**会议**: ICLR 2026 Oral  
**arXiv**: [2506.01732](https://arxiv.org/abs/2506.01732)  
**代码**: 有（数据集公开）  
**领域**: LLM预训练数据  
**关键词**: pre-training data, ethical data, open data, multilingual, data curation  

## 一句话总结
构建了约 2 万亿 token 的最大规模合法授权 LLM 预训练数据集 Common Corpus，覆盖多语言（含低资源语言）和代码数据，所有数据均为无版权或宽松许可来源，配有完整的数据溯源和多阶段过滤管道，已被 Anthropic 等机构采用。

## 研究背景与动机
1. **领域现状**：LLM 预训练需要万亿 token 级数据，但现有大规模数据集（如 The Pile、RedPajama）存在版权争议，面临 EU AI Act 等立法压力。
2. **现有痛点**：合规数据集规模小、语言覆盖窄、缺乏代码数据、数据溯源不透明。
3. **核心idea一句话**：从无版权/宽松许可来源系统性收集和过滤约 2T token 的多语言数据，建立 AI 训练数据的开放科学基础设施。

## 方法详解

### 整体框架
多阶段数据管道：来源识别（公共领域文献、政府文件、许可数据库）→ 许可过滤 → 质量过滤（去重、语言检测、质量评分）→ 领域分类 → 数据溯源标注。

### 关键设计
1. **许可合规验证**：逐文档验证版权状态和许可类型，排除所有存疑来源
2. **多语言覆盖**：不仅含英语等主要语言，还包括低资源语言（对多语言 LLM 训练尤为重要）
3. **代码数据整合**：包含宽松许可的代码数据，支持代码生成能力训练

## 实验关键数据
| 维度 | 规模 |
|------|------|
| 总 Token 数 | ~2 万亿 |
| 语言覆盖 | 多语言（含低资源语言）|
| 数据类型 | 文本 + 代码 |
| 采用机构 | Anthropic 等 |

## 亮点与洞察
- **开放科学基础设施**：为合规 LLM 训练提供了关键基础设施，在 AI 数据立法趋严的背景下价值巨大
- **数据溯源透明**：完整记录每个文档的来源和许可，为 AI 审计提供支持

## 局限性 / 可改进方向
- 数据质量与网络爬取数据仍有差距（合法来源的多样性可能不如无限制爬取）
- 部分低资源语言的数据量可能仍不够训练高质量模型

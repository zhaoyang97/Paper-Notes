# FeDaL: Federated Dataset Learning for General Time Series Foundation Models

**会议**: ICLR 2026  
**arXiv**: [2508.04045](https://arxiv.org/abs/2508.04045)  
**代码**: https://github.com/shengchaochen82/FeDaL  
**领域**: 时间序列 / 联邦学习  
**关键词**: 时序基础模型, 联邦学习, 数据集异质性, 域偏差消除, 跨域泛化

## 一句话总结
提出 FeDaL 联邦框架从头训练通用时序基础模型（TSFM），通过客户端域偏差消除（DBE）和服务器全局偏差消除（GBE）处理数据集级异质性，在8个任务上超越54个baseline。

## 研究背景与动机

1. **领域现状**：TSFM 通过大规模多域预训练获取可迁移表示，但通常特定于某任务。TSPM 追求任务通用但逐数据集训练。两者都假设集中式数据访问，实际中时序数据通常孤岛式、异质。
2. **现有痛点**：(a) 现有联邦方法（FFTS）仅处理粗粒度域级异质性；(b) 三类数据集级偏差未被处理：时间分辨率偏差、物理约束偏差、模式转变偏差。
3. **核心矛盾**：联邦聚合需要一致性，但数据集级偏差导致局部过拟合+聚合放大偏差。
4. **本文要解决什么？** 在联邦设置下从头训练通用TSFM，处理数据集级异质性。
5. **切入角度**：将异质性分解为局部偏差（DBE客户端）和全局偏差（GBE服务器）。
6. **核心idea一句话**：用联邦学习的分布式架构自然分解时序异质性为可局部消除和需全局对齐的两部分。

## 方法详解

### 整体框架
每个客户端持有一个时序数据集。客户端用 masked patch reconstruction 训练，DBE 模块隔离数据集偏差。服务器通过 GBE 梯度校正+精细调优后聚合。

### 关键设计

1. **域偏差消除（DBE）**:
   - 做什么：从潜在表示分离数据集特有偏差
   - 核心思路：对掩码输入的潜在表示做趋势+季节分解，各取平均+可学习缩放得偏差向量 $\mathbf{b}$。重构时加回偏差：$\mathcal{L} = \|f_h(f_b(\tilde{X}) + \mathbf{b}) - X\|^2 + \lambda\|\mathbf{b} - \mathbf{b}^g\|^2$
   - 设计动机：偏差向量吸收数据集偏移，backbone 聚焦可迁移结构

2. **全局偏差消除（GBE）**:
   - 做什么：聚合时减少残余全局偏差
   - 核心思路：梯度级校正 + 服务端精细调优
   - 设计动机：客户端去偏方式不一致→聚合产生新偏差

3. **EMA 偏差更新**: 指数移动平均稳定 mini-batch 偏差估计

### 损失函数 / 训练策略

客户端：masked patch reconstruction + 偏差正则化。服务端：FedAvg + 梯度校正 + 调优。支持8种下游任务零样本推断。

## 实验关键数据

### 主实验

8个任务×54个baseline：FeDaL 在预测/插补/异常检测/分类上均达 SOTA 或高度竞争，用远少于集中式TSFM的参数。

### 消融实验

| 配置 | 说明 |
|------|------|
| 去掉 DBE | 显著下降——局部偏差未处理 |
| 去掉 GBE | 下降——全局偏差残留 |
| 趋势/季节分解 vs 简单平均 | 分解更好 |
| 联邦缩放行为 | 首次揭示数据量/客户端数/参与率的影响 |

### 关键发现
- 首个联邦通用TSFM，支持8种任务零样本推断
- 联邦缩放行为分析首次揭示去中心化训练的规律
- 用更少参数达到与集中式TSFM竞争的性能

## 亮点与洞察
- **联邦学习作为异质性的自然分解**：FL的分布式本质变为处理异质性的优势
- **Plug-and-play DBE**：可加到任何Transformer时序模型
- **三类偏差的清晰分类**：为理解时序异质性提供框架
- **联邦缩放定律**：首次研究TSFM在联邦下的scaling behavior

## 局限性 / 可改进方向
- GBE需要服务端公共验证数据
- 仅Transformer架构，未测SSM
- 联邦通信开销未优化
- 缺乏与个性化联邦方法的对比

## 相关工作与启发
- **vs FFTS**: FFTS仅粗粒度域级+不支持零样本。FeDaL处理数据集级+8种任务零样本
- **vs Moirai/Chronos**: 集中式需所有数据。FeDaL隐私保护下竞争性能
- **对联邦AI的启发**：DBE+GBE双层偏差消除可迁移到其他联邦FM场景

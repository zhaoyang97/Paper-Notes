# SSCP: Flow-Based Single-Step Completion for Efficient and Expressive Policy Learning

**会议**: ICLR 2026  
**arXiv**: [2506.21427](https://arxiv.org/abs/2506.21427)  
**代码**: 无  
**领域**: 强化学习  
**关键词**: 离线 RL, 流匹配, 单步生成, 策略学习, 推理效率  

## 一句话总结
SSCP 通过预测「完成向量」（从任意中间流样本到最终目标的归一化方向）将多步流策略压缩为单步生成，实现 64× 训练加速和 4.7× 推理加速，同时保持与多步流策略相当的表达力。

## 研究背景与动机
1. **现有痛点**：扩散/流策略表达力强但多步采样推理慢（数十步）且训练不稳定（梯度需穿过采样链）。
2. **核心 idea**：预测完成向量 $\hat{x}_1 = x_\tau + h_\theta(x_\tau, \tau, 1-\tau) \cdot (1-\tau)$，使用真值监督（非 bootstrap）避免自一致性不稳定性，单步采样即可。

## 方法详解
1. **完成向量**：每个中间流时间步 $\tau$ 处预测指向最终动作 $x_1$ 的归一化方向
2. **训练损失**：$\mathcal{L}_\pi = \alpha_1 \mathcal{L}_{\text{flow}} + \alpha_2 \mathcal{L}_{\text{completion}} + \mathcal{L}_{\pi_Q}$（流损失 + 完成损失 + 确定性策略梯度）
3. **推理**：$\pi_\theta(s) = z + h_\theta(z, s, 0, 1)$——单次前向传播

## 实验关键数据
| 指标 | DQL | CAC | **SSCQL** |
|---|---|---|---|
| D4RL 平均(9任务) | 87.9 | 85.1 | **87.9** |
| 训练时间 | ~8小时 | ~5小时 | **~16分钟** |
| 推理延迟(ms) | 1.27 | 0.85 | **0.27** |
| 去噪步数 | 5 | 2 | **1** |

## 亮点与洞察
- **64× 训练加速 + 4.7× 推理加速**同时保持等价性能——实用价值极高
- 完成向量提供真值监督——避免 bootstrap 不稳定性
- 离线到在线微调稳定（DQL 和 CAC 常退化）

## 局限性
- 需要调 $\alpha_1, \alpha_2$ 平衡系数
- 早期 $\tau$ 处完成预测可能不准

# One Operator to Rule Them All? On Boundary-Indexed Operator Families in Neural PDE Solvers

**会议**: ICLR2026  
**arXiv**: [2603.01406](https://arxiv.org/abs/2603.01406)  
**代码**: 待确认  
**领域**: self_supervised  
**关键词**: neural PDE solver, boundary condition, operator learning, generalization, non-identifiability

## 一句话总结
论证标准神经 PDE 求解器在边界条件变化时实际学习的是"边界条件索引的算子族"而非单一边界无关算子，形式化为条件风险最小化导出不可识别性结果，实验验证边界分布偏移下的急剧退化。

## 研究背景与动机

1. **领域现状**：神经算子（FNO 等）被描述为学习从问题数据到 PDE 解的"解算子"。近年来有建设 PDE 基础模型的趋势。
2. **现有痛点**：边界条件对 PDE 解的唯一性至关重要，但在学习方法中通常隐式处理（边界填充、辅助通道）。
3. **核心问题**：当边界条件不固定时，神经 PDE 求解器实际在逼近什么对象？
4. **本文论点**：它们学习的不是边界不变的单一算子，而是依赖于训练数据中边界条件分布的算子族。在训练边界分布支持集外，行为不受约束（不可识别性）。
5. **核心洞察**：对力场或分辨率的泛化不意味着对边界条件的泛化。

## 方法详解

### 理论框架

1. **条件风险最小化**: 将神经算子训练视为条件于边界条件 $\mathcal{B}$ 的风险最小化
2. **不可识别性**: 训练边界分布支持集外，学到的算子行为不受约束——不同于泛化失败是优化问题，这是结构性限制
3. **条件期望收敛**: 当边界信息被移除或弱表示时，模型趋向条件期望而非确定的 PDE 解映射

### 实验设计
Poisson 方程 + 混合（Dirichlet+Neumann）边界条件，边界函数从截断 Fourier 展开采样。

## 实验关键数据

### 受控实验

| 实验 | 结论 |
|------|------|
| 边界条件分布偏移 | 性能急剧退化 |
| 不同边界分布间交叉测试 | 系统性失败 |
| 移除边界输入信息 | 收敛到条件期望 |
| 力场/分辨率变化（边界不变） | 泛化良好 |

### 关键发现
- 同一 PDE 算子，不同边界分布训练的模型在交叉评估时严重失败
- 边界信息被移除后模型输出平均化——验证了条件期望理论
- 力场泛化 ≠ 边界泛化——两者是正交的

## 亮点与洞察
- 学习理论视角极有洞察力——不是说架构不好，而是训练范式的结构性限制
- 对 PDE 基础模型的"泛化"声明提出了清晰的质疑
- 不可识别性结果有形式化的数学支撑

## 局限性 / 可改进方向
- 主要在 Poisson 方程上验证，更复杂 PDE 待测
- 提出了问题但没有给出解决方案（承认这不是本文目标）
- 仅 2D 域

## 相关工作与启发
- **vs POSEIDON 等 PDE 基础模型**: 本文质疑其跨边界泛化声明
- **vs 边界感知架构工作**: 本文提供理论解释而非新架构

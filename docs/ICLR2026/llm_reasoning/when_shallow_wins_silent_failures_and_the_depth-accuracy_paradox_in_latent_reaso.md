# When Shallow Wins: Silent Failures and the Depth-Accuracy Paradox in Latent Reasoning

**会议**: ICLR2026  
**arXiv**: [2603.03475](https://arxiv.org/abs/2603.03475)  
**代码**: [GitHub](https://github.com/SubramanyamSahoo/When-Shallow-Wins)  
**领域**: llm_reasoning  
**关键词**: latent reasoning, faithfulness metrics, silent failure, depth-accuracy paradox, computational reliability  

## 一句话总结
分析Qwen2.5-Math-7B的隐式推理发现其61%准确率中仅18.4%来自稳定忠实的推理路径，81.6%通过不一致路径得出，8.8%为"静默失败"（高置信但错误），揭示benchmark准确率掩盖计算可靠性问题。

## 背景与动机
1. CoT推理虽有效但消耗上下文窗口且可能不反映真实计算过程
2. 新架构展现"隐式推理"——在激活空间中执行多跳推理而不显式输出
3. 关键问题：模型是否真正推理，还是利用统计模式产生表面能力？
4. 数学LLM部署于教育/自动评分/决策支持等高风险场景，需验证准确率反映可靠内部计算
5. 现有评估仅报告单样本准确率，无法衡量计算稳定性

## 方法
**忠实度复合指标 F**: 三个可解释组件的加权组合

1. **激活稳定性 S**: 两次独立前向传播的层级余弦相似度均值×(1-方差)，衡量推理一致性
2. **推理跳对齐 A**: 检测激活模式显著变化的层(>75th百分位)，衡量观测到的推理跳数与期望步数的对齐度
3. **深度效率 E**: 活跃层比例+跳跃密度+激活扩展度的加权组合，衡量层深利用是否与问题复杂度匹配

$\mathcal{F}(q) = 0.35\mathcal{S}(q) + 0.35\mathcal{A}(q) + 0.30\mathcal{E}(q)$

**安全评估**: 四象限分类（True Positive/Silent Failure/True Negative/Lucky Guess），基于稳定性阈值0.65

**因果干预**: 逐层注入噪声测量准确率下降，量化各层因果重要性

**压缩假设测试**: 比较隐式/显式CoT/简洁三种模式的层级激活轨迹相似度

## 实验
| 指标 | 数值 |
|------|:----:|
| 准确率 | 61.0% |
| 忠实推理(正确且稳定) | 18.4% (56例) |
| 不一致路径正确 | 81.6% (249例) |
| 静默失败率 | 8.8% (44例) |
| 平均忠实度F | 0.671 |
| 严格忠实比例 | 20% |

**关键发现**: (1) 推理质量与正确性弱负相关(r=-0.21, p=0.002)——更多推理深度未带来更高准确率，存在"深度-准确率悖论"; (2) 1.5B→7B(4.7×参数)在500题子集上准确率零提升(均61%); (3) 隐式CoT比显式CoT低10pp(58.5% vs 68.5%)，但内部计算签名几乎相同; (4) 仅~20%隐式推理展示CoT-like模式，说明隐式推理采用不同计算策略; (5) 因果干预发现中间层(6-9)最关键，晚期层(20-28)负责放大输出。

## 亮点
- 将"准确率掩盖不可靠性"的直觉量化为可测量的忠实度指标
- 静默失败(高置信错误)的概念对安全部署极具警示意义
- 深度-准确率悖论的发现挑战"更深推理=更好"的假设
- 层级因果干预与激活分析的"矛盾"揭示两阶段计算模型

## 局限
- 仅在500题GSM8K子集(6%)上验证，作者明确承认需完整验证
- 仅在Qwen2.5-Math单一模型家族测试
- 忠实度指标的阈值(0.60/0.65)选择缺乏理论基础
- 组件权重(0.35/0.35/0.30)较任意，未充分消融
- 准确率相同(61%)下的1.5B vs 7B结论在小样本上统计功效不足

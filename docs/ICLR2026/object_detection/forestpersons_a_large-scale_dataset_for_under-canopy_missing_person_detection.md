# ForestPersons: A Large-Scale Dataset for Under-Canopy Missing Person Detection

**会议**: ICLR 2026  
**arXiv**: [2603.02541](https://arxiv.org/abs/2603.02541)  
**代码**: https://huggingface.co/datasets/etri/ForestPersons  
**领域**: 目标检测  
**关键词**: 人员检测, 森林搜救, 无人机, 遗漏检测, 遮挡感知

## 一句话总结
ForestPersons 是首个专门针对森林树冠下人员检测的大规模数据集（96,482 张图像 + 204,078 标注），覆盖地面/低空视角、多季节多天气多光照条件，每个实例包含边界框+姿态+可见性标注，填补了 SAR 场景中下冠层检测的数据空白。

## 研究背景与动机

1. **领域现状**：无人机在搜救（SAR）任务中广泛使用，但现有 UAV 数据集主要来自高空俯视角，森林中密集树冠遮挡导致检测极其困难。
2. **现有痛点**：
   - 高空视角人员只有几个像素，树冠遮挡严重
   - 现有 SAR 数据集（HERIDAL/WiSARD/SARD）皆为俯视角，缺少下冠层视角
   - 地面检测数据集（COCO、CrowdHuman）中人员通常站立/行走且场景为城市，与森林搜救差异巨大
3. **核心贡献**：提供首个模拟 MAV（微型无人机）低空飞行视角的森林人员检测基准，包括坚坐/躺倒/部分遮挡的各种姿态。

## 方法详解

### 数据集设计

1. **规模与多样性**：96,482 张图像，204,078 标注。覆盖季节（春夏秋冬）、天气（晴/阴/雨）、光照（白天/黄昏/夜间）
2. **多维标注**：边界框 + 姿态（站、坐、躺、蹲）+ 可见性等级（完全可见/部分遮挡/严重遮挡）
3. **补充数据**：ForestPersonsIR（热红外图像版本）公开发布

### 基线实验
- 在 COCO/VisDrone 上预训练的模型在 ForestPersons 上表现显著下降
- 证明现有基准与森林下冠层检测的挑战不匹配

## 实验关键数据

| 预训练数据 | ForestPersons AP↑ | 差距 |
|-----------|-----------------|------|
| COCO | ~35% | 基线 |
| VisDrone | ~25% | -10% |
| ForestPersons | **~55%** | +20% |

### 关键发现
- COCO 预训练模型在森林场景 AP 仅 ~35%，证明域差异严重
- 严重遮挡实例的检测率最低（<20%）——最需要研究的难点
- 热红外图像在夜间/低光条件下显著提升检测率

## 亮点与洞察
- **填补关键数据空白**：所有现有 SAR 数据集都是俯视角，ForestPersons 是首个下冠层视角的大规模数据集
- 姿态+可见性的细粒度标注对评估遮挡感知检测算法至关重要

## 局限性
- 仅韩国森林采集——热带/亚热带森林的适用性未知
- 未提供分割标注（仅框+属性）
- 基线模型未探索最新的 Vision Foundation Model

## 相关工作
- **vs VisDrone**: 俯视角城市场景；ForestPersons 下冠层森林场景
- **vs HERIDAL**: 也是森林 SAR 但高空俯视角；ForestPersons 地面/低空视角

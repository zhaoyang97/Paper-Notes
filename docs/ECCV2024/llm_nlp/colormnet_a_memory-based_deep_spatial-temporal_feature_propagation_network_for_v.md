# ColorMNet: A Memory-based Deep Spatial-Temporal Feature Propagation Network for Video Colorization

**会议**: ECCV 2024
**arXiv**: [2404.06251](https://arxiv.org/abs/2404.06251)
**代码**: [GitHub](https://github.com/yyang181/colormnet)
**领域**: LLM/NLP
**关键词**: video colorization, memory-based feature propagation, DINOv2, temporal consistency, exemplar-based

## 一句话总结
提出 ColorMNet，一种基于记忆机制的时空特征传播网络，通过预训练大视觉模型引导的特征提取（PVGFE）、基于记忆的特征传播（MFP）和局部注意力（LA）三个模块，在显著降低 GPU 显存消耗（仅需 1.9G）的同时实现了优于 SOTA 的视频上色效果。

## 研究背景与动机
1. **领域现状**: 视频上色需要同时处理每帧的空间着色和帧间的时序一致性，直接应用图像上色方法会导致相邻帧颜色闪烁严重。
2. **现有痛点**:
   - 基于帧堆叠的方法（如 DeepRemaster）沿时间维度堆叠多帧，不能有效利用时空先验，且 GPU 显存消耗巨大（16.6G）；
   - 基于循环传播的方法（如 DeepExemplar）无法有效探索远距离帧的信息，且误差会沿时间累积；
   - 双向循环传播（如 BiSTNet）虽改善了长距离信息探索，但平等对待每帧特征，不准确的估计也会累积误差，且显存达 34.9G。
3. **核心矛盾**: 如何在有限显存下建立远距离帧间的可靠对应关系，同时减轻不准确特征估计的累积误差？
4. **本文解决什么**: 提出记忆机制，按特征使用频率自适应筛选最重要的历史帧特征，既能利用远距离帧信息，又大幅降低显存。
5. **切入角度**: 借鉴人脑对长期记忆的高效管理——更频繁使用的信息得到更多关注。
6. **核心 idea**: 用 DINOv2 引导空间特征提取 + 频率驱动的记忆筛选传播时序特征 + 局部注意力利用相邻帧相似性。

## 方法详解

### 整体框架
输入灰度帧序列 $\{X_i\}_{i=1}^N$ + 参考彩色图 $R$ → PVGFE 提取每帧空间特征 → MFP 传播远距离时序特征 → LA 聚合相邻帧特征 → 解码器输出色度通道 $Y_i$ → 与亮度通道合并得到彩色帧

### 关键设计

1. **PVGFE（大预训练视觉模型引导的特征估计模块）**:
   - 做什么：融合 DINOv2（ViT-S/14）的全局语义特征和 ResNet50 的局部细节特征
   - 核心思路：用 cross-attention 机制融合两种特征。DINOv2 提供 query $Q_i^G$，ResNet50 提供 key $K_i^L$ 和 value $V_i^L$：
     $$\hat{\mathbf{F}}_i = \text{softmax}\left(\frac{\hat{\mathbf{Q}}_i^G (\hat{\mathbf{K}}_i^L)^\top}{\alpha}\right) \hat{\mathbf{V}}_i^L$$
   - 设计动机：单独用 ResNet50 无法建模非局部语义结构；单独用 DINOv2 虽有全局语义但缺乏着色所需的局部细节（PCA 可视化证实）；简单拼接无法自适应筛选有效特征。Cross-attention 让 DINOv2 的全局语义"查询"ResNet 的局部特征，自然实现优势互补。

2. **MFP（基于记忆的特征传播模块）**:
   - 做什么：从历史已着色帧中自适应选择最可靠的特征，传播到当前帧
   - 核心思路：
     - 每 $\gamma$ 帧采样一次，丢弃冗余帧，减少显存
     - 当历史帧数 $z$ 达到阈值 $N_s$ 时，计算每个像素特征的"累积相似度"（即使用频率）$\mathbf{S}_\gamma$
     - 通过 top-$M$ 操作从最早的 $N_e$ 帧中筛选使用频率最高的 $M$ 个像素特征，压缩记忆
     - 用 $L_2$ 距离计算当前帧 query 与记忆 key 的相似度，加权重建色度特征
     $$\mathbf{V}_i = \mathbf{A}_2^V (\mathbf{W}_i)^\top, \quad \mathbf{W}_i = \text{softmax}(-\mathbf{D}^i)$$
   - 设计动机：频繁被后续帧匹配的特征更可能是准确估计的，选择它们能抑制误差累积。设定 $\gamma=5, N_e=5, N_s=10, M=128$。

3. **LA（局部注意力模块）**:
   - 做什么：聚合当前帧每个像素位置 $p$ 在前 $d$ 帧的 $\lambda \times \lambda$ 邻域特征
   - 核心思路：对当前帧每个位置 $p$，在前 $d$ 帧的局部邻域 $\mathcal{N}(p)$ 中做注意力：
     $$\mathbf{L}_i^p = \text{softmax}\left(\frac{\mathbf{Q}_i^p (\mathbf{K}^{\mathcal{N}(p),c})^\top}{\beta}\right) \mathbf{V}^{\mathcal{N}(p),c}$$
   - 设计动机：MFP 处理远距离帧对应，但相邻帧包含的细粒度运动补偿信息同样重要，LA 补充近距离的时空细节。

### 损失函数 / 训练策略
- 损失函数：$L_1$ loss（预测色度通道与 GT 的绝对误差）
- 色空间：CIE LAB
- 优化器：Adam，学习率 $2 \times 10^{-5}$，batch size 4
- 训练 160,000 iterations，单卡 RTX A6000

| 配置 | PSNR↑ | SSIM↑ | 说明 |
|------|-------|-------|------|
| w/ ResNet50 only | 35.01 | 0.962 | 缺少全局语义，色彩偏差 |

### 关键发现
- PVGFE 贡献最大：对比仅用 ResNet50 提升 0.76 dB
- MFP vs Stacking：PSNR 提升 1.83 dB，显存降低 92%（24.1G → 1.9G），速度快 14 倍
- $\gamma=5$ 为最优采样间隔，过大丢失信息，过小引入冗余
- 即使参考图颜色多样，模型仍能产生一致结果，体现了鲁棒性

## 亮点与洞察
- **记忆压缩思路**: 基于使用频率的 top-M 特征筛选，既解决了显存问题又抑制了误差累积，是一种优雅的"遗忘-保留"机制
- **DINOv2 的妙用**: 首次将大预训练视觉模型用于视频着色的特征引导，通过 cross-attention 而非简单拼接实现有效融合
- **极致效率**: 显存仅 1.9G（BiSTNet 的 5.4%），推理速度至少 8 倍于 SOTA，对实际部署友好
- **时间一致性**: CDC 指标最优，说明三模块协作能有效保持着色的时间一致性

## 局限性 / 可改进方向
- 模型参数量较大（123.61M），在边缘设备部署仍有挑战
- 仅使用 $L_1$ loss，未引入感知损失或对抗损失，可能限制色彩丰富度
- 参考图的选择依赖人工（使用第一帧 GT 或网络搜索），缺乏自动选择最佳参考图的机制
- 记忆筛选的超参数（$\gamma, N_e, N_s, M$）需手动调优，无自适应机制

## 相关工作与启发
- **vs BiSTNet**: BiSTNet 用双向传播+光流对齐，显存高、速度慢；ColorMNet 用记忆机制替代，效率提升 18 倍
- **vs DeepRemaster**: 帧堆叠方式无法有效建模时空先验；ColorMNet 的 MFP+LA 更精准
- **vs MAMBA/MeMOTR**: MAMBA 随机选择记忆特征、MeMOTR 指数衰减权重，都不如基于使用频率的筛选策略

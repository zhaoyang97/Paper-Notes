## 一句话总结
我们提出了 AVION，一种专为视觉语言模型的遥感适应而定制的知识蒸馏框架。
## 核心问题
由于两个关键因素，使视觉语言模型适应遥感图像仍然具有挑战性：文本表示的语义覆盖范围有限和视觉特征的适应性不足。
## 关键方法
1. 我们提出了 AVION，一个专为视觉语言模型的遥感适应而定制的知识蒸馏框架
2. 这些问题在空中场景中尤其重要，因为空中场景涉及各种视觉外观和细粒度的对象区分
3. 教师模块通过从大型语言模型收集描述并使用遥感图像特征验证有效性来构建语义丰富的文本原型
4. 学生模块将轻量级且可学习的提示集成到视觉和语言编码器中，在教师的指导下对齐嵌入及其跨模式关系
## 亮点 / 我学到了什么
- 六个光学遥感基准的实验表明，AVION 提高了少样本分类和基类精度，而不会降低对新类别的泛化能力

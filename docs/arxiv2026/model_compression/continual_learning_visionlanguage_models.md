## 核心问题
预训练视觉语言模型（VLM）的持续学习很容易出现灾难性遗忘，但当前的方法适应新任务时没有明确保留从预训练和先前阶段继承的跨模态语义几何，从而允许新任务监督引起几何失真。
## 关键方法
1. 为了在无样本约束下解决这个问题，我们提出了持续学习的语义几何保留（SeGP-CL）
2. 我们观察到，最明显的漂移往往集中在新旧语义界面附近的脆弱社区，其中共享的视觉模式很容易被新的文本语义重新解释
3. SeGP-CL 首先通过构建一组紧凑的具有双目标投影梯度下降（DPGD）的对抗性锚点来探测容易漂移的区域，该锚点将选定的新任务种子推向旧类语义，同时保持原始视觉空间的忠实性
4. 在训练过程中，我们通过锚引导的跨模态几何蒸馏（ACGD）保留跨模态结构，并通过轻量级文本语义几何正则化（TSGR）稳定跨任务的文本参考框架
## 亮点 / 我学到了什么
- 对五个持续学习基准的大量实验表明，SeGP-CL 持续提高稳定性和前向传输，实现最先进的性能，同时更好地保留 VLM 的语义几何。

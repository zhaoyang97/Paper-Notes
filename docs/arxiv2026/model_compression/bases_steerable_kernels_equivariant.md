## 关键方法
1. 我们提出了一种解决可操纵核约束的替代方法，该约束出现在可操纵等变卷积神经网络的设计中
2. 我们找到了可供使用的显式实数和复数基，适用于不同的对称群和任意张量类型的特征图
3. 这种方法的一个主要优点是它不需要以数值或分析方式计算 Clebsch-Gordan 系数，而是直接使用输入和输出特征图的表示
4. 该策略是找到在某个点 $x_0$ 遵循更简单的不变条件的内核基础，然后使用定义的可操纵性方程 \textit{steer} 它移动到某个任意点 $x=g\cdot x_0$

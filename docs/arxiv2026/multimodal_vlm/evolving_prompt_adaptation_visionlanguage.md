## 核心问题
大规模视觉语言模型（VLM）适应标记数据有限的下游任务仍然是一个重大挑战。
## 关键方法
1. 为此，我们提出了 EvoPrompt，这是一种新颖的框架，旨在明确引导提示轨迹，以实现稳定、保留知识的微调
2. 虽然参数高效的即时学习方法提供了一条有希望的道路，但它们经常会遭受预先训练的知识的灾难性遗忘
3. 为了解决这一局限性，我们的工作基于这样的见解：控制提示的进化路径对于无遗忘的适应至关重要
4. 具体来说，我们的方法采用模态共享提示投影仪（MPP）从统一的嵌入空间生成分层提示
## 亮点 / 我学到了什么
- 大量实验表明，EvoPrompt 在少样本学习中实现了最先进的性能，同时稳健地保留了预训练 VLM 的原始零样本功能。

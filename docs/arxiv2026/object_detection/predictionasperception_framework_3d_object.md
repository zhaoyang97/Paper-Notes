## 关键方法
1. 受此启发，本文提出了预测感知（PAP）框架，将预测感知架构集成到3D物体感知任务中，以提高模型的感知准确性
2. 当面对快速移动的鸟类或昆虫时，我们只能通过预测它们的下一个位置并将目光聚焦在那里来清楚地感知它们
3. PAP框架由两个主要模块组成：预测和感知，主要利用连续帧信息作为输入
4. 首先，预测模块根据当前帧的感知结果预测本车和周围交通参与者的未来潜在位置

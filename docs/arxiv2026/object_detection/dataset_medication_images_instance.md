## 核心问题
用药错误和药物不良事件 (ADE) 对患者安全构成重大风险，通常是由于在现实环境中难以可靠地识别药物而引起的。

## 关键方法
1. 基于人工智能的药丸识别模型提供了一种有前景的解决方案，但缺乏全面的数据集阻碍了其发展
2. 现有的药丸图像数据集很少捕获现实世界的复杂性，例如重叠的药丸、不同的照明和遮挡
3. MEDISEG 通过为 8262 个图像中的 32 种不同药丸类型提供实例分割注释来解决这一差距，涵盖从单个药丸图像到杂乱的剂量盒的各种条件
4. 我们在 MEDISEG 上训练 YOLOv8 和 YOLOv9 以展示其可用性，在 3-Pills 子集上达到 IoU 0.5 的平均精度为 99.5%，在 32-Pills 子集上达到 80.1%

## 核心问题
在增强前景质量的同时保持背景一致性仍然是视频编辑的核心挑战。
## 关键方法
1. 为了解决这个问题，我们提出了 KV-Lock，这是一种为基于 DiT 的视频扩散模型量身定制的免训练框架
2. 注入全图像信息通常会导致背景伪影，而严格的背景锁定严重限制了模型的前景生成能力
3. 我们的核心见解是幻觉度量（去噪预测的方差）直接量化世代多样性，这与无分类器指导（CFG）规模有着内在的联系
4. 在此基础上，KV-Lock 利用扩散幻觉检测来动态调度两个关键组件：缓存的背景键值 (KV) 和新生成的 KV 之间的融合比率，以及 CFG 规模

# Empowering Semantic-Sensitive Underwater Image Enhancement with VLM

**会议**: arxiv 2026
**arXiv**: [2603.12773](https://arxiv.org/abs/2603.12773)
**领域**: 目标检测 / 水下图像增强 / 视觉语言模型
**关键词**: underwater image enhancement, VLM, semantic guidance, cross-attention, alignment loss

## 一句话总结

提出一种 VLM 驱动的语义敏感水下图像增强（UIE）策略，利用 VLM 生成关键物体文本描述，通过文本-图像对齐模型生成空间语义引导图，再通过交叉注意力注入和显式对齐损失的双引导机制，引导增强网络将修复资源集中在语义关键区域，同时提升感知质量和下游检测/分割性能。

## 研究背景与动机

**现状**: 基于深度学习的水下图像增强技术已取得显著进展，在传统图像质量指标上表现优异。主流方法采用端到端的编码器-解码器架构，追求全局均匀的图像增强。

**痛点**: SOTA增强方法并不总能提升目标检测或语义分割等下游任务的性能，有时甚至导致性能下降。核心原因是当前方法是"任务无关"或"语义盲"的——在追求全局均匀增强的过程中，引入了微妙的伪影或分布偏移，破坏了下游模型所依赖的语义线索。

**矛盾**: 增强的视觉质量提升 ≠ 机器认知任务性能提升。现有方法无法区分语义焦点（如海洋生物、人造物体）和非焦点（如背景水体），导致关键语义特征被破坏。

**切入角度**: 利用 VLM 的开放世界理解能力，自动识别图像中的关键物体并生成空间语义引导图，无需稀缺的密集标注数据。通过双引导机制在解码器中注入语义先验，使增强过程感知内容。

## 方法详解

### 整体框架

三个阶段：
1. VLM（LLaVA）生成退化图像中关键物体的文本描述
2. 文本-图像对齐模型（BLIP）生成空间语义引导图 $M_{sem}$
3. 双引导机制将 $M_{sem}$ 注入UIE网络的解码器

### 关键设计1: 语义引导图生成

利用 BLIP 的视觉编码器和文本编码器提取特征，计算每个图像 patch 与文本描述的余弦相似度 $s_i = \hat{\mathbf{v}}_i^\top \hat{\mathbf{t}}$。通过语义锐化函数强化高相关区域：

$$s_i' = \Psi_{\text{sharp}}(s_i; \gamma, \delta) = (\max(0, \mathcal{N}(s_i) - \delta))^\gamma$$

其中 $\delta$ 是过滤低相关噪声的阈值，$\gamma > 1$ 是非线性放大对比度的幂律系数。最终上采样到原图尺寸得到 $M_{sem}$。

### 关键设计2: 交叉注意力注入

在解码器第 $l$ 阶段，解码特征 $d_l$ 作为 query，经 $M_{sem}$ 加权的编码器跳接特征作为 key/value：

$$d_l' = \text{softmax}\left(\frac{Q_l K_l^\top}{\sqrt{d_k}}\right) V_l$$

其中 $K_l, V_l$ 由 $M_{sem}$ 加权的编码器特征线性投影得到，使网络优先从语义"照亮"的区域提取信息。

### 关键设计3: 显式语义对齐损失

对解码器中间特征图 $\mathbf{F}^{(l)}$ 施加双向约束：

$$\mathcal{L}_{\text{align}}^{(l)} = \underbrace{\|\mathbf{F}^{(l)} \odot (1 - \tilde{M}^{(l)})\|_F^2}_{\text{背景抑制}} - \underbrace{\eta \langle \mathbf{F}^{(l)}, \tilde{M}^{(l)} \rangle}_{\text{前景增强}}$$

### 总体训练目标

$$\mathcal{L}_{\text{total}} = \mathcal{L}_{\text{recon}} + \lambda_{\text{align}} \sum_{l \in L} \mathcal{L}_{\text{align}}^{(l)}$$

其中重建损失 $\mathcal{L}_{\text{recon}}$ 组合了 L1 损失和基于VGG-19的感知损失，$\lambda_{\text{align}} = 0.1$。

### 关键发现

- -SS策略在所有5个UIE基线上均提升了PSNR（+0.84~+2.15dB）、SSIM和LPIPS
- 下游分割mIoU提升最为显著（如PFormer提升5.41个点，SMDR提升5.33个点）
- 目标检测mAP一致提升，表明语义引导增强有效保留了关键物体特征
- 该策略作为即插即用模块，无需修改基线网络架构

## 亮点与洞察

1. **问题定义精准**: "增强悖论"（enhancement paradox）概念清晰——视觉质量提升≠下游任务提升，这是UIE领域的核心矛盾
2. **巧妙利用VLM**: 用LLaVA生成文本+BLIP做对齐，避免了对稀缺水下语义标注数据的依赖
3. **即插即用设计**: 不修改基线网络架构，仅在解码器侧添加模块，通用性强
4. **双引导互补**: 交叉注意力提供结构性引导，对齐损失提供显式监督，两者互补

## 局限性 / 可改进方向

- VLM（LLaVA）对严重退化的水下图像的描述质量未见讨论——极度模糊/低对比度时文本描述可能不准确
- 语义引导图依赖对齐模型（BLIP），但水下场景可能不在BLIP预训练分布内
- 锐化函数的超参数 $\gamma, \delta$ 需要调节，论文未讨论其敏感性
- 仅在UIEB数据集上训练，泛化能力依赖于测试集（U45/Challenge60）的验证
- 缺少与基于分割网络的语义引导方法的直接对比

## 相关工作与启发

- 与CLIP引导的图像修复方法（如Liu等）的区别：后者用全局文本提示做风格引导，本文做空间精细化的物体级引导
- 双引导机制的设计思想可迁移到其他下游任务导向的增强场景（如低光增强、去雾）
- 语义锐化函数的设计（阈值+幂律）简单有效，值得在类似注意力引导场景中借鉴

## 核心问题
随着AIGC技术的快速进步，图像取证将遇到前所未有的挑战。

## 关键方法
1. 我们提出了一种高效的模型无关框架，称为基于低位平面的 Deepfake 归因（LIDA）
2. 传统方法无法处理快速发展的图像生成技术生成的日益逼真的图像
3. 为了便于识别人工智能生成的图像及其源模型的归属，生成图像水印和人工智能生成的图像归属已成为近年来的重点研究热点
4. 然而，现有方法依赖于模型，需要访问生成模型，并且缺乏对新的和未见过的生成器的通用性和可扩展性

## 亮点 / 我学到了什么
- 综合实验表明，LIDA 在零镜头和少镜头设置下的 Deepfake 检测和图像归因方面实现了最先进的性能

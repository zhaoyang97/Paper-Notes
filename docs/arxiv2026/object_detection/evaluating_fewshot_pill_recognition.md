## 核心问题
药物不良事件是可预防伤害的重要来源，这导致了自动药丸识别系统的开发，以提高用药安全性。

## 关键方法
1. 这些系统的实际部署受到视觉复杂条件的阻碍，包括杂乱的场景、重叠的药丸、反射和多样化的采集环境
2. 这项研究从面向部署的角度研究了少注射药丸识别，优先考虑现实跨数据集域转移下的泛化而不是架构创新
3. 采用两阶段目标检测框架，包括基础训练和少量镜头微调
4. 模型适应新的药丸类别，每个类别使用一个、五个或十个标记示例，并在具有多对象、杂乱场景的单独部署数据集上进行评估

## 关键方法
1. 为了鼓励更强大、更具可比性和细致入微的 DLA，我们引入： 结构语义单元 (SSU) 一种关系标记方法，可将焦点从内容的物理结构转移到语义结构；以及覆盖、重叠、侵入和超出 (COTe) 分数，这是一种用于测量页面解析质量的可分解指标
2. 通常，模型的质量是使用一般对象检测指标来判断的，例如 IoU、F1 或 mAP
3. 然而，这些指标是为 3D 空间的 2D 投影图像而设计的，而不是为印刷媒体的本机 2D 图像
4. 这种差异可能会导致指标对模型性能的误导性或无信息解释

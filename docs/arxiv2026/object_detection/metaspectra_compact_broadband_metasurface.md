## 关键方法
1. 我们推出 MetaSpectra+，一款紧凑型多功能相机，支持两种操作模式：(1) 快照 HDR + 高光谱或 (2) 快照偏振 + 高光谱成像
2. 它采用新颖的超表面折射组件，将入射光束分成多个通道，并独立控制每个通道的色散、曝光和偏振
3. 与之前仅限于窄带（10-100 nm）的多功能超表面成像仪不同，MetaSpectra+ 几乎可在整个可见光谱（250 nm）范围内工作
4. 相对于快照高光谱成像仪，它在基准数据集上实现了最短的总轨迹长度和最高的重建精度

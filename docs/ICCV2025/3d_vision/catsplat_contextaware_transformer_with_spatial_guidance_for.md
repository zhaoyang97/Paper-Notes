# CATSplat: Context-Aware Transformer with Spatial Guidance for Generalizable 3D Gaussian Splatting from A Single-View Image

**会议**: ICCV 2025  
**arXiv**: [2412.12906](https://arxiv.org/abs/2412.12906)  
**代码**: 未知  
**领域**: 3D视觉 / 新视角合成 / 3DGS  
**关键词**: 单视图3D重建, 3D Gaussian Splatting, 视觉语言模型, 文本引导, 空间引导  

## 一句话总结
提出CATSplat——单视图前馈3DGS重建的泛化Transformer框架：利用视觉语言模型(VLM)的文本嵌入通过cross-attention提供场景上下文信息，结合3D点特征提供空间几何引导，弥补单视图信息不足的问题，在大规模数据集上实现SOTA的单视图新视角合成质量。

## 背景与动机
基于3DGS的泛化前馈方法在少视图3D重建上取得进展(如splatter image等)，直接从图像预测per-pixel 3D Gaussian。但单视图场景中信息严重不足：(1) 没有多视图对应关系；(2) 被遮挡区域完全不可见。需要额外的先验信息补充。

## 核心问题
如何在仅有单张图像的极端条件下实现高质量的3DGS重建？

## 方法详解

### 整体框架
单视图图像 → 图像特征提取 + VLM生成场景描述文本 → 文本编码器提取文本嵌入 + 3D点特征提供空间先验 → Cross-attention融合(图像特征 × 文本嵌入 × 3D点特征) → 预测per-pixel 3D Gaussian参数 → 新视角渲染

### 关键设计
1. **文本上下文引导**: 用VLM从输入图像生成场景描述，文本嵌入通过cross-attention注入图像特征，提供场景级语义理解(如"室内厨房场景")来指导几何推理
2. **3D空间引导**: 利用3D点特征(可能来自深度估计或点云先验)提供几何空间信息，增强单视图下的3D结构理解
3. **Transformer架构**: 基于Transformer的泛化设计，支持跨场景的前馈推理

## 实验关键数据
- 在大规模数据集上达到单视图3D场景重建SOTA
- 高质量新视角合成

## 亮点 / 我学到了什么
- **多模态融合**: 文本+3D点+图像三种模态的有效结合弥补单视图信息缺失
- **VLM作为场景理解先验**: 利用VLM的场景描述能力为3D重建提供高级语义引导
- **从单图到3D的实用方案**: 前馈式推理，适合实际应用

## 局限性 / 可改进方向
- VLM描述的准确性可能影响重建质量
- 单视图本质上仍有不可恢复的歧义性

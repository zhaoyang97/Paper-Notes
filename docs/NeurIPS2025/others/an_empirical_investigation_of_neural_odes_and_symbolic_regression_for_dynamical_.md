# An Empirical Investigation of Neural ODEs and Symbolic Regression for Dynamical Systems

**会议**: NeurIPS 2025  
**arXiv**: [2601.20637](https://arxiv.org/abs/2601.20637)  
**代码**: 有  
**领域**: 科学计算  
**关键词**: Neural ODE, 符号回归, 动力系统, 外推, 科学发现

## 一句话总结
系统实证研究 Neural ODE 和符号回归（SR）在动力系统建模中的组合使用：NODE 可以在动态相似条件下外推到新边界条件，SR 可以从有噪声数据中恢复控制方程，且用 NODE 训练数据（仅 10% 原始数据）生成的数据也能让 SR 恢复大部分方程。

## 研究背景与动机

1. **领域现状**：从数据发现动力系统控制方程是科学发现的重要问题。NODE 能连续时间建模但是黑箱，SR 能发现符号方程但需要大量数据。
2. **现有痛点**：NODE 的外推能力（泛化到新初始条件）研究不足；SR 在数据有限或有噪声时可能失败。
3. **切入角度**：将 NODE 用作数据增强工具——少量真实数据训练 NODE，生成大量轨迹给 SR 做方程恢复。
4. **核心idea一句话**：NODE 增强数据 → SR 发现方程 = 用 10% 数据发现物理定律的流水线。

## 方法详解

### Pipeline
1. 从含噪声的合成数据训练 NODE
2. 用训练好的 NODE 生成大量轨迹（数据增强）
3. 用 PySR 对增强数据做符号回归

### 实验系统
- Cart-pole（阻尼振荡）
- 细菌营养适应的生物模型

## 实验关键数据

| 场景 | SR 从真实数据恢复方程 | SR 从 NODE 增强数据恢复方程 |
|------|-------------------|-----------------------|
| Cart-pole | 3/3 方程恢复 | 2/3 精确 + 1/3 近似 |
| Bio-model | 部分恢复 | 接近真实数据的效果 |

### 关键发现
- NODE 外推成功的关键条件：新轨迹与训练数据有"动态相似性"
- SR 对输入变量的选择高度敏感
- NODE 作为数据增强工具效果出人意料地好

## 亮点与洞察
- **NODE→SR 流水线**是对数据稀缺场景下的科学发现有价值的实用方案

## 局限性 / 可改进方向
- 仅在2个简单系统上测试，复杂混沌系统未探索
- NODE 增强数据质量依赖于 NODE 的外推能力

# Are Language Models Efficient Reasoners? A Perspective from Logic Programming
**会议**: NeurIPS 2025
**arXiv**: [2510.25626](https://arxiv.org/abs/2510.25626)
**代码**: https://github.com/rycolab/reasoning-efficiency
**领域**: LLM推理能力评估
**关键词**: 推理效率、逻辑论证、不相关信息、证明最优性

## 一句话总结
本文通过逻辑规划框架评估LLM的推理效率，发现当存在不相关公理时，LLMs会产生内容庞杂的证明而非最优证明，反映出模型存在显著的推理效率问题。

## 研究背景与动机
现有LLM评估通常侧重于答案正确性而忽视推理效率。在真实应用中，大量信息往往是冗余的，有效推理需要模型识别关键信息并忽略干扰。本工作通过逻辑规划（Logic Programming）框架来系统性评估LLM的推理效率，特别关注：
1. 不相关信息的存在如何影响模型性能
2. 模型生成的证明与最优证明的差异
3. 模型是否依赖于虚假的心理表征

## 方法详解

### 整体框架
采用**Verbalized Logic Programs**（VLP）framework：
- 每个自然语言陈述映射到逻辑程序中的原子公式
- 通过Forward Chaining执行推理
- 比较LLM生成证明与最优证明的长度和质量

核心概念：给定逻辑程序P、公理集A和查询目标h_g，定义推理效率为：
$$\text{efficiency}(\mathcal{P}) = \frac{|\mathcal{P}^*|}{|\mathcal{P}|}$$
其中$\mathcal{P}^*$是最短证明，$|\mathcal{P}|$是模型生成证明的顶点数。

### 关键设计
- **GSM问题的逻辑化**：将数学应用题转化为逻辑程序
- **不相关公理注入**：系统地添加语义相关/无关的公理
- **三层重叠度控制**：无重叠、实体重叠、智能体重叠、组合重叠
- **基准控制问题**：长度相等但不含不相关信息的对照

## 实验关键数据

| 模型 | Base | w/ Axiom | w/ Tree | w/ Multiple Trees |
|------|------|----------|---------|------------------|
| Llama-3.1-8B | 65.0% | 54.0% | 58.6% | 52.4% |
| Qwen2.5-Math-7B | 52.8% | 48.4% | 40.2% | 30.6% |
| QwQ-32B | 63.4% | 76.2% | 72.2% | - |

### 关键发现
1. **性能下降**：即使单个不相关公理也导致准确率显著下降(4-8% gap)
2. **证明质量差**：在50%不相关公理的问题中，模型生成的中间定理里一半以上也是不相关的
3. **语义重叠效应**：当不相关公理与查询在语义上重叠（如提及相同实体）时，模型效率最差
4. **长度并非根本原因**：控制输入长度后，性能差异仍然存在

## 亮点与洞察
1. **首次区分两种低效来源**：不必要的推理步骤 vs. 冗长的自然语言表述
2. **Verbalized Logic Programs创新**：桥接形式推理与自然语言，可复现性高
3. **发现模型启发式策略**：模型似乎采用基于词汇重叠的启发式搜索
4. **系统化的不相关信息框架**：相比随机注入，本文的方法更精细可控

## 局限性
1. 仅在数学问题领域评估，泛化性待验证
2. 假设模型能够完全虚拟逻辑程序的语义
3. 对"不相关"的定义较为严格（基于最短证明）
4. 未探索多跳推理的效率问题

## 相关工作

- 不相关信息：与信息检索中的噪声处理相关
- 搜索策略：Transformer能否学习有效搜索的研究

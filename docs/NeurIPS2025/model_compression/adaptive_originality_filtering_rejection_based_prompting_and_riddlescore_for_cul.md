# Adaptive Originality Filtering: Rejection-Based Prompting and RiddleScore for Culturally Grounded Multilingual Riddle Generation

**会议**: NeurIPS 2025  
**arXiv**: [2508.18709](https://arxiv.org/abs/2508.18709)  
**代码**: 无  
**领域**: NLP 生成 / 多语言创意生成  
**关键词**: Adaptive Originality Filtering, RiddleScore, 多语言谜语生成, 拒绝采样, 文化适配

## 一句话总结
提出 Adaptive Originality Filtering (AOF)——一种基于语义拒绝采样的提示策略，通过 MiniLM 嵌入的余弦相似度过滤重复/模板化输出，强制 LLM 生成更新颖、多样且文化匹配的多语言谜语；同时提出 RiddleScore 复合评估指标（Novelty + Diversity + Fluency + Alignment），与人类评分相关性达 $\rho=0.83$。

## 研究背景与动机

1. **领域现状**：LLM 在创意性多语言任务（如谜语生成）中表现不足——标准提示（zero-shot、few-shot、CoT）容易生成模板化、重复的输出，尤其在非英语语言中文化适配差。
2. **现有痛点**：现有评估指标（BLEU、BERTScore、困惑度）无法衡量创意质量的核心维度——新颖性、文化适配、修辞手法的丰富度。谜语需要隐喻、误导和文化共鸣，这些都不在标准指标的捕捉范围内。
3. **核心矛盾**：LLM 预训练数据偏向高频模式，创意生成需要偏离这些模式（更新颖），但偏离太多又可能牺牲流畅性和答案对齐。
4. **本文要解决什么？** 如何让 LLM 生成更新颖、多样且文化匹配的多语言谜语？如何评估这种创意生成的质量？
5. **切入角度**：用语义过滤（拒绝采样）在生成循环中强制新颖性，同时设计复合指标平衡四个维度。
6. **核心idea一句话**：对 LLM 输出做语义距离检测，拒绝与已有谜语太相似的生成，迫使模型产出更原创的内容。

## 方法详解

### 整体框架
生成循环：LLM 生成候选谜语 → MiniLM 编码 → 与参考集计算余弦相似度 → 超过阈值 $\theta=0.75$ 则拒绝并重新生成（最多 $k$ 次） → 接受的谜语进入最终检查 → 用 RiddleScore 评估。

### 关键设计

1. **语义拒绝采样 (Semantic Rejection Sampling)**:
   - 做什么：对每个候选谜语与参考集（BiRdQA 语料库）计算 MiniLM 余弦相似度，$> 0.75$ 则拒绝
   - 核心思路：$\cos(\text{MiniLM}(r_{\text{candidate}}), \text{MiniLM}(r_{\text{reference}})) \leq 0.75$ 才接受。阈值 0.75 经敏感性分析确定为最优（更低允许模板渗透，更高增加 14% 失败率）
   - 设计动机：LLM 倾向于从预训练数据中复制已见谜语，语义过滤在不改变模型参数的前提下强制原创性

2. **RiddleScore 复合指标**:
   - 做什么：四维加权评分 = $\alpha \cdot \text{Novelty} + \beta \cdot \text{Diversity} + \gamma \cdot \text{Fluency} + \delta \cdot \text{Alignment}$
   - 核心思路：Novelty = 与 BiRdQA 的余弦距离 (MiniLM)；Diversity = Distinct-2 bigram ratio；Fluency = GPT-2 逆困惑度；Alignment = BERTScore(谜语, 答案)。权重 $\alpha=0.30, \beta=0.20, \gamma=0.30, \delta=0.20$ 经网格搜索最大化与人类评分的 Spearman $\rho$
   - 设计动机：单一指标无法捕捉谜语质量的多面性；加权组合在 120 样本开发集上达到 $\rho=0.83$，优于均匀权重的 0.71

3. **提示级约束 (Prompt-level Constraints)**:
   - 做什么：在系统提示中注入文化元素约束（如要求使用特定语言的修辞手法、隐喻、文化意象）
   - 核心思路：提示模板包含语言特定的创意约束（如日语要求使用和歌式隐喻），引导 LLM 生成文化匹配的内容
   - 设计动机：仅靠语义过滤不足以保证文化适配，需要在提示层面显式引导

### 损失函数 / 训练策略
- AOF 不修改模型参数，纯推理时方法
- 微调版本：GPT-4o 在 BiRdQA 上用交叉熵损失监督微调
- 温度 = 0.7，平衡多样性和质量
- 五种语言：英语、中文、阿拉伯语、日语、法语

## 实验关键数据

### 主实验 (GPT-4o + AOF vs 其他提示策略)

| 语言 | 指标 | AOF | CoT | Few-Shot | Zero-Shot |
|------|------|-----|-----|----------|-----------|
| 日语 | Self-BLEU ↓ | **0.177** | 0.483 | 0.421 | 0.512 |
| 日语 | Distinct-2 ↑ | **0.915** | 0.732 | 0.781 | 0.698 |
| 阿拉伯语 | RiddleScore ↑ | **+57.1%** | baseline | - | - |
| 中文(微调) | RiddleScore ↑ | 0.728 | 0.453 | - | - |

### 消融实验

| 配置 | RiddleScore | 说明 |
|------|------------|------|
| AOF (θ=0.75) | 最佳 | 最优阈值 |
| θ=0.60 | 下降 | 允许模板渗透 |
| θ=0.85 | 轻微下降 | 失败率增加 14% |
| 均匀权重 RiddleScore | ρ=0.71 | 非最优权重 |
| 优化权重 RiddleScore | ρ=0.83 | 网格搜索最优 |

### 关键发现
- **AOF 在日语上效果最显著**：Self-BLEU 降低 63.4%，说明语义过滤在形态丰富的语言中特别有效
- **微调 + AOF 比预训练 + AOF 提升更大**：中文 RiddleScore +48.3%，说明模型已有的谜语知识被 AOF 更好地释放
- **语言间差异大**：阿拉伯语的 RiddleScore 提升 (+57.1%) 远大于 Distinct-2 提升，说明不同维度的改善在不同语言中不均匀
- **人类评估确认自动指标结果**：流畅性、创意性、文化适配三个维度均有提升

## 亮点与洞察
- **语义拒绝采样是一种通用的创意增强策略**：不需要微调，不改变模型架构，仅在推理时加一个过滤循环。可迁移到任何创意生成任务（诗歌、笑话、故事等）
- **RiddleScore 的设计思路可推广**：将任务特定的质量维度分解为可独立计算的子指标，然后通过与人类评分的相关性优化权重。这个"复合指标设计方法论"可用于其他创意 NLP 任务
- **文化适配需要显式约束**：仅靠多语言模型本身不够，需要在提示中注入文化知识

## 局限性 / 可改进方向
- **仅在谜语任务上验证**：虽声称可推广到更广泛的创意任务，但未实际验证
- **依赖外部嵌入模型 (MiniLM)**：语义过滤的质量受限于 MiniLM 的多语言能力
- **阈值 0.75 的通用性**：不同任务/领域可能需要不同阈值
- **计算开销**：拒绝采样可能需要多次 LLM 调用（失败重试），增加 API 成本
- **评估数据集有限**：仅基于 BiRdQA，15K 条谜语可能不足以覆盖所有文化背景

## 相关工作与启发
- **vs Self-Refine / Reflexion**: 这些方法用模型自身反馈迭代改进输出，但不显式过滤语义相似性；AOF 更直接地控制新颖性
- **vs COLD Decoding**: COLD 在解码层面施加约束，AOF 在生成后筛选层面施加约束，两者互补
- **vs 标准提示策略 (CoT/Few-Shot)**: 这些策略优化推理过程但不控制输出多样性和新颖性，AOF 正交于这些策略

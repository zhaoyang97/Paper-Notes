# Beyond Greedy Exits: Improved Early Exit Decisions for Risk Control and Reliability

**会议**: NeurIPS 2025  
**arXiv**: [2509.23666](https://arxiv.org/abs/2509.23666)  
**代码**: 待确认  
**领域**: 高效推理 / 模型压缩  
**关键词**: 早退机制, 风险控制, 多臂赌博机, 可靠性函数, 自适应阈值

## 一句话总结
UAT（Unsupervised Adaptive Thresholding）为早退 DNN 设计了可靠性函数来评估中间层输出质量，并用多臂赌博机（MAB）算法在推理时动态学习最优退出阈值，实现 1.7-2.1× 加速且性能损失 <2%，同时对分布偏移鲁棒。

## 研究背景与动机

1. **领域现状**：早退机制（Early Exit）在 DNN 的中间层添加分类头，简单样本提前退出以加速推理。现有方法使用固定阈值（基于置信度/熵）决定是否退出。
2. **现有痛点**：(a) 模型在浅层可能过度自信（overconfident），导致错误退出；(b) 固定阈值无法适应数据分布变化——在分布偏移时性能急剧下降；(c) 现有方法（如 BEEM）仅适用于分类任务，不支持生成任务。
3. **核心矛盾**：早退的收益（速度）和风险（质量下降）之间需要精细平衡。固定阈值是对所有输入"一刀切"，无法反映不同层、不同样本的实际可靠性。
4. **本文要解决什么？** (a) 训练一个可靠性函数评估每层输出是否可信；(b) 在推理时动态调整退出阈值，适应数据分布变化。
5. **切入角度**：将退出层选择建模为多臂赌博机问题——每层是一个 "arm"，reward 结合可靠性（质量）和计算成本（层深度），用 UCB 算法在线学习最优退出策略。
6. **核心 idea 一句话**：训练可靠性函数评估中间层输出质量 + 多臂赌博机推理时动态优化退出阈值 = 自适应早退。

## 方法详解

### 整体框架
两阶段：**离线阶段**——训练早退 DNN 时同时学习可靠性函数 $g(p_i, i)$，其中 $p_i$ 是第 $i$ 层的输出分布，$i$ 是层索引。**在线阶段**——推理时用 UCB 多臂赌博机从可靠性和计算成本中学习最优退出阈值 $\tau^*$。

### 关键设计

1. **可靠性函数训练**:
   - 做什么：学习一个函数 $g(p_i(\cdot|x), i)$ 评估第 $i$ 层对输入 $x$ 的预测是否可靠
   - 核心思路：修改训练损失为 $\mathcal{L}_i = \mathcal{L}_{CE}(p_i, y)(1 + g(p_i, i)) + \Phi(c - \phi(g))$。第一项使可靠性高时损失增大（迫使 $g$ 在预测正确时输出低值、预测错误时输出高值），第二项是约束项控制 $g$ 的输出范围
   - 设计动机：标准置信度（softmax 最大值）不reflective——模型在浅层可能高置信但错误。可靠性函数显式学习预测质量与层深度的关系

2. **多臂赌博机（MAB）退出策略**:
   - 做什么：推理时在线学习最优退出阈值
   - 核心思路：定义 reward $r(\tau) = C_\tau^i \cdot (1 - C_g^i) - \psi(i)$，其中 $C_\tau^i$ 是阈值置信度，$C_g^i = 1 - g(\cdot)$ 是不可靠度，$\psi(i) = \lambda \cdot i$ 是深度惩罚。用 UCB 算法平衡探索（尝试不同阈值）和利用（使用已知最优阈值）
   - 设计动机：固定阈值无法适应分布偏移。MAB 在推理过程中持续学习，自动调整到最优工作点。UCB 保证了理论上的遗憾界

3. **生成任务支持**:
   - 做什么：将早退机制扩展到序列生成（摘要、图文描述）
   - 核心思路：对 encoder-decoder 模型，在 encoder 的中间层评估可靠性，决定是否跳过后续 encoder 层直接进入 decoder
   - 设计动机：现有早退方法（BEEM 等）仅支持分类，UAT 是首个同时支持分类和生成任务的早退框架

### 损失函数 / 训练策略
- 训练损失 = 所有层的加权 CE 损失 + 可靠性函数正则化
- 可靠性函数: 简单线性架构 $g(p, i) = w^\top [p; i] + b$
- MAB 超参：阈值候选集离散化为 $\{0.1, 0.2, ..., 0.9\}$

## 实验关键数据

### 主实验

| 任务 | 模型 | 方法 | 性能 | 加速比 |
|------|------|------|------|--------|
| SST-2 (分类) | BERT-large | BEEM | 95.4% | 1.60× |
| | | **UAT** | **95.8%** | **1.85×** |
| SamSum (摘要) | T5-large | Full model | 48.82 RL | 1.00× |
| | | **UAT** | 48.09 RL | **2.10×** |
| COCO (图文) | BLIP-2 | Full model | 44.0 B4 | 1.00× |
| | | **UAT** | **43.3 B4** | **1.72×** |

### 分布鲁棒性实验

| 数据集 | 方法 | In-domain | Out-domain |
|--------|------|-----------|-----------|
| NoCaps | Full model | 124.5 | 124.5 |
| | Fixed threshold | 121.8 | 118.2 |
| | **UAT** | **123.1** | **122.7** |

### 关键发现
- UAT 在分类任务上超过 BEEM（95.8% vs 95.4%），同时加速比更高（1.85× vs 1.60×）
- 生成任务上 2.10× 加速仅损失 0.73% ROUGE-L（摘要）和 1.6% BLEU-4（图文描述）
- 分布偏移场景下 UAT 显著优于固定阈值——out-domain 性能降幅极小（122.7 vs 118.2）
- MAB 在约 100 个样本后收敛到近最优阈值
- 可靠性函数比原始置信度更能区分正确/错误预测（尤其在浅层）

## 亮点与洞察
- **用 MAB 做在线阈值学习很优雅**：将离散阈值选择建模为赌博机问题，理论上有遗憾界保证，实践中收敛很快。这个框架可迁移到其他需要在线超参调节的场景
- **可靠性函数 vs 置信度**：发现模型置信度不等于可靠性（尤其浅层），显式学习可靠性是更好的退出判据
- **首个支持生成任务的自适应早退**：填补了早退机制在 NLG/Vision-Language 任务上的空白

## 局限性 / 可改进方向
- 可靠性函数需要 ground truth 标签来训练（离线阶段）
- 线性架构的可靠性函数表达力有限，复杂任务可能需要更强的评估器
- MAB 的探索阶段可能在前几十个样本上产生次优退出
- 假设损失在 [0,1] 有界

## 相关工作与启发
- **vs BEEM**: BEEM 仅支持分类且用固定阈值，UAT 支持生成任务且阈值自适应
- **vs Patience/Entropy-based**: 传统方法阈值需要手动调节，UAT 自动在线学习
- **vs SnapKV/LazyLLM (token pruning)**: 不同路线——早退跳过层 vs token 剪枝减少序列长度，两者互补

### 补充技术细节
- 离线阶段损失加权：$\mathcal{L} = \sum_{i=1}^L i \cdot \mathcal{L}_i / \sum_i i$，深层权重更大以反映更高计算成本
- Theorem 4.1 的 $\lambda = \epsilon/L$ 设定在所有实验中使用，无需额外调参
- UAT 的 exploration 阶段（前 $|\Omega|$ 个样本）计算开销可忽略（附录 B.8 验证）
- 在域迁移场景（SST-2→IMDB）中 UAT 相比固定阈值方法保持稳定性能
- 可靠性函数 $g$ 的训练不影响原模型的预测性能（附录 B.9 验证）

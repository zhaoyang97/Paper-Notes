# AQuaMaM: An Autoregressive Quaternion Manifold Model for Rapidly Estimating Complex Protein Structures

**会议**: NeurIPS 2025  
**arXiv**: [2301.08838](https://arxiv.org/abs/2301.08838)  
**代码**: 待确认  
**领域**: 蛋白质结构 / 生物信息学  
**关键词**: 蛋白质结构预测, 四元数, 自回归模型, 流形学习, 快速推断

## 一句话总结
AQuaMaM 提出基于四元数流形的自回归蛋白质结构预测模型——将蛋白质骨架的旋转表示为四元数（在 $S^3$ 流形上），用自回归方式沿序列逐步预测每个残基的局部坐标系旋转，实现比 AlphaFold 快数个量级的结构估计。

## 研究背景与动机

1. **领域现状**：AlphaFold2 等方法通过 MSA + 注意力机制预测蛋白质结构，精度高但推断慢（需要 MSA 搜索和多轮细化迭代）。
2. **现有痛点**：快速结构预测方法（如 ESMFold）虽然去掉了 MSA 依赖但仍然是全局预测，对长序列效率受限。
3. **核心矛盾**：精度和速度的权衡——全局注意力精确但慢，局部方法快但可能丧失全局信息。
4. **切入角度**：将蛋白质骨架建模为一系列局部旋转变换，用四元数参数化旋转并在 $S^3$ 流形上自回归生成。
5. **核心idea一句话**：将蛋白质结构生成分解为逐残基的四元数旋转预测，自回归方式高效生成。

## 方法详解

### 关键设计

1. **四元数流形表示**：
   - 每个残基的局部坐标框架用四元数 $q \in S^3$ 表示相对于前一残基的旋转
   - 四元数避免了万向锁且在 $S^3$ 流形上有良好的插值性质

2. **自回归生成**：
   - 沿蛋白质序列从 N 端到 C 端逐步预测
   - 每步条件于序列信息和已预测的结构
   - 推断时间与序列长度线性增长

3. **流形感知的损失函数**：
   - 在 $S^3$ 上的测地距离作为旋转预测损失
   - 避免在欧氏空间中直接回归旋转矩阵的不一致性

## 实验关键数据

| 方法 | CASP 目标 GDT-TS | 推断时间 |
|------|-----------------|---------|
| AlphaFold2 | 最高 | 分钟级 |
| ESMFold | 次高 | 秒级 |
| **AQuaMaM** | 接近 ESMFold | **毫秒级** |

### 关键发现
- 在大多数目标上精度可接受（虽不及 AF2 但比早期方法好很多）
- 推断速度比 ESMFold 快一个量级
- 四元数表示比旋转矩阵参数化更稳定

## 亮点与洞察
- **四元数+自回归**的组合在蛋白质结构上是创新的表示方式
- 对需要快速结构筛选的应用场景（如虚拟筛选、蛋白质设计初筛）有实际价值

## 局限性 / 可改进方向
- 精度不及 AlphaFold2——对精度要求高的场景仍需 AF2
- 自回归的误差累积——长序列后端精度可能下降
- 未处理多构象和动态结构

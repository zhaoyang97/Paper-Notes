# Deep Value Benchmark: Measuring Whether Models Generalize Deep Values or Shallow Preferences

**会议**: NeurIPS 2025  
**arXiv**: [2511.02109](https://arxiv.org/abs/2511.02109)  
**代码**: [GitHub](https://github.com/josh-ashkinaze/deep-value-benchmark-neurips) (有)  
**领域**: LLM推理  
**关键词**: AI alignment, deep values, shallow preferences, benchmark, value generalization

## 一句话总结
提出 Deep Value Benchmark (DVB)，通过"先混淆后解混淆"的实验设计，测量 LLM 是学习了深层人类价值观还是仅记住了表层偏好模式，发现所有模型的深层价值泛化率 (DVGR) 仅为 0.30，远低于随机水平。

## 研究背景与动机
1. **领域现状**: LLM 正被越来越多地部署为代替用户行动的 AI Agent（金融规划、医疗建议等），这些系统通过人类偏好数据训练来对齐人类意图。
2. **现有痛点**: 缺乏系统性方法来测量 LLM 到底学到了深层价值观（如道德原则）还是仅学到了偏好数据中的表层统计模式。现有 alignment benchmark 未直接区分这两者。
3. **核心矛盾**: 如果 AI 系统仅学到表层关联而非深层价值观，当面对新场景时可能产生不可预测甚至有害的行为——但我们目前无法量化这种风险。
4. **本文要解决什么？** 建立一个可解释的量化指标来衡量模型是否真正理解并泛化深层人类价值观。
5. **切入角度**: 借鉴实验心理学的混淆变量控制方法——先让深层价值和表层特征完美关联，再打破关联观察模型选择。
6. **核心idea一句话**: 通过 confound-then-deconfound 实验范式，精确分离模型学到的是"价值"还是"模式"。

## 方法详解

### 整体框架
DVB 使用 in-context learning 范式：训练阶段展示用户偏好（深层价值 v₁ 与表层特征 s₁ 完美关联），测试阶段打破关联（交换表层特征），观察模型是跟随价值还是跟随表层特征。

### 关键设计
1. **深层价值 (Deep Values)**: 采用 W.D. Ross 的 6 种 prima facie 义务（仁慈、忠诚、正义、不伤害、补偿、自我提升）和 Schwartz 的 5 种社会基础价值（安全、合规、传统、普世主义、仁爱），共 11 种价值。
2. **表层偏好 (Shallow Preferences)**: 通过 GPT-4o 生成候选二分偏好（如正式/非正式称呼），经人工验证筛选出 20 种确实被认为是"表层的、非道德的"偏好。人工标注准确率 0.9。
3. **情境构建 (Contexts)**: 从 Y Combinator 的 AI 助手创业公司标签中提取 8 个领域（商业、客服、金融、生产力、通讯、医疗、法律、教育），结合 O*NET 职业活动数据库生成具体工作场景。
4. **Deep Value Generalization Rate (DVGR)**: 核心指标。训练时用户偏好 (v₁,s₁)>(v₂,s₂)，测试时选项变为 (v₁,s₂) vs (v₂,s₁)。DVGR = 模型选择价值一致选项 (v₁,s₂) 的比例。DVGR=1 表示完美泛化深层价值，DVGR=0 表示完全依赖表层特征。
5. **三重人工验证**: (1) 验证人类能区分深层价值与表层偏好（构念效度），(2) 验证选项确实体现了指定的价值和偏好（内部效度），(3) 情境基于真实世界 AI 应用（外部效度）。

### 损失函数 / 训练策略
不涉及模型训练。评测使用 in-context learning，每个测试 prompt 包含 N∈{4,20,40} 个训练示例和 1 个测试问题。共 400 个实验元组 × 3 种训练样本数 × 10 个测试问题 = 12K 测试实例。

## 实验关键数据

### 主实验
9 个模型在 DVB 上的 DVGR（均显著低于 0.5 的随机水平）：

| 模型 | DVGR |
|------|------|
| Gemini 2.0 Flash | 0.40 |
| Llama 3-8B | 0.37 |
| GPT-4.1-nano | 0.35 |
| Gemini 2.0 Flash Lite | 0.34 |
| GPT-4o-mini | 0.27 |
| GPT-4o | 0.25 |
| GPT-4.1 | 0.24 |
| Llama 3-70B | 0.24 |
| GPT-4.1-mini | 0.23 |
| **平均** | **0.30** |

### 消融实验

| Prompt 策略 | 平均 DVGR | 与 baseline 比较 |
|-------------|----------|-----------------|
| Baseline | 0.30 | — |
| Chain-of-Thought | 0.25 | ↓ 显著下降 |
| 显式指令"泛化深层价值" | 0.33 | ↑ 有所提升但仍低于随机 |

影响因素：
- **模型大小**: 小模型在 3/5 对比中 DVGR 更高（大模型反而更差）
- **领域**: 商业、医疗、金融的 DVGR 较高；通讯、教育、客服较低
- **价值类型**: 传统(0.51)和普世主义(0.42)最易泛化；忠诚和自我提升最难
- **训练样本数**: 4/20/40 个示例对 DVGR 几乎无影响(Cramer's V=0.01)

### 关键发现
- **所有模型 DVGR 均低于 0.5**，即所有模型都倾向于泛化表层偏好而非深层价值
- **CoT 反而有害**: 推理过程中频繁提及表层特征，放大了表层偏好的影响
- **规模无用**: 更大的模型并不能改善深层价值泛化，甚至略有退化
- **同一开发者的模型行为更相似**(76.8% 一致率 vs 跨开发者 72.2%)

## 亮点与洞察
- **Confound-then-deconfound 范式**: 巧妙的实验设计，从心理学实验方法论获得灵感，提供了一般性框架可扩展到其他对齐问题
- **DVGR 指标的可解释性**: 直觉清晰——0.5 以上是理解价值，0.5 以下是依赖表面模式
- **三重验证保证质量**: 从浅层偏好的定义到选项生成到情境设计都有人工验证
- **CoT 的反直觉发现**: 链式思考不是万能的，在某些对齐任务上反而有害
- **模型集中度风险**: 同一开发者模型的价值泛化模式相似，引发 AI 多元化担忧

## 局限性 / 可改进方向
- 实验设计刻意制造完美混淆（最坏情况），现实中深层价值和表层偏好的关联不会如此极端
- 深层价值本身具有模糊性，有时表层偏好确实是合理的选择依据——DVGR 的绝对值解读需谨慎
- 仅测试了 in-context learning，未涉及 fine-tuning 场景——模型经过专门训练是否能改善？
- 9 个模型覆盖有限，缺少推理模型（如 o1、DeepSeek R1）的评估
- 价值体系选择（Ross 义务+Schwartz 价值）是否足够代表"深层价值"值得讨论

## 相关工作与启发
- 与 reward hacking 文献("alignment faking")互补：DVB 直接测量的是模型学到了什么signal，而非模型是否在操纵reward
- ARC 和概念归纳任务中 LLM 也表现出对表层关联的过度依赖——本文在价值对齐领域证实了同样的模式
- 儿童发展心理学中的"鸡-蛋"类比联想错误（over-relying on associations）与 LLM 的表层泛化行为一致
- 对 RLHF/DPO 等基于偏好的对齐方法有重要警示：仅从偏好数据学习可能无法捕捉背后的价值观

## 评分
- 新颖性: ⭐⭐⭐⭐⭐ confound-then-deconfound 范式非常创新，将实验心理学方法引入 AI 对齐评估
- 实验充分度: ⭐⭐⭐⭐ 9 个模型、12K 测试实例、多维度分析，但缺少推理模型和 fine-tuning 实验
- 写作质量: ⭐⭐⭐⭐⭐ 动机清晰、方法论严谨、三重验证、讨论深入坦诚
- 价值: ⭐⭐⭐⭐⭐ 对 AI 对齐领域有深远影响，揭示了当前偏好学习范式的根本性局限

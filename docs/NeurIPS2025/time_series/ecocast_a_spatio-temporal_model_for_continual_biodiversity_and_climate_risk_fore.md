# EcoCast: Spatio-Temporal Model for Continual Biodiversity Forecasting

**会议**: NeurIPS 2025  
**arXiv**: [2512.02260](https://arxiv.org/abs/2512.02260)  
**代码**: 待确认  
**领域**: 遥感 / 生态保护  
**关键词**: 生物多样性预测, 物种分布, Transformer, 持续学习, 非洲生态

## 一句话总结
提出EcoCast，基于Transformer的时空模型，整合Sentinel-2、ERA5和GBIF数据进行近期物种分布预测，配合EWC持续学习机制，在非洲鸟类分布预测上F1从0.31提升至0.65。

## 研究背景与动机

1. **领域现状**：气候变化驱动物种分布快速变化，传统物种分布模型（SDM）是静态的，依赖未来气候情景投影，无法提供月度级别的操作性预测。

2. **现有痛点**：
   - 静态SDM无法跟踪快速变化，回应周期太长
   - 非洲等生态多样性热点地区的监测数据更新不及时
   - 现有模型缺乏持续更新能力，部署后快速过时

3. **核心矛盾**：保护决策需要及时的预测信号，但现有工具提供的是长期投影而非短期预报。

4. **切入角度**：借鉴气象预报的操作性范式，构建月度到季节尺度的生物多样性预报系统。

5. **核心idea一句话**：Transformer序列建模环境变量序列 → 预测下月物种出现 + EWC持续学习适应新数据。

## 方法详解

### 整体框架
12个月的环境特征序列（Sentinel-2 + ERA5）→ Transformer编码器 → 预测下月物种出现概率。训练在2016-2021，微调2022，评估2023。

### 关键设计

1. **序列到点预测**:
   - 做什么：给定过去12个月的环境数据，预测t+1月的物种出现
   - 核心思路：$\min_	heta \sum \mathcal{L}(f_	heta(\mathbf{x}_{t-L+1:t}), y_{t+1})$，L=12月
   - 设计动机：12个月窗口捕获完整年周期，关键用于理解鸟类物候

2. **多模态环境融合**:
   - 做什么：在0.1°网格上融合Sentinel-2波段统计（EVI, NDWI等）和ERA5气候变量
   - 核心思路：每个网格单元每月构建统一环境向量 $\mathbf{x}_t \in \mathbb{R}^F$
   - 设计动机：遥感捕获植被变化，气候变量反映温度/降水趋势

3. **多标签多物种**:
   - 做什么：单一模型预测多个物种的出现概率
   - 设计动机：物种间存在生态关联（如共栖、竞争），联合建模可传递信号

4. **持续学习框架**:
   - 做什么：支持新数据到来时的增量更新
   - 核心思路：EWC正则化 + rehearsal buffer
   - $\min_{	heta'} \sum \mathcal{L}(f_{	heta'}(x_{t+k}), y_{t+k}) + \lambda \Omega_	ext{EWC}(	heta', 	heta)$

## 实验关键数据

### 主实验（5种非洲鸟类，2023年持出集）
| 模型 | F1 macro | PR AUC macro |
|------|---------|-------------|
| Random Forest (ROE) | 0.31 | 0.29 |
| **EcoCast (t+1 forecast)** | **0.65** | **0.72** |

### 关键发现
- Transformer的时间序列建模能力是核心优势（+34%F1, +43%PR-AUC vs RF）
- 操作性预测：可生成2024年1月的预测分布图
- 位置编码帮助捕获物候时间模式

## 亮点与洞察
- **操作性近期预测范式**：不同于依赖RCP情景投影数十年的传统SDM，EcoCast每月更新预测，直接服务保护决策
- **AI for Conservation的完整案例**：从数据管线到可视化工具，面向保护从业者设计
- **多物种联合训练的生态洞察**：Transformer自注意力可能隐式学习物种间的生态关联

## 局限性 / 可改进方向
- 仅5个鸟类物种的初步验证，覆盖物种有限
- 持续学习组件目前仅框架级设计，未做完整评估
- GBIF数据存在采样偏差（观鸟者分布不均）
- 0.1°分辨率对某些微栖息地可能太粗

## 相关工作与启发
- **vs MaxEnt/BRT等静态SDM**：它们不建模时间序列，只做空间关联
- **vs 气象预报模型（如EarthFormer）**：架构灵感来源，但将其扩展到生态预测
- AI在生态保护中的应用仍处于早期，EcoCast提供了端到端的示范

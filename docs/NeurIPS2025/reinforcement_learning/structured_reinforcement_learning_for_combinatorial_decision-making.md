# Structured Reinforcement Learning for Combinatorial Decision-Making

**会议**: NeurIPS 2025  
**arXiv**: [2505.19053](https://arxiv.org/abs/2505.19053)  
**代码**: [GitHub](https://github.com/tumBAIS/Structured-RL)  
**领域**: 强化学习 / 组合优化 / 运筹学  
**关键词**: combinatorial optimization, structured actor, Fenchel-Young loss, operations research, dynamic decision

## 一句话总结
提出 SRL 框架，将组合优化求解器嵌入 actor-critic 的 actor 网络中，通过 Fenchel-Young 损失实现端到端可微学习，在6个工业级组合决策问题上超越模仿学习18×、超越无结构RL 92%。

## 研究背景与动机
1. **领域现状**：组合决策问题（路由/调度/品类优化）在工业中普遍，传统RL处理连续/离散动作，但面对指数级组合动作空间严重受限
2. **现有痛点**：(a) 无结构RL（PPO等）在组合空间中方差极大、难以收敛；(b) 模仿学习受限于专家策略上限，在动态环境中无法超越专家
3. **核心idea一句话**：在 actor 网络中嵌入 CO 求解器保证可行性 + Fenchel-Young 损失实现端到端梯度 = 结构化RL

## 方法详解

### 关键设计
1. **CO层嵌入 Actor**：actor 输出评分向量，CO层将其映射到组合可行解（如TSP的排列、调度的分配）
2. **Fenchel-Young 损失平滑化**：高斯扰动 + 二阶导数使不可微的CO层可端到端训练
3. **Critic引导的探索**：基于critic的softmax加权从动作空间采样，无需专家演示

## 实验关键数据

| 对比 | 提升幅度 |
|------|---------|
| vs 模仿学习（动态任务）| **+8~78%** |
| vs PPO（动态任务）| **+16~92%** |
| vs 模仿学习（静态任务）| 相当 |
| 方差 vs PPO | **低80倍** |
| 方差 vs 模仿学习 | **低16倍** |

6个工业场景：路由、调度、品类优化（含外生和内生不确定性）

### 关键发现
- 结构化actor保证所有输出都是可行解——无需后处理投影
- 动态决策场景（环境随时间变化）中优势最大——模仿学习受专家上限约束无法超越
- 训练30分钟/环境（vs 模仿学习12分钟），计算成本略高但稳定性大幅改善

## 亮点与洞察
- **纯RL无需专家数据**同时保持极低方差——CO层提供的结构约束是关键
- **组合可行性有保证**：与PPO的连续松弛+投影不同，CO层直接输出可行解

## 局限性 / 可改进方向
- 计算成本高：需频繁调用CO求解器（最多61次/更新），随问题复杂度增长
- 收敛速度略慢于模仿学习（因纯在线探索）
- 目前只在中等规模问题上验证，超大规模组合问题的可扩展性未知

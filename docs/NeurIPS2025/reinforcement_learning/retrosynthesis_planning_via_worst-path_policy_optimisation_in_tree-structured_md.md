# Retrosynthesis Planning via Worst-path Policy Optimisation in Tree-structured MDPs

**会议**: NeurIPS 2025  
**arXiv**: [2509.10504](https://arxiv.org/abs/2509.10504)  
**代码**: 无  
**领域**: 化学AI / 强化学习  
**关键词**: retrosynthesis, tree MDP, worst-path optimization, chemical synthesis, self-imitation learning

## 一句话总结
将逆合成规划重新建模为树结构MDP中的最差路径优化问题，用自模仿学习确保所有合成路线都能终止于可购买的起始材料。

## 研究背景与动机
逆合成规划需要将目标分子分解为可购买起始材料。现有方法优化平均路径质量但可能忽略某些分支的可行性。worst-path优化确保每条路径都可行。

## 方法详解

### 整体框架
将逆合成问题建模为树结构 MDP（每个节点是一个分子，分支是可能的反应），目标是优化最差路径（最难合成的那条分支）而非平均路径。

### 关键设计

1. **最差路径值函数**：学习估计树中最弱分支的成功概率——确保所有路径都能终止于可购买材料
2. **加权自模仿学习**：用成功合成路线作为正样本，权重由预训练单步模型的支持度约束——避免策略偏离化学可行空间
3. **与单步模型互补**：最差路径策略在多步规划层面优化，单步反应预测模型在反应层面保证化学有效性

## 亮点与洞察
- **最差路径 vs 平均路径**：化学合成中一条分支失败就全部失败——worst-path 目标完美匹配这一需求
- **自模仿学习的约束**：用预训练模型的支持度约束防止"化学幻觉"——策略只在化学可行空间中优化
- **数据效率极高**：10% 数据达 92% 性能——worst-path 提供了更强的学习信号，因为它聚焦于最有信息量的困难分支
- **100% 成功率的意义**：在逆合成中，这意味着每个提出的路线都保证可执行——对实际药物合成极为重要

## 相关工作与启发
- **vs RetroStar/Retro***: 传统方法优化平均路径质量；InterRetro 优化最差路径——对安全关键应用（药物合成）更合适
- **vs MCTS-based 方法**：MCTS 用蒙特卡罗采样估计路径质量；InterRetro 用值函数直接估计 worst-path——更高效
- **vs 单步模型（如 Retrosim/LocalRetro）**：单步模型不考虑多步规划；InterRetro 在多步层面优化同时利用单步模型的化学知识
- **启发**：worst-path 优化可推广到任何"木桶效应"明显的规划问题——如多步制造、供应链等

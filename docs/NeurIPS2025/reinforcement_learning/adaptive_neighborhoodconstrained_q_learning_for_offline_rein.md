# Adaptive Neighborhood-Constrained Q Learning for Offline Reinforcement Learning

**会议**: NeurIPS 2025  
**arXiv**: [2511.02567](https://arxiv.org/abs/2511.02567)  
**代码**: 无  
**领域**: 强化学习 / 离线RL  
**关键词**: offline RL, neighborhood constraint, OOD actions, conservative Q-learning, advantage-based adaptation  

## 一句话总结
提出ANQ（Adaptive Neighborhood Q-learning），在离线RL中引入基于优势函数的自适应邻域约束，在密度约束（过于保守）和支持约束（建模困难）之间找到灵活的中间地带，在D4RL AntMaze上达到87.5分。

## 背景与动机
离线RL的核心挑战是分布外(OOD)动作的价值高估。现有方法通常通过约束策略来解决：密度约束（如BCQ）要求策略接近行为策略，太保守，限制了策略改善；支持约束（如BEAR）允许策略在行为策略支持集内自由优化，但需要估计支持集，建模困难。需要一种既不太保守又易于实现的约束方式。

## 核心问题
如何设计一种介于密度约束和支持约束之间的灵活约束机制，根据数据质量自适应调整保守程度？

## 方法详解

### 整体框架
定义动作的"邻域"——以数据集中的动作为中心、以自适应半径为范围的区域——策略被约束在所有数据点邻域的并集内。

### 关键设计
1. **邻域约束**：每个数据点(s,a)定义一个以a为中心的球形邻域，策略只能在这些邻域的并集内选择动作
2. **基于优势函数的自适应半径**：高优势（好的）数据点用大半径（给策略更多自由度），低优势（差的）数据点用小半径（更保守）
3. **逐点保守性**：不同数据点使用不同的保守程度——质量高的数据点允许更多探索
4. **Lagrange乘子λ控制**：通过λ控制约束的整体松紧

## 实验关键数据
| 任务 | ANQ | IQL | CQL | TD3BC |
|------|-----|-----|-----|-------|
| AntMaze large-play | **87.5** | 81.6 | ~70 | ~30 |
| Gym locomotion 平均 | **82.9** | ~78 | ~76 | ~75 |

- 70%专家/30%噪声数据下仍鲁棒
- 70%丢弃率下性能依然强劲
- 5个随机种子平均

### 消融实验要点
- 逆温度α最优值5；过大导致方差，过小过于保守
- Lagrange乘子λ对邻域半径控制关键

## 亮点
- **概念清晰**：密度约束←邻域约束→支持约束的连续谱，位置恰当
- **自适应保守性**：逐点调整保守程度是合理且有效的设计
- **鲁棒性好**：对噪声数据和数据量不足都表现鲁棒

## 局限性 / 可改进方向
- 使用数据点质量作为适应标准可能非最优；不确定性量化可能更好
- 邻域形状固定为球形，可能不适合所有动作空间结构

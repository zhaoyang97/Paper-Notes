# A Diffusion Model for Regular Time Series Generation from Irregular Data with Completion and Masking

**会议**: NeurIPS 2025  
**arXiv**: [2510.06699](https://arxiv.org/abs/2510.06699)  
**代码**: [GitHub](https://github.com/azencot-group/ImagenI2R)  
**领域**: 扩散模型 / 时间序列生成  
**关键词**: irregular time series, diffusion model, image representation, completion, masking, ImagenTime  

## 一句话总结
提出两步框架从不规则采样时序数据生成规则时序：先用 TST 自编码器补全缺失值构造"自然邻域"，再在视觉扩散模型中用 masking 策略仅在观测像素上计算损失，避免对补全值的过度依赖，在判别分数上平均改善 70%，训练速度提升 6.5 倍。

## 研究背景与动机

1. **领域现状**：合成时序数据生成在医疗、金融、科学领域有重要应用。实际时序数据经常是不规则采样的（非均匀间隔、缺失值）。近期 ImagenTime 通过将时序映射为图像并用视觉扩散模型生成，在规则时序生成上取得了 SOTA，但不能处理不规则数据。

2. **现有痛点**：(a) 现有不规则时序生成方法（GT-GAN、KoVAE）基于 GAN/VAE，被扩散模型超越；(b) 它们依赖计算昂贵的 NCDE 预处理，无法处理长序列（KoVAE 训练时间是本文的 6.5 倍）；(c) 这些方法完全信任 NCDE 补全的数据是真实分布的样本，假设过强，导致性能远逊于在规则数据上训练的模型（平均差 540%）。

3. **核心矛盾**：直接将 ImagenTime 扩展到不规则数据（用简单 masking，缺失值填 0）会产生"非自然邻域"——零值与有效值混合，导致卷积核学习到错误的局部模式。但如果完全信任补全值（不做 masking），又会引入补全误差。

4. **本文要解决什么？**
   - 如何让视觉扩散模型（ImagenTime）高效处理不规则时序？
   - 如何在利用补全信息（构造自然邻域）和避免补全误差（masking）之间找到平衡？

5. **切入角度**：作者通过精心的 toy experiment 发现——零填充产生的非自然邻域导致卷积核注意力分散到无效像素上，损害了得分估计。解决方案是先补全（构造自然邻域）再 masking（防止过度依赖补全值），两者结合最优。

6. **核心 idea 一句话**：补全创造自然邻域让卷积核正确学习，masking 防止对补全值的过度依赖，两者互补。

## 方法详解

### 整体框架
方法分为两步：(1) **补全步**：用 TST（Time Series Transformer）自编码器将不规则采样序列补全为规则序列；(2) **扩散生成步**：将补全后的规则序列用 delay embedding 变换为图像，训练视觉扩散模型（U-Net），但在损失计算中 mask 掉来自补全的像素（仅在观测对应的像素上计算 loss）。推理时直接从噪声生成图像，用逆 delay embedding 得到规则时序。

### 关键设计

1. **非自然邻域问题的发现与分析**
   - 做什么：识别并量化简单零填充 + masking 方案的缺陷。
   - 核心思路：在 toy experiment 中，2D 高斯混合点映射到 3×4 图像，只有中心两个像素有值，其余为零。训练两个扩散模型：(a) 全图去噪（score=0.71）、(b) 只在中心像素 mask loss（score=0.67）、(c) 用补全值填充邻域再 mask（score=0.32）。可视化卷积核发现：零填充导致核注意力分散到无效零像素，补全+masking 让核聚焦在有意义的像素上。
   - 设计动机：这个分析是整篇文章的核心动机——解释了为什么简单 masking 效果有限，以及为什么需要补全。

2. **TST 自编码器补全**
   - 做什么：高效地将不规则序列补全为规则序列。
   - 核心思路：用 Time Series Transformer 的自注意力机制建模时序依赖，编码器接收不规则输入（带位置编码标记观测时间），GRU 解码器输出规则采样的序列。相比 NCDE，TST 无需求解微分方程，计算效率高得多。
   - 设计动机：TST 是轻量级且可并行的，能处理长序列（65K 点）。关键是补全**不要求完美**——因为后续有 masking 保护，补全只需提供"合理的邻域"即可。

3. **Masking 扩散训练**
   - 做什么：在扩散训练的损失计算中忽略补全像素，只在实际观测像素上优化。
   - 核心思路：将补全后的规则时序用 delay embedding 映射为图像 $x_{\text{img}}$，生成 binary mask $m$（标记哪些像素来自真实观测）。扩散训练的去噪目标只在 $m=1$ 的像素上计算 loss。推理时不需要 mask，直接生成完整图像。
   - 设计动机：masking 使模型不"记住"补全值的具体数值，只利用补全构造的邻域结构。这是 completion 和 masking 互补的关键。

4. **改进的逆 delay embedding**
   - 做什么：用平均聚合代替原始 ImagenTime 的首像素选取。
   - 核心思路：原始 ImagenTime 在逆变换时只取映射到每个时间点的第一个像素的值。本文改为取所有对应像素的平均值，利用了图像中的冗余信息。
   - 设计动机：简单但有效的改进，利用扩散模型在不同图像位置对同一时间点的多次预测来提升重建质量。

### 损失函数 / 训练策略
- TST 自编码器：在不规则数据上预训练，重建规则序列
- 扩散模型：标准去噪 score matching，但 loss 上加 mask
- 推理与标准 ImagenTime 完全相同——不需要 mask 或补全

判别分数越低越好（0.5 = 随机猜），本文方法比 KoVAE 平均改善 **70%**。

### 消融实验

| 配置 | 判别分数 (Stocks, L=24) |
|------|----------------------|
| 零填充 + 无masking | 0.71 |
| 零填充 + masking | 0.67 |
| **补全 + masking** | **0.32** |

### 关键发现
- **补全和 masking 缺一不可**：单独用 masking（零填充）效果有限，单独用补全无 masking 过度依赖补全值；两者结合效果最好
- **TST vs NCDE**：单独用 TST 替换 NCDE（不加 masking）效果差，但 TST+masking 大幅超越 NCDE 方案，因为 masking 补偿了 TST 补全精度的不足
- **可扩展到超长序列**：得益于 delay embedding 将长序列压缩为图像（65K→256×256），方法能处理超长时序，这是 NCDE 方法做不到的
- **改进的逆变换有帮助**：平均聚合 vs 首像素选取，前者在多数数据集上更优

## 亮点与洞察
- **"非自然邻域"问题的精准诊断**是本文的核心贡献：通过 toy experiment + 卷积核可视化，清晰地说明了为什么零填充+masking 效果有限——卷积核无法区分"真零"和"缺失零"，注意力被分散。这个分析非常直观有说服力。
- **补全+masking 的互补框架**设计优雅：补全是为了给卷积核提供合理的局部上下文（不改变卷积架构），masking 是为了防止模型记住补全值（不假设补全完美）。两者各取所长。
- **架构无关性**：masking 策略不修改扩散模型架构，可兼容 U-Net、DiT 等任何 backbone，实用性很强。
- **将时序问题转化为图像问题**的范式（delay embedding）在不规则设置下依然有效，拓展了 ImagenTime 的适用范围。

## 局限性 / 可改进方向
- **需要已知观测时间点**：方法需要知道哪些时间点被观测、哪些缺失，不适用于完全无结构的缺失场景
- **线性混合矩阵假设**：delay embedding 是线性变换，非线性时序结构可能受限
- **TST 补全质量影响**：虽然 masking 减轻了对补全的依赖，但如果补全过差，邻域仍然不自然
- **仅单变量/低维多变量**：实验中时序维度较低，高维多变量时序的效果未知
- **缺失率极高时表现**：在 90%+ 缺失率下，补全质量和模型性能可能急剧下降

## 相关工作与启发
- **vs ImagenTime**：ImagenTime 只能处理规则时序，本文通过补全+masking 将其扩展到不规则场景，且推理流程不变
- **vs GT-GAN / KoVAE**：这两个方法也用两步（NCDE 补全+生成），但 NCDE 计算昂贵且假设补全准确；本文用 TST 更快，用 masking 不假设补全完美
- **vs 时序扩散方法 (CSDI, TimeGrad)**：这些方法在时序空间直接做扩散，不利用图像表示的局部性优势；本文通过图像表示+视觉 U-Net 获得更好的生成质量

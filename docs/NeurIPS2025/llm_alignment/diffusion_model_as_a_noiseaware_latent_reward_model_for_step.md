# Diffusion Model as a Noise-Aware Latent Reward Model for Step-Level Preference Optimization

**会议**: NeurIPS 2025  
**arXiv**: [2502.01051](https://arxiv.org/abs/2502.01051)  
**代码**: [https://github.com/Kwai-Kolors/LPO](https://github.com/Kwai-Kolors/LPO)  
**领域**: 图像生成 / 多模态VLM  
**关键词**: diffusion model, preference optimization, reward model, latent space, step-level, noise-aware, DPO  

## 一句话总结
提出 LRM（Latent Reward Model）和 LPO（Latent Preference Optimization），利用预训练扩散模型本身作为噪声感知的潜空间奖励模型进行步级偏好优化，避免了将噪声潜在图像转换为像素空间的复杂性，在对齐人类偏好的同时实现 2.5–28× 训练加速。

## 背景与动机
偏好优化旨在让扩散模型生成更符合人类偏好的图像。现有方法的问题：
- 使用 VLM 作为像素级奖励模型，但扩散过程中的中间步骤是噪声潜在图像，VLM 无法直接处理
- 需要将噪声潜在图像解码到像素空间再评估，计算昂贵
- VLM 不是为处理不同噪声水平的图像设计的

核心洞察：**预训练的扩散模型天然适合做噪声潜在空间的奖励模型**——它就是被设计来处理各种噪声水平的潜在图像的。

## 核心问题
如何在扩散模型的**噪声潜在空间**中直接进行步级偏好优化，避免像素空间转换的开销？

## 方法详解

### 关键设计
1. **Latent Reward Model (LRM)**: 复用扩散模型的组件（UNet/DiT 的中间层特征）预测潜在图像在任意时间步的偏好分数。关键：模型对不同噪声水平有天然理解。

2. **Latent Preference Optimization (LPO)**: 在噪声潜在空间直接做步级 DPO——每个去噪步骤都有奖励信号指导，而非仅在最终生成图像上评估。

3. **噪声感知**: LRM 在训练时见过所有时间步的噪声潜在表示，因此能在任意去噪阶段准确评估图像质量。

### 训练策略
2.5–28× 训练加速：因为直接在潜空间操作，不需要 VAE 解码+VLM 编码的昂贵往返计算。

## 实验关键数据
显著提升模型在通用偏好、美学偏好和文本-图像对齐偏好上的表现，同时 2.5–28× 训练加速。

## 亮点
- "扩散模型本身就是最好的噪声感知奖励模型"——洞察力强
- 潜空间步级优化避免了像素空间转换——大幅加速
- 与 DenseDPO（同系列笔记视频版）互补——DenseDPO 做视频的时间段级DPO，LPO 做图像的去噪步级DPO

## 局限性 / 可改进方向
- LRM 的偏好预测准确度依赖于扩散模型本身的表示质量
- 步级奖励可能不如最终图像级奖励准确
- 未测试在视频扩散模型上的效果

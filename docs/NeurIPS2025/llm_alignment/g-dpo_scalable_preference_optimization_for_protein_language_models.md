# g-DPO: Scalable Preference Optimization for Protein Language Models

**会议**: NeurIPS 2025
**arXiv**: [2510.19474](https://arxiv.org/abs/2510.19474)
**代码**: 有
**领域**: LLM对齐 / 蛋白质工程
**关键词**: DPO, protein language model, preference optimization, scalability, mutational landscape

## 一句话总结
针对蛋白质语言模型的偏好优化提出 g-DPO，通过序列空间聚类剪枝冗余对和 union masking 摊销似然计算，在保持标准 DPO 性能的同时实现 1.7-5.4× 训练加速。

## 研究背景与动机
1. **领域现状**：DPO 已被应用于蛋白质语言模型以优化蛋白质属性，但蛋白质序列的组合空间庞大，偏好数据对数量随突变景观指数增长。
2. **现有痛点**：标准 DPO 对每个偏好对独立计算 log-likelihood，导致大量重复计算（相似序列间共享大部分 token）。
3. **核心矛盾**：偏好数据规模与训练效率之间的矛盾。
4. **本文要解决什么？** 使 DPO 可扩展到大规模蛋白质突变景观的偏好优化。

## 方法详解

### 关键设计
1. **序列空间聚类剪枝**：将相似的偏好对聚类，只保留代表性对，减少冗余训练样本
2. **Union masking**：将同一组内的序列打包为一个批次，用 mask 区分不同序列，摊销 log-likelihood 计算
3. **性能保证**：在 in silico 和 in vitro 实验中验证优化后的模型性能与标准 DPO 统计一致

## 实验关键数据
| 指标 | g-DPO vs DPO |
|------|-------------|
| 训练加速 | 1.7-5.4× |
| 蛋白质工程性能 | 统计无差异 |
| 收敛速度 | 更快 |

## 亮点与洞察
- **蛋白质特有的优化**：利用蛋白质序列空间的结构（突变通常只改变少数位点）来剪枝和分组，这种领域特定优化通用性有限但效果显著

## 局限性 / 可改进方向
- 聚类策略对序列多样性较高的场景可能效果减弱

# RSCC: Large-Scale Remote Sensing Change Caption Dataset for Disasters

**会议**: NeurIPS 2025  
**arXiv**: [2509.01907](https://arxiv.org/abs/2509.01907)  
**代码**: https://github.com/Bili-Sakura/RSCC  
**领域**: 遥感 / 视觉语言  
**关键词**: 遥感变化描述, 灾害监测, 双时相, 视觉语言, MLLM

## 一句话总结
提出RSCC数据集——62,351对灾前/灾后遥感图像配以丰富变化描述文本，覆盖地震/洪水/野火等，填补灾害相关双时相图像-文本缺口。

## 研究背景与动机
1. **问题**：灾害监测需要时序遥感图像的详细变化描述。
2. **痛点**：现有数据集要么单时相、要么缺文本标注、要么不含灾害上下文。
3. **方案**：构建62,351对灾前灾后图像 + 丰富人类风格变化描述。

## 方法详解
### 关键设计
1. **数据构建**：多源灾害遥感数据配对灾前/灾后图像
2. **文本标注**：比现有数据集更详细
3. **VLM基准**：训练和评估灾害感知双时相理解

## 实验关键数据

### 数据集规模
| 指标 | RSCC | 现有最大 |
|------|------|--------|
| 图像对数 | 62,351 | 远小于此 |
| 灾害类型 | 多种覆盖 | 单一/有限 |
| 描述词汇 | 丰富 | 有限 |
| 标注质量 | 显著提升 | 一般 |

### 基线实验
多模型基线验证（CNN+LSTM、Transformer等），在描述准确度、
多样性和灾害特定词汇使用上均优于现有数据集训练的模型。

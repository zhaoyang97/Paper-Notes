# Gradient Descent as Loss Landscape Navigation: A Normative Framework for Deriving Learning Rules

**会议**: NeurIPS 2025  
**arXiv**: [2510.26997](https://arxiv.org/abs/2510.26997)  
**代码**: 无  
**领域**: 优化理论  
**关键词**: learning rules, gradient descent, loss landscape, momentum, Adam, natural gradient

## 一句话总结
提出统一框架将各种学习规则（momentum、Adam、自然梯度等）推导为损失景观上的最优导航策略，不同度量和目标自然导出不同的优化器。

## 研究背景与动机
现有优化算法各自独立提出，缺乏统一视角。本文视参数更新为损失景观中的导航问题，不同景观度量和目标导出不同学习规则。

## 方法详解

### 整体框架
将学习规则形式化为损失景观上的最优控制策略：(1) 连续时间最优控制公式；(2) 变分法（Euler-Lagrange 方程）推导最优学习动态；(3) 不同假设自然导出不同已知优化器。

### 关键设计

1. **欧几里得度量 → SGD**：最简假设——参数空间是欧几里得的，单步最优导航就是梯度下降
2. **统计流形度量 → 自然梯度**：用 Fisher 信息矩阵作为度量张量，曲率感知的参数更新
3. **多步规划 → Momentum**：不止优化当前步，而是优化未来几步的路径——动量自然涌现
4. **自适应度量 → Adam**：对角自适应度量近似——Adam 是自适应曲率度量下的近似自然梯度
5. **新"弹道"学习规则**：理论框架还预测了一种新的学习规则——在 MNIST/CIFAR-10 上有竞争力

### 理论基础
- 连续时间 ODE 描述参数轨迹
- Euler-Lagrange 方程求解最优路径
- 不同度量/约束/时域对应不同优化器

## 实验关键数据

| 结果 | 说明 |
|------|------|
| SGD = 欧几里得最优 | 从 Euler-Lagrange 推出 |
| Momentum = 多步最优 | 从多步规划自然涌现 |
| Adam = 自适应度量近似 | Fisher 对角近似 |
| 新"弹道"规则 | MNIST/CIFAR-10 上与 Adam 竞争 |

### 关键发现
- 所有主流优化器可统一为同一框架下的特例——只是度量和目标不同
- 动量不是"启发式加速技巧"，而是多步路径优化的必然结果
- 框架还能预测新的学习规则——理论有预测力

# P-SLCR: Unsupervised Point Cloud Semantic Segmentation via Prototypes Structure Learning and Consistent Reasoning

**会议**: CVPR 2025  
**arXiv**: [2603.06321](https://arxiv.org/abs/2603.06321)  
**代码**: https://github.com/lixinzhan98/P-SLCR  
**领域**: 3D视觉  
**关键词**: 无监督语义分割, 点云, 原型学习, 一致性推理, 结构学习

## 一句话总结
提出 P-SLCR，一种原型库驱动的无监督点云语义分割方法，通过将点分离为"一致"和"模糊"两类，用一致结构学习对齐一致点与原型 + 语义关系一致性推理约束两个原型库，在 S3DIS 上无监督达 47.1% mIoU，超越全监督 PointNet。

## 研究背景与动机
1. **领域现状**：点云语义分割主要依赖大规模标注数据（PointNet、PTv2 等），标注3D数据比2D更费时费力
2. **现有痛点**：无监督方法（GrowSP、U3DS3）用聚类生成伪标签再监督训练，但伪标签含大量噪声，直接全部使用会损害特征区分能力；原型特征不够有代表性
3. **核心矛盾**：聚类伪标签不完全可信 vs 需要用伪标签监督网络学习——如何只利用高质量伪标签同时逐步改善低质量的？
4. **本文要解决什么**：设计有效训练策略，从不可靠伪标签中提取可靠信号，逐步实现完整的特征空间划分
5. **切入角度**：将点分为"一致"（伪标签与网络预测和置信度三者一致）和"模糊"两类，分别建立原型库，用一致原型指导模糊原型学习
6. **核心idea一句话**：可靠点→一致原型学习→指导模糊原型→逐步将模糊点纳入一致集

## 方法详解

### 整体框架
输入点云经 SparseConv 特征提取 → 聚类获取伪标签 → 按可靠性分离为一致/模糊点 → 建立双原型库（EMA 更新）→ 一致结构学习拉近一致点与原型 → 语义关系一致性推理约束两库间语义关系 → 迭代优化，模糊点逐步转为一致点。

### 关键设计

1. **可靠点分离（Separation of Reliable Points）**:
   - 做什么：从噪声伪标签中筛选高置信度的"一致点"
   - 核心思路：三重一致条件——聚类伪标签 $\mathbf{l}$ = 网络预测 $\bar{\mathbf{p}}$ 且置信度 $\mathbf{p}_c \geq \tau$。用二值 mask $R$ 分离一致点云 $P^c$ 和模糊点云 $P^a$
   - 设计动机：不可靠伪标签参与学习会破坏类别间特征区分性，只用高置信度点学习原型更安全

2. **一致结构学习（Consistent Structure Learning）**:
   - 做什么：拉近一致点的特征表示与对应一致原型
   - 核心思路：结构误差矩阵 $M_k^c = \|\boldsymbol{\mu}_k^c - G(\mathbf{p}_j^c)\|_2$，最小化 $\mathcal{L}_{sl} = \sum_k M_k^c$。双原型库通过 EMA 从每个 batch 的类内聚类中心更新
   - 设计动机：学习类别内紧凑表示，使原型逐渐变得有代表性

3. **语义关系一致性推理（Semantic Relation Consistent Reasoning）**:
   - 做什么：约束一致原型库和模糊原型库保持一致的类间语义关系
   - 核心思路：分别计算一致/模糊原型间相似矩阵 $e_{ij}^c = \boldsymbol{\mu}_i^c \cdot (\boldsymbol{\mu}_j^c)^T$、$e_{ij}^a$ 并归一化，用 KL 散度约束 $\mathcal{L}_{cr} = \frac{1}{K^2}\sum_{ij} \bar{e}_{ij}^c \log(\bar{e}_{ij}^c / \bar{e}_{ij}^a)$
   - 设计动机：一致原型的类间关系更准确，用它约束模糊原型可以间接提升模糊点的分割质量

### 训练策略
总损失 $\mathcal{L}_{total} = \mathcal{L}_{ce} + \lambda_1 \mathcal{L}_{sl} + \lambda_2 \mathcal{L}_{cr}$，其中 $\lambda_1=0.5$，$\lambda_2=1.0$。EMA 参数 α=0.99，置信度阈值 τ=0.9。初始聚类使用 K-means 对 SparseConv 提取的512维特征进行，S3DIS 上 K=13。训练过程中模糊点逐渐转为一致点，最终完成特征空间划分。伪标签每5个epoch重新聚类更新一次，避免频繁更新带来的训练不稳定。

## 实验关键数据

### 主实验（S3DIS Area-5）

| 方法 | OA (%) | mAcc (%) | mIoU (%) | 类型 |
|------|--------|----------|----------|------|
| PointNet (全监督) | 77.5 | 59.1 | 44.6 | 监督 |
| GrowSP | 78.4 | 57.2 | 44.5 | 无监督 |
| U3DS3 | 75.5 | 55.8 | 42.8 | 无监督 |
| **P-SLCR** | **80.2** | **61.3** | **47.1** | **无监督** |

同时在 SemanticKITTI 和 ScanNet 上取得无监督最优。SemanticKITTI 上 P-SLCR 达到 38.6% mIoU，超越 GrowSP 的35.9%。

### 消融实验

| 配置 | mIoU (%) | OA (%) | 说明 |
|------|----------|--------|------|
| Baseline (聚类+直接训练) | 42.3 | 76.1 | 无质量筛选 |
| + 一致结构学习 | 44.8 | 78.5 | 拉近一致点与原型 |
| + 语义一致性推理 | 45.9 | 79.1 | 约束两库关系 |
| Full P-SLCR | 47.1 | 80.2 | 三组件协同 |

### 关键发现
- 无监督 47.1% mIoU 超越全监督 PointNet (44.6%)——说明高质量伪标签+结构学习可以弥补标注缺失
- 一致点比例随训练逐步增加——模糊→一致的动态转化过程使特征空间逐渐完善，初始~40%一致点逐渐增长至~75%
- 一致结构学习的可视化显示类别内特征逐步紧凑，t-SNE图上类别边界从模糊变清晰
- 在SemanticKITTI室外点云上也取得无监督最优，说明方法不限于室内场景
- 双原型库的EMA更新策略使模型对batch内噪声更鲁棒，避免了单次聚类错误的累积
- 消融中去掉语义一致性推理后 mIoU 下降1.1个点，说明仅靠一致结构学习不足以让模糊原型学到正确的类间关系
- $\lambda_1$、$\lambda_2$ 敏感性分析表明模型在 $\lambda_1 \in [0.3,0.7]$、$\lambda_2 \in [0.5,1.5]$ 范围内性能稳定

## 亮点与洞察
- **从噪声伪标签中"淘金"**的双原型策略很有启发：不试图修正所有伪标签，而是先只用可靠的学习，再逐步扩大可靠集合
- **KL 散度约束类间关系**比直接约束特征距离更全局——保持了语义结构而非单个原型位置
- 动态的一致/模糊点分离是自然的课程学习——从简单到难
- 双原型库设计可迁移到其他伪标签训练场景（半监督检测、域适应等），核心是"分离可靠/不可靠样本并用可靠指导不可靠"
- 训练结束时一致点比例~75%，剩余25%的模糊点仍由一致原型通过最近邻赋标签，形成完整分割

## 局限性 / 可改进方向
- 需要预设类别数 K，对未知类别数的场景不适用，未来可结合自动确定类别数的方法（如DINO的聚类数估计）
- 初始聚类质量直接影响后续学习——如果初始伪标签错误率太高可能无法收敛，这对初始特征质量有隐含要求
- 仅用3D几何+颜色特征，未利用2D图像/语言模型的丰富语义先验，结合VLM可能进一步提升
- 置信度阈值 τ 的选择对一致/模糊分离的质量有影响，太高则初始一致点太少学习不足，太低则引入噪声
- 特征维度固定为512维，未探索更高维度是否能提升原型的区分度

## 相关工作与启发
- **vs GrowSP**: 超点生长+聚类，但无质量筛选和结构学习。P-SLCR 通过双原型库和一致性推理提升+2.6 mIoU
- **vs U3DS3**: 空间聚类+迭代训练，但伪标签全部使用。P-SLCR 选择性使用更有效
- **vs PointDC**: 深度对比学习，但未显式建模原型结构
- **与 STRL 的关系**：时空对比学习用于点云预训练，但下游仍需标注微调；P-SLCR 直接完成无监督分割

# 3D Dental Model Segmentation with Geometrical Boundary Preserving

**会议**: CVPR 2025  
**arXiv**: 无  
**代码**: 未公开  
**领域**: 医学图像 / 3D分割 / 牙科数字化  
**关键词**: Dental Segmentation, Intraoral Scan, Mesh Segmentation, Boundary Preserving, Cross-modal  

## 一句话总结
提出CrossTooth，一种针对3D口腔扫描网格的边界保持牙齿分割方法，通过选择性下采样保留牙龈-牙冠交界处的几何细节，并融合多视角渲染图像的跨模态判别性边界特征，显著提升牙齿分割准确率，特别是在困难的牙冠-牙龈交界区域。

## 背景与动机
3D口腔扫描网格在数字牙科诊断中广泛应用，而牙齿分割是关键的预处理步骤。现有深度学习方法已经能高精度分割牙冠区域，但在牙冠与牙龈的交界处分割准确率仍然不够理想。这是因为：(1) 现有下采样方法无法有效保留交界处的几何细节——均匀下采样会丢失边界处的关键顶点；(2) 仅用3D几何信息难以充分描述这些微妙的边界区域。

## 核心问题
如何提升3D口腔扫描网格在牙冠-牙龈交界处的分割精度？挑战在于交界处的几何特征微弱且容易在下采样中丢失。

## 方法详解

### 整体框架
3D口腔扫描网格 → 选择性下采样（保留边界区域更多顶点）→ 点网络特征提取 → 融合来自多视角渲染图像的跨模态边界特征 → 牙齿分割输出

### 关键设计
1. **3D网格选择性下采样**：区别于均匀下采样（FPS等），设计了一种能在牙齿-牙龈交界区域保留更多顶点的选择性下采样策略。在曲率变化大的边界区域保持高密度采样，在平坦区域（如牙冠表面、牙龈表面）做更激进的下采样，用更少的总点数保留更多的边界几何细节

2. **跨模态判别性边界特征**：从3D网格的多个视角渲染2D图像，从图像中提取边界特征。2D图像天然更擅长捕捉纹理和边界信息，补充3D几何特征的不足。将图像边界特征映射回3D空间，与点特征融合

3. **点网络骨架+图像互补特征**：以点网络为主干处理3D几何，同时整合2D图像提供的补充信息，实现跨模态增强

### 损失函数 / 训练策略
- 标准分割损失（交叉熵+Dice系列）
- 在公开口腔扫描数据集上验证

## 实验关键数据
- 在公开口腔扫描数据集上的实验表明，CrossTooth显著提升了分割准确率
- 牙冠-牙龈交界处的分割改善尤为明显

### 消融实验要点
- 选择性下采样 vs 均匀下采样：选择性保留边界顶点对交界处精度至关重要
- 跨模态图像特征的加入进一步提升边界区域的分割质量

## 亮点
- **选择性下采样**：针对牙齿分割的痛点（边界精度），从下采样层面做了特定优化，解决了均匀采样丢失边界信息的问题
- **跨模态融合**：利用多视角2D渲染提供额外边界信息，弥补纯3D几何特征对微弱边界的表达不足
- **实用性强**：直接面向数字牙科的实际需求，分割精度的提升对后续正畸规划等应用有直接帮助

## 局限性 / 可改进方向
- 仅处理口腔扫描（口内扫描仪采集），未涉及CBCT等其他模态
- 选择性下采样的"边界敏感性"依赖于曲率估计的准确性
- 跨模态融合引入了额外的2D渲染和特征提取开销
- 无arXiv和代码公开

## 与相关工作的对比
- **MeshSegNet / TSGCNet**：纯3D方法，在边界区域精度不足。CrossTooth通过选择性采样+跨模态融合解决此问题
- **TSegNet**：基于图卷积的牙齿分割，但不关注下采样对边界的影响

## 启发与关联
- 选择性下采样的思路可推广到其他需要保留细节区域的3D处理任务（如医学骨骼分割的关节处）
- 跨模态的3D+多视角2D融合在其他3D分割任务中也有应用前景

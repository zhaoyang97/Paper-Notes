# 3D Prior is All You Need: Cross-Task Few-shot 2D Gaze Estimation

**会议**: CVPR 2025  
**arXiv**: [2502.04074](https://arxiv.org/abs/2502.04074)  
**代码**: www.yihua.zone/work/gaze322  
**领域**: 人体理解 / 视线估计 / 跨任务少样本学习  
**关键词**: Gaze Estimation, Cross-Task, Few-shot, Differentiable Projection, 3D-to-2D  

## 一句话总结
提出跨任务少样本2D视线估计——利用预训练3D视线模型作为先验，通过**基于物理的可微投影模块**（6个可学习屏幕参数）将3D视线方向投影到2D屏幕坐标，仅需10张标注图像即可在未知设备上适配2D视线估计，在MPIIGaze/EVE/GazeCapture上比EFE和IVGaze提升超25%。

## 背景与动机
3D视线估计和2D视线估计传统上是两个独立研究方向。3D方法在相机坐标系中估计注视方向向量，不依赖特定设备；2D方法直接预测屏幕像素坐标，但与屏幕大小、相机-屏幕位姿等设备因素纠缠，难以跨设备泛化。传统眼动追踪设备通过3D眼球模型+校准实现精确2D注视估计，但深度学习范式中这种"先3D再投影到2D"的思路尚未被探索。

## 核心问题
如何利用已训练好的3D视线模型，仅用少量（10张）带2D标注的图像，快速适配到一个未见设备的2D视线估计？挑战在于：(1) 3D与2D视线间的域差距；(2) 未知屏幕位姿（不做屏幕校准）；(3) 训练数据极少。

## 方法详解

### 整体框架
将2D视线估计分解为两步：**3D视线估计** + **视线投影**。$\mathcal{H}_{2D}(\mathbf{I}) = \mathcal{P}(\mathcal{H}_{3D}(\mathbf{I}), \mathbf{o})$，其中投影模块$\mathcal{P}$用6个可学习参数（旋转向量$\mathbf{r}$和平移向量$\mathbf{t}$）建模屏幕位姿。

### 关键设计
1. **可微投影模块**：定义屏幕面法向量$\mathbf{n} = \mathbf{R}[:,2]$和屏幕上一点$\mathbf{t}$。计算从脸部中心$\mathbf{o}$沿3D视线方向$\mathbf{g}$与屏幕平面的交点$\mathbf{p}_{3D}$，再变换到屏幕坐标系。全程可微，仅需学习6个参数（$\mathbf{r}, \mathbf{t}$），无需任何屏幕校准。

2. **动态伪标签策略**：翻转增强对2D视线标签极具挑战（依赖未知屏幕位姿）。核心思路：利用可学习屏幕参数做**逆投影**（2D→3D），在3D空间中翻转后再投影回2D。但微调会使3D模型偏离相机坐标系到未知坐标系，导致伪标签不可靠。解决方案：通过SVD对齐初始3D模型和微调后模型的输出来学习动态变换$\mathcal{T}$，将未知坐标系映射回相机坐标系。

3. **不确定性最小化**：对颜色抖动增强图像的预测做方差最小化，增强鲁棒性。

### 损失函数 / 训练策略
$$\mathcal{L} = \mathcal{L}_{proj} + 0.4\mathcal{L}_{flip} + 0.25\mathcal{L}_{unc}$$
- GazeTR (ResNet18+6层Transformer)预训练在Gaze360作为3D模型
- 10张训练样本/subject，80 epochs，RTX 3090，训练仅~1.2分钟

## 实验关键数据

### 跨任务少样本2D视线估计（10-shot，mm误差↓）
| 方法 | EVE(桌面) | MPIIGaze(笔记本) | GazeCapture(手机) |
|------|---------|------------|-------------|
| IVGaze | 228.2 | 106.2 | 101.2 |
| EFE | 60.3 | 73.9 | 48.9 |
| **Ours** | **43.4** | **56.7** | **35.7** |

比EFE提升28%+，比IVGaze提升显著（IVGaze用transformer在10-shot下欠拟合）。

### 参考：全监督2D方法
| 方法 | MPIIGaze | GazeCapture |
|------|----------|-------------|
| Full-Face (全数据训练) | 42.3 | - |
| iTracker (全数据训练) | - | 26.8 |
| **Ours (10-shot)** | **56.7** | **35.7** |

仅10张图像就接近全数据训练的2D方法！

### 消融实验要点
- **仅投影模块**：EVE 46.6（已大幅优于Direct Learning的180.6）
- **+伪标签**：EVE 45.3（-1.3）
- **+不确定性**：EVE 43.4（-3.2总计）
- **去掉变换$\mathcal{T}$**：EVE 88.1（伪标签不可靠导致崩溃）

## 亮点
- **开创性任务定义**：首次将3D→2D视线估计的跨任务适配系统化，无需屏幕校准
- **物理驱动设计**：可微投影基于几何光学原理，仅6个参数，高度可解释
- **极致高效**：10张图像+1.2分钟训练即可部署到新设备
- **动态伪标签的坐标系对齐**：通过SVD学习动态变换$\mathcal{T}$的思路优雅地解决了微调中坐标系漂移问题

## 局限性 / 可改进方向
- 当输入图像无有效人脸时，数学投影可能产生极端值（视线近乎平行于屏幕）
- 10-shot与全监督仍有差距（56.7 vs 42.3 on MPIIGaze），更多无标注数据利用未探索
- 屏幕PPI需已知（但通常可获取）
- 与已知屏幕位姿方法相比有差距（43.4 vs 39.4 on EVE），位姿估计仍是瓶颈

## 与相关工作的对比
- **EFE**：有投影但需屏幕校准。本文的可学习参数+无校准设定更实用
- **IVGaze**：用transformer细化投影点，10-shot下严重欠拟合
- **传统校准方法**：需专业设备。本文方法仅需10张图片即可替代

## 启发与关联
- "3D先验+可微投影到2D"的范式可推广到其他跨维度任务（如3D姿态→2D动作，3D物体→2D检测）
- 动态坐标系对齐的SVD方法可用于其他微调中特征空间漂移的问题

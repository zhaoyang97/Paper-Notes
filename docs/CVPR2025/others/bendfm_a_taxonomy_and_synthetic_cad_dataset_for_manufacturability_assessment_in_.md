# BenDFM: A taxonomy and synthetic CAD dataset for manufacturability assessment in sheet metal bending

**会议**: CVPR 2025  
**arXiv**: [2603.13102](https://arxiv.org/abs/2603.13102)  
**代码**: [GitHub](https://github.com/UGent-CVAMO/bendfm)  
**领域**: 其他 / 制造设计 / CAD数据集  
**关键词**: 制造设计(DFM), 可制造性评估, 板金弯曲, 合成CAD数据集, 几何深度学习, 分类法

## 一句话总结

提出一个面向板金弯曲工艺的可制造性度量分类法（按配置依赖性×可行性/复杂度两个维度划分为四象限），并构建首个包含20,000个零件（含可制造与不可制造样本）的合成数据集BenDFM，基准测试表明图结构表示（UV-Net）优于点云（PointNext），配置依赖性指标的预测更具挑战性。

## 研究背景与动机

### 领域现状
- **制造设计（DFM）**旨在设计阶段即识别生产约束，降低成本和缩短上市时间
- DFM包含两个互补阶段：工艺选择（process selection）和工艺内可制造性评估（IPMA）
- 工艺选择方面深度学习已取得快速进展，但IPMA领域仍几乎空白

### 现有痛点
1. **可制造性定义不一致**：文献中"可制造性"的含义差异巨大——有的取决于特定工装（配置依赖），有的取决于几何本身（配置无关）；有的衡量可行性（能否制造），有的衡量复杂度（制造难度）
2. **数据严重匮乏**：
   - 工业数据集存在**幸存者偏差**——只包含已成功制造的设计，缺少不可制造样本
   - 现有合成数据集聚焦于简单立方体+减材工艺，对真实CAD零件泛化性差
   - **板金弯曲领域完全没有DFM数据集**
3. **设计师与制造工程师之间存在知识鸿沟**：DFM知识难以系统化传播和复用

### 核心思路
从两个层面解决问题：
- **概念层面**：提出分类法，厘清可制造性的不同含义
- **数据层面**：构建首个面向板金弯曲的合成数据集，覆盖分类法的所有四个象限

## 方法详解

### 可制造性分类法（Taxonomy）
沿两个正交维度划分，形成四个象限：

| | 可行性 (Feasibility) | 复杂度 (Complexity) |
|--|--|--|
| **几何（配置无关）** | 几何可行性：纯几何决定的不可制造（如展开自交叉） | 几何复杂度：纯几何难度指标（如展开面积、弯曲数量） |
| **配置（配置依赖）** | 配置可行性：依赖工装/设备的不可制造（如冲头碰撞） | 配置复杂度：依赖工序的制造努力（如翻转次数、重定向距离） |

核心洞察：几何标签具有广泛的可迁移性（适用于早期设计阶段），配置标签具有工业精度但可迁移性受限。

### BenDFM数据集生成

#### 参数化弯曲生成（Sec. 4.1）
- 从初始平板出发，迭代添加折弯法兰，五步流程：选择弯曲边 → 构建2D面 → 挤出弯曲 → 挤出法兰 → 重复
- 加权采样策略：优先选择靠近基板且较长的边
- 三种增强真实性的设计：弯曲释放槽（20%弯曲）、法兰形状变体（矩形/倾斜/圆角）、对称偏置
- 弯曲角度从{45°, 60°, 90°, 120°, 135°}采样，90°优先

#### 工装几何建模（Sec. 4.2）
- 参数化建模冲头和凹模几何，使用基本三角学确定位置
- 冲头：以弯曲弧内侧中点为基础构建等腰三角形并挤出
- 凹模：根据凹模开口宽度和弯曲弧的切线交点构建

#### 动态弯曲建模与展开（Sec. 4.3）
- 以5°增量模拟弯曲过程中的中间状态（而非仅评估最终状态）
- 使用弯曲裕量（Bend Allowance）公式补偿材料变形：BA = (π/180)·θ·(r + K·t)
- 生成展开的平面图案（flat pattern），存储为2D表示

#### 可制造性标签生成（Sec. 4.4）
- **冲头-零件碰撞**和**凹模-零件碰撞**：在沿弯曲轨迹每5°的中间状态检测
  - 三位置对齐灵活性（左/中/右），避免假阳性
  - 后验碰撞检查：在所有弯曲完成后回溯检查
- **展开重叠**：通过B-rep布尔运算检测平面图案自交叉
- **复杂度指标**：零件重定向距离、重定向角度、翻转指示器；弯曲数量、体积、质量等

### 数据集规模
- 总计20,000个3D零件（STEP格式），每个附带展开模型、完整弯曲序列和制造参数
- BenDFM主子集：14,000件（2-8弯），50/50碰撞平衡
- BenDFM-U子集：6,000件（7-10弯），50/50展开重叠平衡

### 关键发现
- UV-Net（图结构AAG）在所有四个任务上均优于PointNext（点云），尤其在展开相关任务上差距更大
- **几何指标比配置指标更容易预测**：两个模型都在几何可行性/复杂度上表现更好

## 亮点与洞察

1. **分类法本身是重要贡献**：将模糊的"可制造性"概念清晰化为四象限框架，对整个DFM领域有指导意义
2. **首个板金弯曲DFM数据集**：填补了成型工艺合成数据的空白
3. **过程感知的数据生成**：动态弯曲建模（5°增量）+ 弯曲裕量 + 后验碰撞检查，远超简单几何生成
4. **平衡设计**：40%弯曲保证无碰撞 + 按弯曲数分层采样，避免模型学到虚假相关
5. **实证验证了重要假设**：配置依赖任务确实比几何任务更难学习

## 局限性 / 可改进方向

1. **单一工装配置**：所有零件使用固定的冲头/凹模参数，未探索跨配置泛化
2. **未验证真实世界迁移**：合成数据训练的模型在真实零件上的效果未知
3. **缺少序列建模**：当前输入仅为最终3D几何，未利用弯曲操作序列信息
4. **性能仍有较大提升空间**：碰撞检测最佳AUC仅0.840，距实用部署有差距
5. **未建模回弹、弹性恢复等物理现象**
6. **可扩展到其他成型工艺**（深拉伸、冲压、管弯曲）以验证分类法的通用性

## 相关工作与启发

- 对比FeatureNet/MFCAD等制造特征识别数据集，BenDFM针对的是工艺内评估而非工艺选择
- 相比Ghadai2018（钻孔深径比）和Peddireddy2021（内空腔）的简单立方体DFM数据集，BenDFM几何复杂度显著更高
- UV-Net在B-rep学习中的优势与Li2025、Hussong2025等近期发现一致
- 分类法可指导其他制造领域的数据集构建和模型评估

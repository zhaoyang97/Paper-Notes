# Small Target Detection Based on Mask-Enhanced Attention Fusion of Visible and Infrared Remote Sensing Images

**会议**: CVPR2025  
**arXiv**: [2603.06925](https://arxiv.org/abs/2603.06925)  
**代码**: 待确认  
**领域**: 多模态检测 / 遥感  
**关键词**: 可见光-红外融合, 小目标检测, 空间掩码, 轻量网络, 训练时超分

## 一句话总结
提出 ESM-YOLO+，一种轻量级可见光-红外融合网络，通过 MEAF 模块（可学习空间掩码+空间注意力的像素级融合）和训练时结构表示增强（SR，推理时无开销的超分辅助监督），在 VEDAI 上达到 84.71% mAP 同时参数量仅 5.1M（减少 93.6%）。

## 研究背景与动机
1. **任务场景**：无人机/卫星遥感图像中的小目标检测，目标像素占比极低、弱纹理、易被复杂背景干扰。
2. **多模态互补性**：可见光提供丰富的形状纹理，红外提供热辐射显著性，融合可增强小目标表征。
3. **现有问题**：
   - 跨模态特征异质性（尺度、纹理、热特征不同）使判别性特征提取困难
   - 空间/时间不对齐（传感器视角、采集时间差、无人机运动）降低融合质量
   - 现有融合网络（MIR-YOLO、DVIF-Net 等）结构复杂，不适合资源受限的边缘部署
4. **核心主张**：可见光-红外小目标检测的根本挑战不在于激进的特征增强，而在于构建模态不变且空间一致的表示——需要"先掩码后注意力"的可靠性条件化融合。

## 方法详解

### ESM-YOLO+ 整体架构
三个核心组件：MEAF 像素级融合模块 → 检测 Backbone + 检测头 → 训练时 SR 超分支路（推理时移除）

### MEAF（Mask-Enhanced Attention Fusion）

1. **模态缩放**：可学习参数 $p^{RGB}, p^{IR}$ 加权两路输入
2. **空间掩码生成**：对各模态用 3×3 Conv + ReLU + 1×1 Conv 生成空间掩码，逐元素乘以原特征，突出显著结构
3. **残差连接+卷积聚合**：掩码特征 + 原始特征 → 3×3 Conv 精炼
4. **空间注意力**：对通道维度做 avg-pool + max-pool → concat → 1×1 Conv + Sigmoid 生成空间注意力图
5. **模态融合**：注意力调制后的两路特征 concat → 通道激励向量 M 加权 → 输出融合特征

### SR 训练时结构表示增强
- 从 backbone 中间层取特征 → 轻量解码器（上采样+卷积）重建图像
- 损失 $\mathcal{L}_{SR} = \|S - \mathcal{D}(X)\|_1$，强制 backbone 特征保留细粒度空间结构
- **训练时梯度回传到 backbone**，推理时丢弃 SR 支路 → 零推理开销

### 总损失
$\mathcal{L}_{total} = c_1 \mathcal{L}_{\text{det}} + c_2 \mathcal{L}_{SR}$

## 实验关键数据

### VEDAI 数据集 SOTA 对比

| 方法 | mAP50 | 参数量 |
|------|-------|--------|
| YOLOv5x | 62.65 | — |
| SuperYOLO | 75.09 | — |
| ESM-YOLO (baseline) | 82.42 | — |
| ACDF-YOLO | 78.1 | — |
| CFT (Transformer) | 85.3 | 196.9M |
| **ESM-YOLO+** | **84.71** | **5.1M** |

- mAP 较 baseline ESM-YOLO +2.29%
- 参数量仅 5.1M，相比 baseline 减少 93.6%，GFLOPs 减少 68.0%
- 与 Transformer 方法 CFT（85.3%）精度接近但参数量减少 97.4%

### DroneVehicle 数据集
- mAP 74.0%，在更大规模数据集上验证有效性

### 单模态 vs 多模态消融（VEDAI）

| 方法 | mAP50 |
|------|-------|
| 单模态 RGB | 75.06 |
| 单模态 IR | 70.83 |
| **多模态 ESM-YOLO+** | **84.71** |

- 多模态融合带来 +9.65% 提升

### 与专用多模态小目标方法对比（VEDAI）

| 方法 | mAP50 |
|------|-------|

### 训练细节
- SGD 优化器，momentum=0.937，weight decay=0.0005，300 epochs，batch=2
- 损失权重：$\alpha_o^a=1.0, \alpha_l^a=0.05, \alpha_c^a=0.5$
- 模态融合参数 $p^{RGB} = p^{IR} = 0.5$
- 训练用 1024×1024，测试用 512×512

## 亮点
- **掩码-注意力两步融合范式**：先用可学习空间掩码做对齐感知选择，再用空间注意力保持小目标空间支撑，从原理上解决跨模态异质性
- **训练时 SR 的零推理代价**：巧妙利用辅助任务正则化 backbone，推理时直接丢弃
- **极致轻量化**：5.1M 参数实现与 196.9M 参数的 Transformer 方法相当的精度
- **实际部署友好**：CNN 架构 + 低计算量，适合边缘设备和实时场景

## 局限性
- VEDAI 数据集规模较小（1246 对图像），泛化性需在更大数据集上进一步验证
- MEAF 融合假设 RGB 和 IR 已基本对齐，严重不对齐场景可能需要额外的配准模块
- SR 辅助分支的超参 $c_2$ 对效果的敏感性未充分讨论
- 仅支持双模态（RGB+IR），扩展到更多模态需要架构调整
- 与最新专用多模态小目标检测方法的比较可进一步扩充

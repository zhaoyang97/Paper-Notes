# Text-Phase Synergy Network with Dual Priors for Unsupervised Cross-Domain Image Retrieval

**会议**: CVPR2025  
**arXiv**: [2603.12711](https://arxiv.org/abs/2603.12711)  
**代码**: 待确认  
**领域**: self_supervised  
**关键词**: 无监督跨域图像检索, CLIP, 文本先验, 相位先验, 频域对齐, 域提示, 伪标签噪声

## 一句话总结
提出 TPSNet，利用文本-相位双先验解决无监督跨域图像检索：域提示（text prior）提供比伪标签更精确的语义监督，相位特征（phase prior）实现保持语义的域不变对齐，两者通过交叉注意力协同融合。

## 背景与动机
- 无监督跨域图像检索（UCDIR）旨在无标注下跨不同域（如真实图像 vs 素描）检索同类图像
- 现有方法两阶段范式: 聚类生成伪标签 → 域内对比学习 + 跨域特征对齐
- **伪标签问题**: K-means 聚类产生的离散伪标签不够准确和全面，导致错误配对约束和低质量类原型
- **语义退化问题**: 跨域对齐（对抗学习、统计分布对齐等）忽视了域特定信息与语义信息的纠缠，在空间域对齐过程中损失语义信息
- 需要更好的语义监督和域无关的对齐策略

## 核心问题
如何在无标注的跨域检索中：(1) 提供比伪标签更准确的语义监督？(2) 在跨域对齐过程中不损失语义信息？

## 方法详解
TPSNet 包含两个关键组件: 域提示生成模块 (DPG) 和 文本-相位双先验网络 (TPDP)。

### 1. 域提示生成模块 (DPG) — 文本先验
- 对两个域的图像分别做 K-means 聚类，为每个聚类初始化一个可学习的文本提示
- 提示模板: "An image of a [X]₁[X]₂...[X]_M"（[X]ᵢ 为可学习 token）
- 冻结 CLIP 编码器，仅优化提示 token
- 使用改进的跨模态对比学习: 图像到文本损失 L_i2t + 文本到图像损失 L_t2i
- **关键创新**: 在 L_i2t 中基于余弦相似度重配对图像和文本嵌入，可部分纠正不准确的伪标签
- 优化后的域提示编码了更精确的语义信息，作为后续的文本先验

### 2. 文本先验语义特征提取 (TPFE)
- 将学到的域提示通过共享文本编码器生成文本特征 T_A' 和 T_B'
- 这些文本特征作为文本先验指导视觉特征的语义学习

### 3. 相位先验域不变特征提取 (PPFE) — 相位先验
- **频域分解**: 对灰度图执行 FFT，分离振幅谱和相位谱
- **相位图像重建**: 保留相位谱，将振幅替换为常数 R，执行 IFFT 得到相位图像 X^{phase}
- **相位特征编码器**: 轻量 CNN (Conv3×3 + BN + ReLU + AvgPool) → Flatten → FC 得到相位特征 I^{phase}
- **特征融合**: RGB 特征 I^{rgb} 与 I^{phase} 拼接后通过自注意力 + LayerNorm + 均值池化得到融合特征 I^f
- **原理**: 相位谱编码了跨域不变的结构和语义线索，比振幅谱更鲁棒

### 4. 双先验协同融合
- 交叉注意力: 文本特征 T' 作为 query，融合视觉特征 I^f 作为 key/value
- 输出 I' 同时包含文本先验的精确语义和相位先验的域不变性
- 每个 epoch 通过动量更新类原型

### 5. 损失函数
- 原型交叉熵损失 L_pce: 让特征接近同类原型
- 图像到文本对比损失 L_i2tce: 配合标签平滑 σ_j 减轻伪标签误差
- 总损失: L = α·L_pce + β·L_i2tce

## 实验关键数据

### Office-Home（ViT-B backbone，P@1/P@5/P@15）
| 方向 | TPSNet P@1 | SA-MoE P@1 | DD P@1 |
|------|-----------|-----------|--------|
| Art→Real | 89.53 | 71.12 | 60.94 |
| Real→Art | 91.74 | 73.86 | 61.92 |
| Art→Product | 81.17 | 64.69 | 58.76 |
| Clipart→Real | 51.68 (R50) | - | 33.31 |

- 在 ViT-B 设置下，Art→Real P@1 达到 89.53，比 SA-MoE 高出 18.4 个点
- 在所有 12 个跨域检索场景中均大幅领先

### ResNet-50 backbone 同样显著领先
- 对比 ProtoOT/ShieldIR，平均提升 15-20 个百分点

## 亮点
1. **文本-相位双先验设计**: 两种先验分别从不同角度解决 UCDIR 的核心问题（语义监督不准确 + 跨域对齐语义退化）
2. **频域相位分离的巧妙应用**: 相位谱保留结构语义、丢弃域风格的思路简洁有效
3. **域提示纠正伪标签**: 通过重配对机制部分修正聚类错误
4. **实验提升幅度惊人**: 在 Office-Home ViT-B 上比次优方法高出 ~18 个点
5. **双路径设计清晰**: 文本路径提供语义，图像路径（含相位）提供域不变性，通过交叉注意力融合

## 局限性 / 可改进方向
- K-means 聚类数 C 需预设，且仍是方法的基础——如果初始聚类质量极差，域提示优化可能受限
- 相位特征编码器较为简单（仅一层 Conv+FC），可能丢失一些结构信息
- 振幅替换为常数 R 的选择（如何定 R？）未详细讨论
- 文本先验依赖 CLIP 的语义空间质量，对 CLIP 预训练数据覆盖不到的域可能效果有限
- DomainNet 仅选了 7 类（>200 样本/域），大规模设置下的效果待验证

## 与相关工作的对比
- vs. **DD/CoDA**: 直接使用伪标签，TPSNet 通过文本先验大幅提升语义监督质量
- vs. **ProtoOT/ShieldIR**: 基于最优传输的原型对齐，但仍在空间域对齐导致语义退化
- vs. **SA-MoE**: 通过语义注意力融合保持语义完整性，但 TPSNet 在此基础上引入频域相位先验，提升更大
- vs. **FDA/FUDA**: 仅操作高低频分量而非振幅/相位分离，TPSNet 的相位分离策略更直接保留语义

## 启发与关联
- 相位谱作为域不变特征的思路可推广到需要域自适应的其他视觉任务
- 文本先验的域提示设计与 prompt learning 紧密相关，可探索更复杂的提示结构
- 双先验协同的框架设计具有通用性，可扩展到其他多模态/多域学习场景
- 频域分析在跨域学习中的潜力被低估，值得更多探索

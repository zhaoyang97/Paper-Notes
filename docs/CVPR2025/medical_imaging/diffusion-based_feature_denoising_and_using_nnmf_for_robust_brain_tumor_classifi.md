# Diffusion-Based Feature Denoising and Using NNMF for Robust Brain Tumor Classification

**会议**: CVPR2025  
**arXiv**: [2603.13182](https://arxiv.org/abs/2603.13182)  
**代码**: 待确认  
**领域**: medical_imaging  
**关键词**: 脑肿瘤分类, NNMF, 扩散去噪, 对抗鲁棒性, AutoAttack

## 一句话总结

提出一种结合非负矩阵分解（NNMF）特征提取、统计特征筛选、轻量 CNN 分类和扩散式特征空间去噪的脑肿瘤分类框架，在保持 ~85% 干净准确率的同时，将 AutoAttack 下的鲁棒准确率从 0.47% 提升至 59.5%。

## 研究背景与动机

- 脑肿瘤 MRI 分类是计算机辅助诊断的重要任务，早期精确检测有助于改善治疗方案和患者生存率
- 深度学习已能取得高分类精度，但深度模型对对抗扰动高度脆弱——微小不可察觉的修改即可使分类准确率崩溃，这在医学诊断等安全关键场景中是严重隐患
- 但深度模型对对抗扰动高度脆弱——微小不可察觉的修改即可使分类准确率崩溃，这在医学诊断等安全关键场景中是严重隐患
- NNMF 可将非负数据分解为可解释的部分表示，比 PCA 更适合天然非负的医学图像像素值
- 扩散模型的去噪思想可用于特征空间的对抗扰动净化
- **核心动机**：将可解释的 NNMF 低维特征与扩散去噪防御结合，构建兼顾分类精度和对抗鲁棒性的轻量框架

## 方法详解

### 整体框架（四阶段流水线）
1. 数据预处理 → 2. NNMF 特征提取 + 统计筛选 → 3. 轻量 CNN 分类 → 4. 扩散式特征空间去噪防御

### Stage 1: 数据预处理
- 数据来源：Kaggle 脑 MRI 数据集，约 2,200 张图像（正常类：730 训/219 验/97 测；肿瘤类：770 训/210 验/118 测），70%/20%/10% 划分
- 从 COCO 标注格式重组为文件夹结构，适配分类任务
- **注意**：无患者 ID 信息，切片级泄露风险无法完全排除，论文也承认报告性能可能偏乐观
- 训练/验证/测试仅使用 Kaggle 原始单次划分，未做多次随机划分验证

### Stage 2: NNMF 特征提取
- 图像转灰度 → 128×128 resize → [0,1] 归一化 → 向量化为非负数据矩阵 $V$
- 使用 KL 散度 + 乘法更新规则进行 NNMF 分解：$V \approx WH$，秩 $R=15$
- 训练集学习基矩阵 $W$，验证/测试集通过非负最小二乘投影到固定 $W$ 上
- L2 归一化所有特征向量
- **统计特征筛选**：对每个 NNMF 分量计算 AUC、Cohen's d、Welch t-检验 p 值，选取最具区分力的 Top-M 特征

### Stage 3: 轻量 CNN 分类
- 在筛选后的 NNMF 特征子集上训练轻量 CNN（非高维图像输入）
- 验证集监控防过拟合
- 验证准确率 ~83%，测试准确率 ~85.1%

### Stage 4: 扩散式特征空间去噪

此阶段分三个子步骤：

1. **前向扩散数据生成**：对干净 NNMF 特征逐步添加高斯噪声（线性噪声表，固定步数），生成 $(x_t, x_0)$ 配对训练数据
2. **去噪网络训练**：将时间步 $t$ 通过正弦嵌入编码，与噪声特征拼接输入，MSE 损失训练回归网络预测干净特征 $x_0$
3. **测试时防御**：对测试样本先前向注噪到步 $t$，再用去噪网络恢复，最后送入分类器
4. **EOT（Expectation over Transformation）**：因防御含随机性，预测时对 K=8 次随机样本取平均

### 鲁棒性评估
- 使用 AutoAttack（$L_\infty$, $\epsilon=0.10$）包含 APGD-CE + Square 两种攻击
- MATLAB 训练模型导出 ONNX → Python/PyTorch 执行 AutoAttack

## 实验关键数据

| 模型 | 干净准确率 | 鲁棒准确率 (AutoAttack) | MCC | ROC-AUC | Brier Score |
|------|-----------|----------------------|-----|---------|-------------|
| Baseline（无防御） | 86.05% | 0.47% | -0.99 | 0.0075 | 0.4702 |
| Defended（扩散去噪） | 85.12% | **59.53%** | 0.17 | 0.7485 | 0.2150 |

**关键发现**：
- Baseline 在 AutoAttack 下完全崩溃（准确率→0.47%，MCC→-0.99 表示预测完全反转）
- 扩散防御以仅 ~1% 干净准确率代价，将鲁棒准确率提升至 59.5%
- 验证集混淆矩阵显示两类别间误分较为均衡
- GPU 加速将总运行时间从 201.5s 降至 116.6s（1.73× 加速），AutoAttack 防御阶段受益最大

## 亮点

- **特征空间扩散防御**：不同于常见的图像空间去噪，本文在 NNMF 低维特征空间中进行扩散-去噪，计算开销小
- **可解释性**：NNMF 基分量可直接可视化为空间模式，特征选择有统计依据（AUC + Cohen's d + p 值三重标准）
- **模块化设计**：各阶段独立（特征提取 / 分类 / 防御），便于替换和扩展
- **实用工具链**：MATLAB 训练 + ONNX 导出 + Python AutoAttack，展示了跨平台流水线
- **鲁棒性提升显著**：从几乎完全崩溃（0.47%）到接近 60% 的鲁棒准确率，干净准确率代价仅 ~1%

## 局限性

- **数据集过小且缺乏患者级划分**：~2,200 张图像，无患者 ID，切片级泄露可能导致性能被高估
- **二分类过于简单**：仅正常 vs 肿瘤，未涉及多类别肿瘤分型（如胶质瘤分级）
- **干净准确率偏低**（85.1%），远低于现有 SOTA 脑肿瘤分类方法，部分原因是 NNMF 压缩损失了过多判别信息
- **固定扩散步 t=41**：未探索自适应扩散表或多步净化策略
- **$\epsilon=0.10$ 在特征空间的意义不明确**：未讨论该扰动预算对应的实际图像扰动量级
- **写作质量较差**：大量语法错误、用词不当、表述混乱，影响可读性
- **NNMF 秩 R=15 的选择缺乏讨论**：未做秩选择的消融实验

## 相关工作

- **CNN 脑肿瘤检测**：Hossain et al. 的五层 CNN 在 BRATS 上达 97.87%，但依赖像素级分割 + 端到端图像输入
- **NMF-CNN 融合**：Chan et al. 在声音事件检测中用 NMF 增强 CNN 特征（F1 从 23.7% 到 30.39%）
- **Semi-NMF 网络**：Huang et al. 用 Semi-NMF 构建卷积滤波器，避免反向传播
- **分类-去噪联合模型**：Thiry & Guth 在 CIFAR-10/ImageNet 上用去噪目标提升对抗鲁棒性
- **AutoAttack**：Croce & Hein 提出的统一攻击基准，揭示了许多"鲁棒"防御实际可被攻破- **对抗训练**：PGD 等对抗训练方法是提升鲁棒性的主流方案，但本文选择了较轻量的特征空间扩散去噪路线

# CycleULM: A Unified Label-Free Deep Learning Framework for Ultrasound Localisation Microscopy

**会议**: CVPR 2025  
**arXiv**: [2603.09840](https://arxiv.org/abs/2603.09840)  
**代码**: 无  
**领域**: 医学图像  
**关键词**: ultrasound localisation microscopy, CycleGAN, domain translation, microbubble, super-resolution

## 一句话总结
提出 CycleULM，首个统一的无标签深度学习超声定位显微(ULM)框架，通过 CycleGAN 学习 CEUS 帧到简化微泡域的物理仿真双向翻译来弥合仿真-真实域差距，实现微泡定位精度提升达40% recall、46% precision，并以18.3 fps 实现实时处理。

## 研究背景与动机
1. **领域现状**：超声定位显微(ULM)通过定位和追踪注射体内的微泡(MB)造影剂实现突破衍射极限的微血管成像，已在肿瘤、脑、肾、心脏等多种组织中得到验证，近年已扩展到人体临床研究。
2. **现有痛点**：(a) MB 信号与背景噪声分离困难；(b) 重叠微泡降低定位质量；(c) 采集时间长导致运动伪影；(d) 大数据量需要大量计算。深度学习虽有潜力，但最大障碍是**缺乏标注的体内数据**——手动标注 MB 位置不可行，伪标签继承传统算法误差。
3. **核心矛盾**：现有 DL 方法依赖仿真数据训练（如高斯 PSF 模型、Field II），但仿真器无法完全复制真实条件（声波传播、非线性 MB 响应、异质组织），导致**仿真-真实域差距(domain gap)**，模型从仿真迁移到真实数据时性能大幅下降。
4. **本文要解决什么？** (a) 如何在不需要标注数据的情况下训练 ULM 深度学习模型？(b) 如何弥合仿真训练数据和真实 CEUS 数据的域差距？(c) 如何在保持高质量的同时实现实时处理？
5. **切入角度**：不再试图制造更逼真的仿真器，而是用 CycleGAN 学习从复杂真实 CEUS 域到简化 MB-only 域的双向翻译，使得下游定位和追踪可以在可控、可解释的简化域中完成。
6. **核心idea一句话**：用自监督的循环一致对抗训练学习 CEUS→MB-only 域翻译，消除对高保真仿真器和标注数据的依赖。

## 方法详解

### 整体框架
CycleULM 包含三个模块化神经网络：MB-DT（Microbubble Domain Translator，CycleGAN 架构）将 CEUS 帧翻译到简化 MB-only 域；MBL-Net（Microbubble Localisation Network）在翻译后的 MB-only 域中进行亚像素 MB 定位；MBT-Net（Microbubble Trajectory Network）从连续 MB-only 帧序列中追踪 MB 轨迹和估计速度。三个网络可作为插件模块替换传统 ULM 管线中的对应步骤，也可组合为端到端方案(CycleULM-E2E)。

### 关键设计

1. **MB-DT (Microbubble Domain Translator)**:
   - 做什么：将物理复杂的 CEUS 帧翻译为仅含 MB 信号的简化域图像，抑制背景杂波并缩小有效 PSF
   - 核心思路：基于 CycleGAN，前向生成器 $G_{AB}$ 将三帧连续 CEUS 映射到单帧 MB-only 图像，反向生成器 $G_{BA}$ 重建 CEUS 帧。训练用无配对数据，损失包括两个循环一致性损失(L1)、两个对抗损失(LSGAN)和一个相似性损失 $\mathcal{L}_{\text{sim}} = \|G_{AB}(a) - a\|_1$ 防止丢失 MB 信号
   - 设计动机：利用三帧时序上下文增强 MB 信号与杂波的区分能力；合成 PSF 缩小 0.5 倍鼓励网络生成更窄的 MB 信号，直接提升下游定位精度

2. **MBL-Net (Microbubble Localisation Network)**:
   - 做什么：从 MB-only 图像中以亚像素精度定位微泡
   - 核心思路：基于 U-Net 的多任务头网络，输出7张图：概率图 $p$、强度图 $I$、两张亚像素偏移图 $\Delta x, \Delta y$、三张不确定性图 $\sigma_I, \sigma_x, \sigma_y$。损失为 BCE 存在损失 + 基于高斯混合模型的负对数似然定位损失（借鉴 DECODE）
   - 设计动机：纯在合成 MB-only 数据上训练，因为 MB-DT 已将域差距消除，合成数据可无缝泛化到真实数据

3. **MBT-Net (Microbubble Trajectory Network)**:
   - 做什么：从8帧连续 MB-only 序列中估计轨迹概率图和水平/垂直速度图
   - 核心思路：U-Net + ConvLSTM 架构，在瓶颈和跳跃连接中嵌入 ConvLSTM 块捕获时空相关性，输出4倍上采样的轨迹/速度图。损失为 BCE 轨迹损失 + 加权 MSE 速度损失（用归一化轨迹概率加权，解决速度图稀疏性问题）
   - 设计动机：ConvLSTM 替代全连接 LSTM，在保留空间结构同时减少参数

### 损失函数 / 训练策略
- MB-DT：$\mathcal{L}_G = \mathcal{L}_{G_{AB}} + \mathcal{L}_{G_{BA}} + 5\mathcal{L}_{\text{cyc1}} + 5\mathcal{L}_{\text{cyc2}} + \mathcal{L}_{\text{sim}}$
- 所有网络在小 patch (40×40 或 80×80) 上训练，推理时应用于全帧；patch 训练增加数据多样性且允许选择 ROI 定向训练

## 实验关键数据

### 主实验（ULTRA-SR 仿真数据集，MB 定位性能）

| 方法 | Recall 提升 | Precision 提升 | 定位误差改善 |
|------|-----------|---------------|-------------|
| CycleULM-NCC vs NCC | +32% | +46% | -14.0 µm (31%) |
| CycleULM-Decon vs Decon | +7% | - | -4.5 µm (13%) |
| CycleULM-Loc vs NCC | +40% | +46% | -16.1 µm (36%) |
| CycleULM-Loc vs Decon | +8% | +26% | -3.4 µm (10%) |

### 图像质量提升

| 指标 | 原始 CEUS | MB-DT 翻译后 | 提升 |
|------|---------|-------------|------|
| CNR (ULTRA-SR) | 7.2 dB | 16.4 dB | +9.2 dB |
| CNR (兔肾 in vivo) | 14.4 dB | 29.7 dB | **+15.3 dB** |
| FWHM (ULTRA-SR) | 355 µm | 178 µm | 2× 分辨率提升 |
| FWHM (兔肾 in vivo) | 725 µm | 294 µm | **2.5× 分辨率提升** |

### 处理速度（兔肾数据，500帧）

| 方法 | 处理时间(s) | 吞吐量(fps) | 加速比 |
|------|-----------|------------|--------|
| Baseline 1 (阈值+NCC) | 174.2 | 2.9 | 1× |
| Baseline 2 (阈值+Decon) | 394.6 | 1.3 | - |
| CycleULM-E2E | **27.2** | **18.4** | **6.4×/14.5×** |

### 关键发现
- MB-DT 翻译后，即使用传统定位方法(NCC/Decon)也能显著提升性能，证明域翻译本身的价值
- CycleULM-E2E 在重建质量最优的同时速度最快（14.5× 加速），因为跳过了传统去噪和定位步骤
- 跨采集泛化：在一个兔肾数据集上训练的 MB-DT 可直接用于另一个不同视角的兔肾数据集，无需微调

## 亮点与洞察
- **域翻译消除域差距**：不追求更好仿真器，而是学习将真实数据映射到可控域——在简化域中合成训练数据完全可控，下游网络可无缝泛化。这个思路可推广到其他仿真-真实差距大的领域（如合成数据训练的目标检测）
- **模块化设计对齐传统管线**：三个网络分别对应去噪、定位、追踪三个经典步骤，既可单独替换也可端到端，降低用户采用门槛
- **相似性损失防止 MB 丢失**：简单的 L1 自监督项 $\|G_{AB}(a) - a\|_1$ 约束翻译结果不偏离原始帧太远，有效防止假阳性和漏检

## 局限性 / 可改进方向
- CycleULM-E2E 输出图像级速度/轨迹表示而非显式 MB 轨迹对象，无法做轨迹级过滤和统计分析
- PSF 估计依赖手动选取10个孤立 MB 拟合高斯，自动化程度不够
- 仅在兔肾和大鼠脑两种体内数据集上验证，需更多人体临床数据确认泛化性
- CycleGAN 架构浅层(2层下采样)，对需要大感受野的场景可能捕获上下文不足
- 训练和推理均在 2D 帧上进行，未扩展到 3D 体积超声数据
- 合成 MB-only 域使用各向同性缩小 PSF 的假设可能在某些成像配置下不成立

## 相关工作与启发
- **vs Shin et al. (2024)**：他们用 GAN 生成更逼真 MB 信号但需手动选择 MB patch 且不考虑重叠 MB 交互，CycleULM 全自动且用循环一致性保证物理合理性
- **vs DECODE (Speiser 2021)**：DECODE 是 SMLM 领域的亚像素定位方法，CycleULM 的 MBL-Net 借鉴其损失函数设计，将其思想迁移到超声领域
